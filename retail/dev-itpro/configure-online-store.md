# <a name="configure-an-online-store"></a>Määritä online-myymälä

Tässä artikkelissa on linkkejä aiheisiin, joiden avulla voit keskitetysti määrittää ja hallita verkkokaupan toimintoihin Microsoft Dynamics-365.

Seuraavassa taulukossa luetellut aiheet auttavat määrittämään työvaiheiden - Retail-komponentit ja toiminnot asiakkaan Dynamics-365-vähittäiskaupan verkkokaupan Dynamics 365.

|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Määritä vähittäismyynnin osia.                        | Määritä ja Ylläpidä vähittäismyyntitoimintojen tiedot. Näitä tietoja ovat kaupat, veroja, tuotteet, lahjakortteja, ylennykset ja alennukset.                                                                                                                                                                                                          | [Määrittäminen ja ylläpito vähittäiskaupan](https://technet.microsoft.com/en-us/library/hh597201.aspx) (TechNet-sisältöä Microsoft Dynamics AX 2012: n)                                                                                                                                                                                                                                                                                          |
| Määritä vähittäismyynnin kanava-siirtymishierarkiassa.    | Vähittäismyynnin kanavan siirtyminen luokkahierarkian luominen tuotteita, joita voit tarjota verkkokaupan Luokitusrakenteen määrittämiseen. Voit sitten liittää luokkahierarkian verkkokauppaan.                            | [Vähittäismyyntihierarkian määrittäminen](https://technet.microsoft.com/en-us/library/hh580593.aspx) (TechNet-sisältöä Microsoft Dynamics AX 2012: n) [määritteet ja attrib. tyypit](https://technet.microsoft.com/en-us/library/hh227548.aspx) (TechNet-sisältöä Microsoft Dynamics AX 2012: n) [määrittää määritteen vähittäismyyntiryhmiin](https://technet.microsoft.com/en-us/library/jj728713.aspx) (TechNet-sisältöä Microsoft Dynamics AX 2012: n) |
