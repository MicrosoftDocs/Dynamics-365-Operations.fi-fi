# <a name="import-users-in-bulk"></a>Käyttäjien tuominen joukkotoimintona

[!include[task guide banner](../../includes/task-guide-banner.md)]

Näiden ohjeiden avulla järjestelmänvalvoja voi tuoda suuren määrän käyttäjiä Azure Active Directorysta.


## <a name="run-as-a-batch-job"></a>Suorita työ komentojonotyönä
1. Valitse Järjestelmänhallinta > Käyttäjät > Käyttäjät.
2. Valitse Erän tuonti.
3. Laajenna Suorita taustalla -osa.
4. Valitse Eräkäsittely-kentässä Kyllä.
5. Kirjoita arvo Tehtävän kuvaus -kenttään.
6. Anna tai valitse arvo Eräryhmä-kentässä.
    * Tämä on valinnainen vaihe.  
7. Valitse Yksityinen-kentässä Kyllä.
    * Tämä on valinnainen vaihe.  
8. Valitse Kriittinen työ -kentässä Kyllä.
    * Tämä on valinnainen vaihe.  
9. Valitse vaihtoehto Luokan valvonta -kentässä.
10. Valitse OK.

## <a name="run-in-a-sandbox-environment"></a>Eristysympäristön suorittaminen
1. Valitse Erän tuonti.
2. Valitse OK.
