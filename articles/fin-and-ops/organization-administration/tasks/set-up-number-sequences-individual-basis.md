[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="ed7b4-104">Tässä ohjeaiheessa selitetään, miten numerosarjat määritetään yksittäin.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-104">This topic explains how to set up number sequences on an individual basis.</span></span> <span data-ttu-id="ed7b4-105">Numerosarjoja käytetään muodostamaan päätietotietueille ja niitä tarvitsevilla tapahtumatietueille luettavia, yksilöllisiä tunnisteita.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-105">Number sequences are used to generate readable, unique identifiers for master data records and transaction records that require them.</span></span> <span data-ttu-id="ed7b4-106">Tunnisteen vaativaa päätietoa tai tapahtumatietuetta kutsutaan numellä viite.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-106">A master data or transaction record that requires an identifier is referred to as a reference.</span></span> <span data-ttu-id="ed7b4-107">Ennen kuin voit luoda viitteelle uusia tietueita, määritä numerosarja ja kohdista se viitteeseen.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-107">Before you can create new records for a reference, you must set up a number sequence and associate it with the reference.</span></span> <span data-ttu-id="ed7b4-108">Voit määrittää kaikki tarvittavat numerosarjat samaan aikaan käyttämällä **Määritä numerosarjat** -ohjattua toimintoa tai voit luoda tai muokata yksittäisiä numerosarjoja käyttämällä **Numerosarjat**-sivua.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-108">You can set up all required number sequences at the same time by using the **Set up number sequences** wizard, or you can create or modify individual number sequences by using the **Number sequences** page.</span></span>

2. <span data-ttu-id="ed7b4-110">Valitse **Numerosarja**.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-110">Select **Number sequence**.</span></span>
3. <span data-ttu-id="ed7b4-111">Kirjoita arvo **Numerosarjan koodi** -kenttään.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-111">In the **Number sequence code** field, type a value.</span></span>
4. <span data-ttu-id="ed7b4-112">Kirjoita arvo **Nimi**-kenttään.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-112">In the **Name** field, type a value.</span></span>

6. <span data-ttu-id="ed7b4-118">Laajenna **Segmentit**-osa.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-118">Expand the **Segments** section.</span></span>
    - <span data-ttu-id="ed7b4-119">Määritä numerosarjalle muoto lisäämällä, poistamalla ja uudelleenjärjestelemällä segmenttejä.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-119">Define the format for the number sequence by adding, removing, and rearranging segments.</span></span>  
    - <span data-ttu-id="ed7b4-120">Kaikkien alueiden numerosarjat voivat sisältää *kiinteitä segmenttejä* ja *aakkosnumeerisia segmenttejä*.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-120">Number sequences of all scopes can contain *Constant segments* and *Alphanumeric segments*.</span></span> <span data-ttu-id="ed7b4-121">Kiinteät segmentit sisältävät sarjan aakkosnumeerisia merkkejä, jotka eivät muutu.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-121">Constant segments contain a set of alphanumeric characters that do not change.</span></span> <span data-ttu-id="ed7b4-122">Tämän segmenttityypin avulla voit lisätä väliviivan tai muun erottimen numerosarjasegmenttien välille.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-122">Use this segment type to add a hyphen or other separators between number sequence segments.</span></span> <span data-ttu-id="ed7b4-123">Aakkosnumeeriset segmentit sisältävät yhdistelmän ristikkomerkkejä (#) ja et-merkkejä (&).</span><span class="sxs-lookup"><span data-stu-id="ed7b4-123">Alphanumeric segments contain a combination of number signs (#) and ampersands (&).</span></span> <span data-ttu-id="ed7b4-124">Nämä merkit vastaavat kirjaimia ja numeroita, jotka kasvavat aina, kun sarjan numeroa käytetään.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-124">These characters represent letters and numbers that increment every time that a number from the sequence is used.</span></span> <span data-ttu-id="ed7b4-125">Käyttää ristikkomerkkiä (#) osoittamaan kasvavia numeroita ja et-merkkiä (&) osoittamaan kasvavia kirjaimia.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-125">Use a number sign (#) to indicate incrementing numbers and an ampersand (&) to indicate kuten yritys tai oikeushenkilö, eivät ole pakollisia.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-128">Scope segments, such as company or legal entity, are not required.</span></span> <span data-ttu-id="ed7b4-129">Vaikka et sisällytä alueen segmenttejä muotoon, valitun viitteen numerot luodaan edelleen aluekohtaisesti.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-129">However, if you do not include scope segments in the format, numbers for the selected reference are still generated per scope.</span></span>  
7. <span data-ttu-id="ed7b4-130">Laajenna **Viitteet**-osa.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-130">Expand the **References** section.</span></span> <span data-ttu-id="ed7b4-131">Valitse asiakirjan tyyppi tai tietue jolle tämä numerosarja määritetään.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-131">Select the document type or record to assign this number sequence to.</span></span> <span data-ttu-id="ed7b4-132">Tämä vaihe on valinnainen niille numerosarjoille, jotka on määritetty erityisiä sovelluksia varten.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-132">This step is optional for sequences that are defined for special application usage patterns.</span></span> <span data-ttu-id="ed7b4-133">Näissä tilanteissa uusi numero luodaan ilman viittausta käyttämällä numerosarjan koodin tai tunnuksen arvoa.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-133">In these scenarios, a new number is generated by using the value of a number sequence code or ID, without using a reference.</span></span> <span data-ttu-id="ed7b4-134">Esimerkki erityisestä hakemuksen käyttötavasta on tositesarja jota käytetään tiettyihin kirjauskansion nimiin.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-134">An example of a special application usage pattern is a voucher series that is used for specific journal names.</span></span> <span data-ttu-id="ed7b4-135">Emme kuitenkaan suosittele, että käytät näitä kuvioita.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-135">However, we do not recommend that you use such patterns.</span></span>

9. <span data-ttu-id="ed7b4-143">Valitse **Tallenna**.</span><span class="sxs-lookup"><span data-stu-id="ed7b4-143">Click **Save**.</span></span>
