[!include [banner](../includes/banner.md)]

<span data-ttu-id="76257-104">Tässä artikkelissa kerrotaan, miten sivut voidaan näyttää rinnakkain Microsoft Dynamics 365 for Finance and Operationsissa.</span><span class="sxs-lookup"><span data-stu-id="76257-104">This article explains how to display pages side-by-side in Microsoft Dynamics 365 for Finance and Operations.</span></span>

<span data-ttu-id="76257-105">Microsoft Dynamics 365 for Finance and Operations auttaa suorittamaan tehtävät tehokkaasti.</span><span class="sxs-lookup"><span data-stu-id="76257-105">Microsoft Dynamics 365 for Finance and Operations helps you perform tasks efficiently.</span></span> <span data-ttu-id="76257-106">Joskus haluat ehkä suorittaa tehtäviä nopeasti ja tarkastella useita sivuja rinnakkain.</span><span class="sxs-lookup"><span data-stu-id="76257-106">In some cases, you may want to view multiple pages side-by-side to complete tasks quickly.</span></span> <span data-ttu-id="76257-107">Voit esimerkiksi haluta tarkistaa usean kirjauskansion rivejä tai syöttää niitä useaan kirjauskansioon.</span><span class="sxs-lookup"><span data-stu-id="76257-107">As an example, you might want to validate or enter lines in more than one journal.</span></span> <span data-ttu-id="76257-108">Yleensä tämä vaatii palaamisen kirjauskansioluettelon sisältävälle sivulle ja takaisin tietyn kirjauskansion sisältävälle sivulle.</span><span class="sxs-lookup"><span data-stu-id="76257-108">Typically, to do this you would have to go back and forth between the page that displays a list of journals, and the page that displays lines for a given journal.</span></span> <span data-ttu-id="76257-109">**Avaa uudessa ikkunassa** -toiminnon avulla voit kuitenkin näyttää nämä sivut rinnakkain, jolloin tehtävien suorittaminen nopeutuu.</span><span class="sxs-lookup"><span data-stu-id="76257-109">However, the **Open in new window** feature enables you to display these pages side-by-side so that you can perform your tasks quickly.</span></span>

<span data-ttu-id="76257-111">[![open-in-new-window-icon](./media/open-in-new-window-icon.png)](./media/open-in-new-window-icon.png)</span><span class="sxs-lookup"><span data-stu-id="76257-111">[![open-in-new-window-icon](./media/open-in-new-window-icon.png)](./media/open-in-new-window-icon.png)</span></span>

<span data-ttu-id="76257-112">Kun **Avaa uudessa ikkunassa** -kuvake valitaan, rivien sivu avataan uudessa selainikkunassa ponnahdusikkunana. <span data-ttu-id="76257-115">[![pages-show-side-by-side](./media/pages-show-side-by-side.png)](./media/pages-show-side-by-side.png)</span><span class="sxs-lookup"><span data-stu-id="76257-115">[![pages-show-side-by-side](./media/pages-show-side-by-side.png)](./media/pages-show-side-by-side.png)</span></span>

<span data-ttu-id="76257-116">Dynaaminen linkitys ja päivittäminen tapahtuu näiden sivujen taustalla olevien tietojen suhteiden vuoksi.</span><span class="sxs-lookup"><span data-stu-id="76257-116">The dynamic linking and refreshing happens due to the relations that exist between the data that is backing these pages.</span></span> <span data-ttu-id="76257-117">Jos järjestelmä ei ota huomioon tietojen välisiä suhteita, ponnahdusikkunaa ei päivitetä automaattisesti alkuperäisessä ikkunassa tehtävien muutosten perusteella.</span><span class="sxs-lookup"><span data-stu-id="76257-117">If the system is not aware of the relation between the data, the pop-up window will not refresh automatically in response to a change in the window it originated from.</span></span>

<span data-ttu-id="76257-118">Joillakin sivuilla on useita näkymiä, kuten ruudukko-, otsikko- ja tietonäkymä.</span><span class="sxs-lookup"><span data-stu-id="76257-118">Some pages have multiple views such as the Grid view, Header view, and Details view.</span></span> <span data-ttu-id="76257-119">**Avaa uudessa ikkunassa** -kuvake avaa koko sivun uudessa selainikkunassa.</span><span class="sxs-lookup"><span data-stu-id="76257-119">The **Open in new window** icon causes the entire page to be opened in the new browser window.</span></span> <span data-ttu-id="76257-120">Tämän vuoksi **Avaa uudessa ikkunassa** -toiminnon avulla samaa sivua ei voi näyttää rinnakkain.</span><span class="sxs-lookup"><span data-stu-id="76257-120">Therefore, you cannot keep two views of the same page side-by-side using the **Open in new window** feature.</span></span> <span data-ttu-id="76257-121">Melkein kaikilla tällaisilla sivuilla on kuitenkin siirtymisluettelo, jonka avulla tietueiden välillä voi siirtyä. Tällöin käyttökokemus on samanlainen.</span><span class="sxs-lookup"><span data-stu-id="76257-121">However, almost all such pages have a navigation list that you can use to switch between records and achieve a similar experience.</span></span>

<span data-ttu-id="76257-122">Määritä selaimen ponnahdusikkunoiden estotoiminto sallimaan ponnahdusikkunat Finance and Operations -sivuston URL-osoitteesta, ennen kuin käytät **Avaa uudessa ikkunassa** -toimintoa.</span><span class="sxs-lookup"><span data-stu-id="76257-122">Before using the **Open in new window** feature, you should configure your browser's pop-up blocker to allow pop-ups from the URL of the Finance and Operations site.</span></span> <span data-ttu-id="76257-123">Ponnahdusikkunat voidaan sallia esimerkiksi osoitteesta \*.dynamics.com.</span><span class="sxs-lookup"><span data-stu-id="76257-123">As an example, you could allow pop-ups from "\*.dynamics.com".</span></span>

<span data-ttu-id="76257-124">**Avaa uudessa ikkunassa** -toiminto on käytettävissä vain, kun ikkunassa on avoinna useita sivuja.</span><span class="sxs-lookup"><span data-stu-id="76257-124">The **Open in new window** feature is only available when there is more than one page open in the window.</span></span> <span data-ttu-id="76257-125">Ponnahdusikkunan suljetaan automaattisesti, kun sivuja ei ole enää avoinna (eli sitten, kun ikkunan viimeinen sivu suljetaan).</span><span class="sxs-lookup"><span data-stu-id="76257-125">Also, the pop-up window automatically closes when there are no more pages open (that is, when the last page in that window is closed).</span></span> <span data-ttu-id="76257-126">Finance and Operations sulkee sivut myös silloin, kun siirryt sovelluksen toiselle alueelle.</span><span class="sxs-lookup"><span data-stu-id="76257-126">Finance and Operations also closes open pages when you navigate to a different area in the application.</span></span> <span data-ttu-id="76257-127">Jos ponnahdusikkunoita siis on avoinna silloin, kun siirryt sovelluksen toiselle alueelle, ponnahdusikkunat suljetaan automaattisesti, koska järjestelmä sulkee ikkunoissa olevat sivut.</span><span class="sxs-lookup"><span data-stu-id="76257-127">Therefore, if you have pop-up windows open and navigate to a different area in the application, the pop-up windows are automatically closed because the pages in those windows were closed by the system.</span></span>

<span data-ttu-id="76257-128">Ponnahdusikkunoiden yläpalkeissa on tietoja yrityksestä, jolle sivu on avattu. Tiedot ovat vain luku -muodossa.</span><span class="sxs-lookup"><span data-stu-id="76257-128">The top bar in the pop-up windows displays information about the company the page was opened in and is read-only.</span></span> <span data-ttu-id="76257-129">Ponnahdusikkunat ovat riippuvaisia myös Finance and Operations -selainikkunasta.</span><span class="sxs-lookup"><span data-stu-id="76257-129">The pop-up windows also rely on the main Finance and Operations browser window.</span></span> <span data-ttu-id="76257-130">Jos pääikkuna suljetaan tai päivitetään, kaikki avoimet ponnahdusikkunat siirtyvät vain luku -tilaan.</span><span class="sxs-lookup"><span data-stu-id="76257-130">If the main window is closed or refreshed, all open pop-up windows will become read only.</span></span> <span data-ttu-id="76257-131">Tällöin voit yhä tarkastella ikkunoiden tietoja, mutta et voi käsitellä niitä.</span><span class="sxs-lookup"><span data-stu-id="76257-131">This means that you can still view the information in these windows, but you will not be able to interact with it.</span></span>
