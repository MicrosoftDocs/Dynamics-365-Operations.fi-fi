# <a name="axerd-page-has-been-retired"></a>AxErd-sivu on jätetty pois

[!include [banner](../includes/banner.md)]

AxErd-verkkosivu on valitettavasti poistettu käytössä. Tällä sivulla oli monien Microsoft Dynamics AX 2012 -taulujen ERD- eli tietuesuhdekaavioissa. Projektin lähdekoodi ja verkkosivun HTML-tiedostot on julkaistu GitHub-tietosäilöön: https://github.com/Microsoft/ax-2012-doc-tools

HTML-sivut ovat repossa osoitteessa [https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Default.htm](https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Default.htm)

Kuvaus siitä, miten voit luoda omia ERD- eli tietuesuhdekaavioita osoitteessa: [https://github.com/Microsoft/ax-2012-doc-tools/blob/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Help-HowToUse-AxErd.htm](https://github.com/Microsoft/ax-2012-doc-tools/blob/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Help-HowToUse-AxErd.htm)

Ratkaisu on AxErd-kansion päätasolla [https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd](https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd).
