# <a name="loan-item-to-a-worker"></a>Lainaa kohde työntekijälle

[!include [task guide banner](../../includes/task-guide-banner.md)]

Tässä menettelyssä kerrotaan, miten kohde lainataan työntekijälle sekä se, miten työntekijän tekemä kohteen palautus tallennetaan. Työntekijät voivat pyytää lainan kohteita myös Työntekijän itsepalvelu -sivujen kautta. Tämän menettelyn luomisessa käytetty esittely-yritys on USMF.


## <a name="loan-item-to-a-worker"></a>Lainaa kohde työntekijälle
1. Siirry kohtaan Henkilöstöhallinto > Työntekijät > Lainan kohteet > Lainatut laitteet.
2. Valitse Uusi.
3. Syötä tai valitse arvo Henkilö-kenttään.
4. Anna tai valitse arvo Lainanimike-kenttään.
5. Syötä Suunniteltu palautus -kenttään päivämäärä, jolloin työntekijän on palautettava lainan kohde.
6. Valitse Tallenna.
7. Sulje sivu.

## <a name="return-a-loan-item"></a>Lainan kohteen palauttaminen
1. Siirry kohtaan Henkilöstöhallinto > Työntekijät > Lainan kohteet > Lainatut laitteet.
2. Valitse Muokkaa.
3. Syötä Todellinen palautuspvm -kenttään päivämäärä.
