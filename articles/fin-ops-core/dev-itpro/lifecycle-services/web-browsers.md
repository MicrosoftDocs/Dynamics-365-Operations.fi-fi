# <a name="supported-web-browsers-for-lifecycle-services"></a>Tuetut verkkoselaimet Lifecycle Servicesille 

[!include[banner](../includes/banner.md)]

Käyttäjät voivat käyttää Microsoft Dynamics Lifecyle Services (LCS) -palveluita näiden yleisimpien selainten uusimpien versioiden avulla: 

- Microsoft Edge (suositeltava: [Chromium-pohjainen Edge](https://support.microsoft.com/microsoft-edge/download-the-new-microsoft-edge-based-on-chromium-0f4a3dd7-55df-60f5-739f-00010dba52cf))
- Google Chrome
- Apple Safari

Ohjelmistovalmistajien sivustot sisältävät tietoja kunkin selaimen uusimmasta versiosta. Parhaan suorituskyvyn ja optimaalisen käyttökokemuksen takaamiseksi suosittelemme, että käytät modernin selaimen viimeisintä versiota, erityisesti Microsoft Edgeä.
