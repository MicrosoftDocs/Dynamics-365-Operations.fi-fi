# <a name="get-support-for-finance-and-operations-apps-or-lifecycle-services-lcs"></a>Finance and Operations -sovellusten tai elinkaaripalveluiden (LCS) tuen saaminen

[!include [banner](../includes/banner.md)]

Tässä ohjeaiheessa kerrotaan Finance and Operations -sovellusten ja Microsoft Dynamics Lifecycle Servicesin (LCS) tuesta. 

<table>
<thead>
<tr>
<th>Tehtävä</th>
<th>Lisätietoja</th>
</tr>
</thead>
<tbody>
<tr>
<td>Kysy yhteisöltä.</td>
<td>Siirry in <a href="https://community.dynamics.com/">Dynamics 365 -yhteisön</a> sivulle, jolla voit saada apua kysymyksiisi Microsoft Dynamics -yhteisöltä.</td>
</tr>
<tr>
<td>Vastauksia kysymyksiin lisensoinnista.</td>
<td>Ota yhteyttä kumppaniin tai Microsoftin myyntiedustajan.</td>
</tr>
<tr>
<td>Käytä <strong>Ongelmahaku</strong>-työkalua.</td>
<td>Voit etsiä <a href="https://lcs.dynamics.com/">LCS</a>-sivustossa <strong>Ongelmanhaku</strong>-työkalun avulla nopeasti Microsoft Knowledge Base (KB) -artikkeleita, hotfix-korjauksia ja ilmoitettujen ongelmien ratkaisuehdotuksia. Näet, mitä ilmoitettuja ongelmia korjataan parhaillaan tai mitkä on jo korjattu tiettyjen toiminta-alueiden osalta. Lisätietoja on kohdassa <a href="issue-search-lcs.md">Ongelmahaku (Lifecycle Services, LCS)</a>.</td>
</tr>
<tr>
<td>Hanki sovelluksen tuki.</td>
<td>Valitse sovelluksen oikeasta yläkulmasta <strong>Ohje</strong>-painike (<strong>?</strong>) ja valitse sitten <strong>Tuki</strong>. Ongelmista ilmoitetaan LCS:n <strong>Aktiiviset ongelmat</strong> -välilehdessä. Siellä järjestelmänvalvojat voivat määrittää, haluavatko he tarjota sisäistä tukea vai lähettää ongelmat Microsoftille.</td>
</tr>
<tr>
<td>Avaa tukipalvelupyyntö Microsoftin tukitiimissä.</td>
<td><a href="https://lcs.dynamics.com/">LCS:n</a> <strong>Tuki</strong>-ruutu avaa työkalun, jolla voit hallita tukitapahtumia. Voit lähettää ongelmat suoraan valitsemalla LCS-projektin <strong>Tuki</strong>-ruudun. Sitten voit lähettää ongelmia kahdella tavalla:
<ul>
<li>Valitse <strong>Aktiivinen ongelma</strong> -välilehdessä ensin ongelma ja sitten <strong>Lähetä Microsoftille</strong>.</li>
<li>Valitse <strong>Lähetetty Microsoftille</strong>-välilehdessä <strong>Lähetä ongelma</strong> ja lähetä ongelma sitten noudattamalla ruudussa näkyviä ohjeita. Kun olet lähettänyt ongelman, saat sähköpostiviestin Microsoftin tukihenkilöltä, jolle palvelupyyntösi on osoitettu.</li>
</ul>
</td>
</tr>
<tr>
<td>Uusien ominaisuuksien ja toimintojen pyytäminen.</td>
<td><a href="https://experience.dynamics.com/ideas/">Dynamics 365 -sovellusideat</a> -sovelluksessa voit tarkastella, etsiä ja äänestää aiemmin luotuja tai lisätä uusia ideoita.</td>
</tr>
</tbody>
</table>
