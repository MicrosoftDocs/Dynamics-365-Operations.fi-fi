# <a name="create-a-record-template-to-facilitate-data-entry"></a>Tietuemallin luonti helpottamaan tietojen kirjaamista

[!include [banner](../../includes/banner.md)]

Tässä ohjeessa kerrotaan, miten voit luoda tietuemallin, jolloin usein käytettyjen kenttien arvoja ei tarvitse lisätä erikseen kussakin uudessa tietueessa. Tässä menettelyssä luodaan uusi tietue uusille käyttöomaisuuteen lisättäville kannettaville. Tässä menettelyssä käytetään USMF-malliyritystä.

1. Siirry siirtymisruudussa kohtaan **Moduulit > Käyttöomaisuuserät > Käyttöomaisuuserät > Käyttöomaisuuserät**.
2. Valitse **Uusi**.
3. Syötä tai valitse arvo **Käyttöomaisuusryhmä**-kentässä.
4. Kirjoita arvo **Nimi**-kenttään. Kirjoita esimerkiksi **Yritysliidi, kannettava**.  
5. Kirjoita arvo **Hakunimi**-kenttään. Kirjoita esimerkiksi **kannettava**.  
6. Laajenna **Teknistä tietoa** -osa.
7. Kirjoita arvot kenttiin **Valmistaja**, **Malli** ja **Mallin vuosi**.
8. Valitse toimintoruudussa **Asetukset**.
9. Valitse **Tietueen tiedot**.
10. Valitse **Käyttäjämalli**.
11. Kirjoita arvo **Nimi**-kenttään.
12. Kirjoita **Kuvaus**-kenttään arvo.
13. Valitse **OK**.
14. Valitse **Sulje**.
