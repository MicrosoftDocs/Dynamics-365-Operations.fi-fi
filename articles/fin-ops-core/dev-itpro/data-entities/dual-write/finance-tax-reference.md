# <a name="access-to-finance-and-tax-reference-data"></a>Talous- ja veroviitetietojen käyttöoikeus

[!include [banner](../../includes/banner.md)]

[!include [rename-banner](~/includes/cc-data-platform-banner.md)]

Jokainen yritys käyttää taloustietojen perusjoukkoa, kuten kirjanpidon vuosikalenteria, yrityksen käyttövaluuttaa, tilejä saapuvaa ja lähtevää rahaliikennettä varten, veroprosentteja ja maksusuorituksia. Nämä tiedot sijaitsevat Finance and Operations -sovelluksissa. Se kuitenkin näkyy Dataverselle, jotta asiakasvuorovaikutussovelluksilla voi olla yksi lähde taloushallinnon ja verojen tiedoille. Näin tiedot ovat yhdenmukaisia koko yritysympäristössä.

Talous- ja verotiedot integroidaan seuraavien määritysten avulla:

+ [Integroitu kirjanpito](ledger-mapping.md)
+ [Integroidut verojen päätiedot](tax-mapping.md)
