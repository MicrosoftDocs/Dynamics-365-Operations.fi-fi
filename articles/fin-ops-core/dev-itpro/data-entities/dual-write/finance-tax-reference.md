# <a name="access-to-finance-and-tax-reference-data"></a>Talous- ja veroviitetietojen käyttöoikeus

[!include [banner](../../includes/banner.md)]



Jokainen yritys käyttää taloustietojen perusjoukkoa, kuten kirjanpidon vuosikalenteria, yrityksen käyttövaluuttaa, tilejä saapuvaa ja lähtevää rahaliikennettä varten, veroprosentteja ja maksusuorituksia. Nämä tiedot sijaitsevat talous- ja toimintosovelluksissa. Se kuitenkin näkyy Dataverselle, jotta asiakasvuorovaikutussovelluksilla voi olla yksi lähde taloushallinnon ja verojen tiedoille. Näin tiedot ovat yhdenmukaisia koko yritysympäristössä.

Talous- ja verotiedot integroidaan seuraavien määritysten avulla:

+ [Integroitu kirjanpito](ledger-mapping.md)
+ [Integroidut verojen päätiedot](tax-mapping.md)
