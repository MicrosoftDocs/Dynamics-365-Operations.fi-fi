# <a name="access-to-finance-and-tax-reference-data"></a>Talous- ja veroviitetietojen käyttöoikeus

[!include [banner](../../includes/banner.md)]



Jokainen yritys käyttää taloustietojen perusjoukkoa, kuten kirjanpidon vuosikalenteria, yrityksen käyttövaluuttaa, tilejä saapuvaa ja lähtevää rahaliikennettä varten, veroprosentteja ja maksusuorituksia. Nämä tiedot sijaitsevat Finance and Operations -sovelluksissa. Ne ovat kuitenkin Common Data Servicen käytössä, joten mallipohjaisilla sovelluksilla Microsoft Dynamics 365:ssä voi olla yksi talous- ja verotietojen lähde. Näin tiedot ovat yhdenmukaisia koko yritysympäristössä. 

Talous- ja verotiedot integroidaan seuraavien määritysten avulla:

+ [Integroitu kirjanpito](ledger-mapping.md)
+ [Integroidut verojen päätiedot](tax-mapping.md)
