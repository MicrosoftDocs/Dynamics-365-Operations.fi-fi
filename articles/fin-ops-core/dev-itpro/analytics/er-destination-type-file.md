# <a name="file-destination"></a>Tiedostokohde

[!include [banner](../includes/banner.md)]

Voit määrittää tiedostokohteen kullekin lähteviä asiakirjoja luomaan määritetylle sähköisen raportoinnin (ER) muodon KANSIO- tai TIEDOSTO-komponentille. Luotu asiakirja on ladattavissa verkkoselaimesta kohteen asetusten mukaan.

Jos asetat **Kohdeasetukset** -sivulla **Käytössä**-kohdan arvoksi **Kyllä**, avaamis- tai tallennusikkuna avautuu, kun määritys on valmis.
