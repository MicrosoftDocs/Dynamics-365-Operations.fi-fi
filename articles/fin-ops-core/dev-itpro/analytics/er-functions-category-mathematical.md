# <a name="list-of-er-functions-in-the-mathematical-category"></a>Luettelo ER-funktioista matemaattisessa luokassa

[!include [banner](../includes/banner.md)]

Sähköisen raportoinnin (ER) matemaattisten funktioiden avulla voidaan tehdä useita yleisiä matemaattisia laskutoimituksia. Tässä aiheessa on yhteenveto näistä funktioista.

## <a name="list-of-supported-functions"></a>Luettelo tuetuista toiminnoista

| Toiminto | Kuvaus |
|----------|-------------|
| [Itseisarvo](er-functions-mathematical-abs.md)             | Tämä funktio palauttaa määritetyn luvun absoluuttisen arvon (jakojäännös) *todelliseksi* arvoksi. Toisin sanoen, se palauttaa luvun ilman etumerkkiä. |
| [Teho](er-functions-mathematical-power.md)         | Tämä funktio palauttaa *todellisen* arvon, joka ilmaisee määritetyn positiivisen luvun nostamisesta määritettyyn tehoon. |
| [Round](er-functions-mathematical-round.md)         | Tämä funktio palauttaa määritetyn numeron *Todellisen* arvon sen jälkeen, kun se on pyöristetty määritettyyn määrään desimaaleja. |
| [RoundDown](er-functions-mathematical-rounddown.md) | Tämä funktio palauttaa määritetyn numeron *Todellisen* arvon sen jälkeen, kun se on pyöristetty alaspäin määritettyyn määrään desimaaleja. |
| [RoundUp](er-functions-mathematical-roundup.md)     | Tämä funktio palauttaa määritetyn numeron *Todellisen* arvon sen jälkeen, kun se on pyöristetty ylöspäin määritettyyn määrään desimaaleja. |
