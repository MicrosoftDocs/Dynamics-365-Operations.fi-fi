# <a name="ScreenDestinationType">Näyttökohde</a>

[!include [banner](../includes/banner.md)]

Voit määrittää näyttökohteen kullekin lähteviä asiakirjoja luomaan määritetylle sähköisen raportoinnin (ER) muodon kansio- tai tiedostokomponentille. Luotu asiakirja avataan esikatselua varten erillisessä selainvälilehdessä kohteen asetusten mukaan.

Jos määrität **Käytössä**-kentän arvoksi **Kyllä**, tulosteen esikatselu luodaan. Voit tarkastella joitakin tiedostotyyppejä, kuten XML, TXT tai PDF, suoraan selainikkunassa. Muiden tiedostotyyppien, kuten Microsoft Excel tai Word, osalta käytetään Microsoft Office Online -palvelua.
