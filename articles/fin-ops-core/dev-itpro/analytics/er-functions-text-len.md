# <a name="LEN">LEN ER -funktio</a>

[!include [banner](../includes/banner.md)]

`LEN`-funktio palauttaa määritetystä luettelosta *Kokonaisluku*-arvon, joka esittää merkkien määrän määritetyssä merkkijonossa.

## <a name="syntax"></a>Syntaksi

```vb
LEN (text)
```

## <a name="arguments"></a>Argumentit

`text`: *Merkkijono*

*Merkkijono*-arvo, joka määrittää tekstin.

## <a name="return-values"></a>Palautusarvot

*Kokonaisluku*

Tuloksena oleva numeroarvo.

## <a name="example"></a>Esimerkki

`LEN ("Sample")` palauttaa **6**.
