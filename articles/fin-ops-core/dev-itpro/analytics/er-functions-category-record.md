# <a name="list-of-er-functions-in-the-record-category"></a>Luettelo ER-funktioista tietueluokassa

[!include [banner](../includes/banner.md)]

Sähköisen raportoinnin (ER) tietuefunktioiden avulla voidaan poimia tietoja *Säilö (tietue)* -tietotyypin tietolähteistä ja suorittaa niitä. Tässä aiheessa on yhteenveto näistä funktioista.

## <a name="list-of-supported-functions"></a>Luettelo tuetuista toiminnoista

| Toiminto | Kuvaus |
|----------|-------------|
| [NullContainer](er-functions-record-nullcontainer.md) | Tämä funktio palauttaa tyhjäarvo *-säilön (tietueen)* arvon, jolla on sama rakenne kuin määritetyllä tietueluettelolla tai tietueella. |
| [EmptyRecord](er-functions-record-emptyrecord.md) | Tämä funktio palauttaa tyhjäarvo *-säilön (tietueen)* arvon, jolla on sama rakenne kuin määritetyllä tietueluettelolla tai tietueella. |
