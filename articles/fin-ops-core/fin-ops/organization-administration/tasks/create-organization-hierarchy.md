# <a name="create-an-organization-hierarchy"></a>Luo organisaatiohierarkia

[!include [banner](../../includes/banner.md)]

Luo organisaatiohierarkia seuraavien ohjeiden mukaan. Voit käyttää organisaatiohierarkioita liiketoiminnan tarkastelemiseen ja raportointiin eri näkökulmista. Voit esimerkiksi määrittää yhden hierarkian veroraportteja tai oikeudellista tai lakisääteistä raportointia varten. Voit määrittää toisen hierarkian raportoimaan sellaiset taloustiedot, joiden raportointia lainsäädäntö ei edellytä mutta joita käytetään sisäisessä raportoinnissa. 

Ennen organisaatiohierarkian luomista sinun on luotava organisaatiot. Lisätietoja on tehtävissä "Yrityksen luominen" tai "Toimintayksikön luominen". Voit myös luoda osastoja ja tiimejä. 

Tämän menettelyn luomisessa käytetty esittely-yritys on USMF.

## <a name="create-a-hierarchy"></a>Luo hierarkia
1. Valitse **Siirtymisruutu > Moduuli > Organisaation hallinto > Organisaatiot >Organisaatiohierarkiat**.
2. Napsauta **Toimintoruudussa** **Uusi**.
3. Kirjoita arvo **Nimi**-kenttään.
4. Valitse **Tarkoitus** -osassa **Määritä tarkoitus**.
5. Etsi haluamasi tietue luettelosta ja valitse se. Valitse tarkoitus määritettäväksi organisaatiohierarkiaan.  
6. Valitse **Määritetyt hierarkiat** -osassa **Lisää**.
7. Merkitse valittu rivi luettelossa. Etsi juuri luomasi hierarkia.  
8. Valitse **OK**.

## <a name="add-organizations-to-the-hierarchy"></a>Organisaatioiden lisääminen hierarkiaan
1. Etsi haluamasi tietue luettelosta ja valitse se. Valitse hierarkia.  
2. Valitse **Määritetyt hierarkiat** -osassa **Näytä hierarkia**.
    - Lisää organisaatioita tarvittaessa.  
    - Lisää organisaatio napsauttamalla **Muokkaa** ja sitten **Lisää**. Kun olet tehnyt haluamasi muutokset, voit tallentaa luonnoksena (**Tallenna**) ja/tai julkaista muutokset (**Julkaise**).
