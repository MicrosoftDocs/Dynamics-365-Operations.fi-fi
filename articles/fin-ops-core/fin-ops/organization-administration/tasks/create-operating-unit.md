# <a name="create-an-operating-unit"></a><span data-ttu-id="0762d-103">Luo toimintayksikkö</span><span class="sxs-lookup"><span data-stu-id="0762d-103">Create an operating unit</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="0762d-104">Toimintayksikkö on organisaatio, jota käytetään jakamaan liiketoiminnan taloudellisten resurssien hallinta ja operationaaliset prosessit.</span><span class="sxs-lookup"><span data-stu-id="0762d-104">An operating unit is an organization that is used to divide the control of economic resources and operational processes in a business.</span></span> <span data-ttu-id="0762d-105">Toimintayksikön henkilöiden vastuulla on rajattujen resurssien maksimointi, prosessien tehostaminen ja suorituskyvyn raportointi.</span><span class="sxs-lookup"><span data-stu-id="0762d-105">People in an operating unit have a duty to maximize the use of scarce resources, improve processes, and account for their performance.</span></span> <span data-ttu-id="0762d-106">Toimintayksiköihin kuuluvat kustannuspaikat, liiketoimintayksiköt, osastot ja arvovirrat.</span><span class="sxs-lookup"><span data-stu-id="0762d-106">The types of operating units include cost centers, business units, departments, and value streams.</span></span> <span data-ttu-id="0762d-107">Luo toimintayksikkö seuraavien ohjeiden mukaan.</span><span class="sxs-lookup"><span data-stu-id="0762d-107">Use the following procedure to create an operating unit.</span></span> <span data-ttu-id="0762d-108">Tämän menettelyn luomisessa käytetty esittely-yritys on USMF.</span><span class="sxs-lookup"><span data-stu-id="0762d-108">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="0762d-109">Valitse **Siirtymisruutu > Moduuli > Organisaation hallinto > Organisaatiot >Käyttöyksiköt**.</span><span class="sxs-lookup"><span data-stu-id="0762d-109">Go to **Navigation pane > Modules > Organization administration > Organizations > Operating units.**</span></span>
2. <span data-ttu-id="0762d-110">Avaa valintaikkuna valitsemalla **Uusi**.</span><span class="sxs-lookup"><span data-stu-id="0762d-110">Click **New** to open the drop dialog.</span></span>
3. <span data-ttu-id="0762d-111">Etsi haluamasi tietue luettelosta ja valitse se.</span><span class="sxs-lookup"><span data-stu-id="0762d-111">In the list, find and select the desired record.</span></span> <span data-ttu-id="0762d-112">Valitse luotavan toimintoyksikön tyyppi.</span><span class="sxs-lookup"><span data-stu-id="0762d-112">Select the type of operating unit you want to create.</span></span>  
4. <span data-ttu-id="0762d-113">Napsauta luettelossa valitulla rivillä olevaa linkkiä.</span><span class="sxs-lookup"><span data-stu-id="0762d-113">In the list, click the link in the selected row.</span></span>
5. <span data-ttu-id="0762d-114">Kirjoita arvo **Nimi**-kenttään.</span><span class="sxs-lookup"><span data-stu-id="0762d-114">In the **Name** field, type a value.</span></span>
    + <span data-ttu-id="0762d-115">Laajenna **Yleiset**-osa tarvittaessa.</span><span class="sxs-lookup"><span data-stu-id="0762d-115">Expand the **General** section, if necessary.</span></span>  
    + <span data-ttu-id="0762d-116">Ilmoita toimintayksikön yleistiedot, kuten tunnusnumero, DUNS-numero ja päällikkö.</span><span class="sxs-lookup"><span data-stu-id="0762d-116">Provide general information about the operating unit, such as an identification number, DUNS number, and manager.</span></span>    
    + <span data-ttu-id="0762d-117">Laajenna **Osoitteet**-osa tarvittaessa.</span><span class="sxs-lookup"><span data-stu-id="0762d-117">Expand the **Addresses** section, if necessary.</span></span>  
    + <span data-ttu-id="0762d-118">Määritä osoitetiedot, kuten katuosoite ja -numero, postinumero ja postitoimipaikka.</span><span class="sxs-lookup"><span data-stu-id="0762d-118">Enter address information, such as the street name and number, postal code, and city.</span></span> <span data-ttu-id="0762d-119">Anna uusi osoitetietue valitsemalla **Lisää** tai muokkaa aiemmin luotua osoitetietuetta valitsemalla Muokkaa.</span><span class="sxs-lookup"><span data-stu-id="0762d-119">Click **Add** to enter a new address record, or click Edit to modify an existing address record.</span></span>   
    + <span data-ttu-id="0762d-120">Laajenna **Yhteystiedot**-osa tarvittaessa.</span><span class="sxs-lookup"><span data-stu-id="0762d-120">Expand the **Contact information** section, if necessary.</span></span>  
    + <span data-ttu-id="0762d-121">Ilmoita viestintämenetelmien tiedot, kuten sähköpostiosoitteet, URL-osoitteet ja puhelinnumerot.</span><span class="sxs-lookup"><span data-stu-id="0762d-121">Enter information about methods of communication, such as email addresses, URLs, and telephone numbers.</span></span> <span data-ttu-id="0762d-122">Voit syöttää uuden viestintätietueen valitsemalla Uusi.</span><span class="sxs-lookup"><span data-stu-id="0762d-122">To enter a new communication record, click New.</span></span> <span data-ttu-id="0762d-123">Voit muokata aiemmin luodut viestintätietue valitsemalla **Lisäasetukset > Lisäasetukset**.</span><span class="sxs-lookup"><span data-stu-id="0762d-123">To modify an existing communication record, click **More options > Advanced**.</span></span>   
6. <span data-ttu-id="0762d-124">Voit halutessasi muuttaa **Toimintayksikön numero** -asetusta tarpeen mukaan.</span><span class="sxs-lookup"><span data-stu-id="0762d-124">Optionally, change the **Operating unit number** as needed.</span></span> <span data-ttu-id="0762d-125">Huomaa, että tämä numero on vastaavan **Osapuoli**-tietueen yksilöivä tunnus, eikä se voi olla sama kuin mikä tahansa muu toimintayksikkö.</span><span class="sxs-lookup"><span data-stu-id="0762d-125">Note that this number is a unique idenitifier for the correspondng **Party** record and cannot be the same as any other operating unit.</span></span>
7. <span data-ttu-id="0762d-126">Valitse **Tallenna**.</span><span class="sxs-lookup"><span data-stu-id="0762d-126">Select **Save**.</span></span>
