[!include [banner](../../includes/banner.md)]

<span data-ttu-id="cd210-104">Voit määrittää tällä menettelyllä yleisen osoitekirjan oletusarvot ja suojauskäytännöt.</span><span class="sxs-lookup"><span data-stu-id="cd210-104">Use this procedure to set the default values and security policies for the global address book.</span></span> 

<span data-ttu-id="cd210-105">Tämän tehtävän luomisessa käytetty esittely-yritys on USMF.</span><span class="sxs-lookup"><span data-stu-id="cd210-105">The demo data company used to create this task is USMF.</span></span> <span data-ttu-id="cd210-106">Tämä tehtävä on tarkoitettu suunnittelu- ja määritysryhmälle.</span><span class="sxs-lookup"><span data-stu-id="cd210-106">This task is intended for the Planning and configuration team.</span></span>

3. <span data-ttu-id="cd210-109">Valitse **Poista osapuolet, joilla ei ole rooleja** -kohdassa, jos haluat poistaa osapuolia, joille ei ole määritetty roolia.</span><span class="sxs-lookup"><span data-stu-id="cd210-109">In **Delete parties with no roles**, select whether to delete parties with that have not been assigned a role.</span></span>

6. <span data-ttu-id="cd210-112">Valitse **Tarkista yksilöllinen DUNS-numero** -kohdassa, tarkistetaanko ainutlaatuiset DUNS-numerot.</span><span class="sxs-lookup"><span data-stu-id="cd210-112">In **Check for unique DUNS number**, select whether to check for unique DUNS numbers.</span></span>
7. <span data-ttu-id="cd210-113">Valitse **Osapuoli**-kentässä vaihtoehto.</span><span class="sxs-lookup"><span data-stu-id="cd210-113">In the **Party** field, select an option.</span></span>
8. <span data-ttu-id="cd210-114">Valitse **Asiakas**-kentässä vaihtoehto.</span><span class="sxs-lookup"><span data-stu-id="cd210-114">In the **Customer** field, select an option.</span></span>
9. <span data-ttu-id="cd210-115">Valitse **Toimittaja**-kentässä vaihtoehto.</span><span class="sxs-lookup"><span data-stu-id="cd210-115">In the **Vendor** field, select an option.</span></span>
10. <span data-ttu-id="cd210-116">Valitse **Prospekti**-kentässä vaihtoehto.</span><span class="sxs-lookup"><span data-stu-id="cd210-116">In the **Prospect** field, select an option.</span></span>
11. <span data-ttu-id="cd210-117">Valitse **Kilpailija**-kentässä vaihtoehto.</span><span class="sxs-lookup"><span data-stu-id="cd210-117">In the **Competitor** field, select an option.</span></span>
12. <span data-ttu-id="cd210-118">Valitse **Yksityisen sijainnin suojaus** -välilehti.</span><span class="sxs-lookup"><span data-stu-id="cd210-118">Click the **Private location security** tab.</span></span>
13. <span data-ttu-id="cd210-119">Etsi haluamasi tietue luettelosta ja valitse se.</span><span class="sxs-lookup"><span data-stu-id="cd210-119">In the list, find and select the desired record.</span></span> <span data-ttu-id="cd210-120">Voit lisätä useita rooleja **Valitut roolit** -ruutuun painamalla Vaihto-näppäintä ja lisäämällä sitten valitut roolit nuolta napsauttamalla.</span><span class="sxs-lookup"><span data-stu-id="cd210-120">Press the Shift key to select multiple roles to add to the **Selected roles** pane and then click the arrow to add the selected roles.</span></span>  
14. <span data-ttu-id="cd210-121">Valitse **Tallenna**.</span><span class="sxs-lookup"><span data-stu-id="cd210-121">Click **Save**.</span></span>
