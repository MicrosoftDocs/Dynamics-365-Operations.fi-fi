6. <span data-ttu-id="76bfe-118">Laajenna **Segmentit**-osa.</span><span class="sxs-lookup"><span data-stu-id="76bfe-118">Expand the **Segments** section.</span></span>
    - <span data-ttu-id="76bfe-119">Määritä numerosarjalle muoto lisäämällä, poistamalla ja uudelleenjärjestelemällä segmenttejä.</span><span class="sxs-lookup"><span data-stu-id="76bfe-119">Define the format for the number sequence by adding, removing, and rearranging segments.</span></span>  
    - <span data-ttu-id="76bfe-120">Kaikkien alueiden numerosarjat voivat sisältää *kiinteitä segmenttejä* ja *aakkosnumeerisia segmenttejä*.</span><span class="sxs-lookup"><span <span data-ttu-id="76bfe-122">Tämän segmenttityypin avulla voit lisätä väliviivan tai muun erottimen numerosarjasegmenttien välille.</span><span class="sxs-lookup"><span data-stu-id="76bfe-122">Use ja et-merkkiä (&) osoittamaan kasvavia kirjaimia.</span><span class="sxs-lookup"><span data-stu-id="76bfe-125">Use a number sign (#) to indicate incrementing numbers and an ampersand (&) to indicate <span data-ttu-id="76bfe-129">Vaikka et sisällytä alueen segmenttejä muotoon, valitun viitteen numerot luodaan edelleen aluekohtaisesti.</span><span class="sxs-lookup"><span
