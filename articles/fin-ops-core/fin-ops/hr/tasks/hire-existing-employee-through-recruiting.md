# <a name="hire-existing-employees-through-recruitment"></a><span data-ttu-id="594ef-103">Nykyisten työntekijöiden työhönotto rekrytoinnin kautta</span><span class="sxs-lookup"><span data-stu-id="594ef-103">Hire existing employees through recruitment</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="594ef-104">Joskus avoimiin toimiin palkataan ehdokkaita, jotka ovat jo organisaatiossa töissä.</span><span class="sxs-lookup"><span data-stu-id="594ef-104">Sometimes open positions can be filled by candidates who are already employees in your organization.</span></span> <span data-ttu-id="594ef-105">Tässä menettelyssä kerrotaan nykyisen työntekijän palkkaamisesta työhönottoprosessin kautta.</span><span class="sxs-lookup"><span data-stu-id="594ef-105">This procedure walks through the steps of hiring an existing employee through the recruiting process.</span></span> <span data-ttu-id="594ef-106">Tässä menettelyssä työhönottoprojekti on jo määritetty ja nykyinen työntekijä on jo lähettänyt hakemuksen työhönottoprojektiin.</span><span class="sxs-lookup"><span data-stu-id="594ef-106">In this procedure, a recruitment project has already been set up, and an existing employee has already submitted an application for the recruitment project.</span></span> <span data-ttu-id="594ef-107">Tämän menettelyn luomisessa käytetty esittely-yritys on USMF.</span><span class="sxs-lookup"><span data-stu-id="594ef-107">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="594ef-108">Siirry kohtaan Henkilöstöhallinto > Työhönotto > Hakemukset > Hakemukset.</span><span class="sxs-lookup"><span data-stu-id="594ef-108">Go to Human resources > Recruitment > Applications > Applications.</span></span>
2. <span data-ttu-id="594ef-109">Etsi luettelosta palkattavan työntekijän hakemus.</span><span class="sxs-lookup"><span data-stu-id="594ef-109">In the list, find application for the employee that you would like to hire.</span></span> <span data-ttu-id="594ef-110">Esimerkki: 00002 John Emory</span><span class="sxs-lookup"><span data-stu-id="594ef-110">Example:  00002  John Emory</span></span>
3. <span data-ttu-id="594ef-111">Valitse Hakemuksen tila.</span><span class="sxs-lookup"><span data-stu-id="594ef-111">Click Application status.</span></span>
    * <span data-ttu-id="594ef-112">Sovelluksen tila ilmaisee, missä vaiheessa työhönottoprosessia hakemus on.</span><span class="sxs-lookup"><span data-stu-id="594ef-112">The application status indicates where an application is at in the recruitment process.</span></span>  <span data-ttu-id="594ef-113">Kaikki nämä vaiheet ovat valinnaisia.</span><span class="sxs-lookup"><span data-stu-id="594ef-113">Each of these steps is optional.</span></span> <span data-ttu-id="594ef-114">Yleensä hakemuksen tila vaihtelevat seuraavasti: vastaanotettu, vahvistettu ja haastateltu.</span><span class="sxs-lookup"><span data-stu-id="594ef-114">Typically, an application would move the statuses in the following order:  received, confirmed, and interviewed.</span></span> <span data-ttu-id="594ef-115">Haastatteluprosessin jälkeen voidaan tehdä palkkauspäätös.</span><span class="sxs-lookup"><span data-stu-id="594ef-115">After the interview process, a hiring decision would be made.</span></span>  
4. <span data-ttu-id="594ef-116">Valitse Muuta toimea.</span><span class="sxs-lookup"><span data-stu-id="594ef-116">Click Change position.</span></span>
5. <span data-ttu-id="594ef-117">Valitse toimi, johon työntekijä palkataan.</span><span class="sxs-lookup"><span data-stu-id="594ef-117">Select the position that you are hiring the employee into.</span></span>
6. <span data-ttu-id="594ef-118">Syötä Uuden toimeksiannon alkupäivämäärä -kenttään päivämäärä, jona työntekijä aloittaa uudessa toimessa työskentelyn.</span><span class="sxs-lookup"><span data-stu-id="594ef-118">In the New Assignment Start Date field, enter the date that the employee will begin working in the new position.</span></span>  
7. <span data-ttu-id="594ef-119">Syötä Uuden toimeksiannon päättymispäivämäärä -kenttään päivämäärä, jona työntekijä päättää uudessa toimessa työskentelyn.</span><span class="sxs-lookup"><span data-stu-id="594ef-119">In the Assignment end date, enter the date that the employee will stop working in their current position.</span></span>
    * <span data-ttu-id="594ef-120">Uuden ja vanhan toimen aloituspäivä ja päättymispäivä voivat olla päällekkäisiä.</span><span class="sxs-lookup"><span data-stu-id="594ef-120">The starting date for the new position and the ending date of the old position may overlap.</span></span> <span data-ttu-id="594ef-121">Näin voi tapahtua silloin, kun henkilöllä on tehtäviä molemmissa toimissa siirtojakson aikana.</span><span class="sxs-lookup"><span data-stu-id="594ef-121">This can happen when a person is performing duties for both positions during a transition period.</span></span>  
8. <span data-ttu-id="594ef-122">Vaihtoehtoisesti voit valita syykoodin.</span><span class="sxs-lookup"><span data-stu-id="594ef-122">Optionally, you can select a reason code.</span></span> <span data-ttu-id="594ef-123">Esimerkki: Uudelleenorganisointi</span><span class="sxs-lookup"><span data-stu-id="594ef-123">Example: Reorganization</span></span>
9. <span data-ttu-id="594ef-124">Valitse Muuta toimea.</span><span class="sxs-lookup"><span data-stu-id="594ef-124">Click Change position.</span></span>
    * <span data-ttu-id="594ef-125">Voit myös muuttaa kompensaatiota täksi ajaksi.</span><span class="sxs-lookup"><span data-stu-id="594ef-125">You may also change compensation at this time.</span></span> <span data-ttu-id="594ef-126">Jos et määritä kompensaatiota nyt, voit muuttaa sitä siirtymällä työntekijän lomakkeeseen, valitsemalla Kompensaatio-välilehden ja valitsemalla Kiinteä suunnitelma.</span><span class="sxs-lookup"><span data-stu-id="594ef-126">If you do not assign compensation at this time, you can change it by going to the worker form, selecting the Compensation tab, and choosing 'Fixed Plan'.</span></span> <span data-ttu-id="594ef-127">Kun valitset Muuta toimea, hakemuksen tilaksi tulee Toimessa.</span><span class="sxs-lookup"><span data-stu-id="594ef-127">After you select 'Change position', the status on the application will be updated to 'Employed'.</span></span>
