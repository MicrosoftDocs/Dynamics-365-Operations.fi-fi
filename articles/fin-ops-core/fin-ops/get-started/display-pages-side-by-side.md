# <a name="show-pages-side-by-side-using-the-open-in-new-window-feature"></a><span data-ttu-id="a4202-103">Sivujen näyttäminen rinnakkain Avaa uudessa ikkunassa -toiminnon avulla</span><span class="sxs-lookup"><span data-stu-id="a4202-103">Show pages side-by-side using the Open in new window feature</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="a4202-104">Tässä artikkelissa käsitellään sivujen näyttämistä rinnakkain.</span><span class="sxs-lookup"><span data-stu-id="a4202-104">This article explains how to display pages side by side.</span></span>

<span data-ttu-id="a4202-105">Joskus tehtäviä halutaan suorittaa nopeasti tarkastelemalla useita sivuja rinnakkain.</span><span class="sxs-lookup"><span data-stu-id="a4202-105">You may want to view multiple pages side by side to complete tasks quickly.</span></span> <span data-ttu-id="a4202-106">Voit esimerkiksi haluta tarkistaa usean kirjauskansion rivejä tai syöttää niitä useaan kirjauskansioon.</span><span class="sxs-lookup"><span data-stu-id="a4202-106">As an example, you might want to validate or enter lines in more than one journal.</span></span> <span data-ttu-id="a4202-107">Yleensä oikeellisuustarkistuksen tekeminen tai rivien syöttäminen useampaan kuin yhteen kirjauskansioon vaatii palaamisen kirjauskansioluettelon sisältävälle sivulle ja takaisin tietyn kirjauskansion sisältävälle sivulle.</span><span class="sxs-lookup"><span data-stu-id="a4202-107">Typically, to validate or enter lines in more than one journal, you would have to go back and forth between the page that displays a list of journals, and the page that displays lines for a given journal.</span></span> <span data-ttu-id="a4202-108">**Avaa uudessa ikkunassa** -toiminnon avulla voit kuitenkin näyttää nämä sivut rinnakkain, jolloin tehtävien suorittaminen nopeutuu.</span><span class="sxs-lookup"><span data-stu-id="a4202-108">However, the **Open in new window** feature enables you to display these pages side-by-side so that you can perform your tasks quickly.</span></span>

<span data-ttu-id="a4202-109">Kun haluat tarkastella rivejä, voit valita **Avaa uudessa ikkunassa** -kuvakkeen.</span><span class="sxs-lookup"><span data-stu-id="a4202-109">Continuing with the example mentioned above, when viewing the lines, you can click the **Open in new window** icon.</span></span>

<span data-ttu-id="a4202-110">[![Napauta Avaa uudessa ikkunassa -kuvaketta.](./media/open-in-new-window-icon.png)](./media/open-in-new-window-icon.png)</span><span class="sxs-lookup"><span data-stu-id="a4202-110">[![Click the Open in new window icon.](./media/open-in-new-window-icon.png)](./media/open-in-new-window-icon.png)</span></span>

<span data-ttu-id="a4202-111">Kun **Avaa uudessa ikkunassa** -kuvake valitaan, rivien sivu avataan uudessa selainikkunassa ponnahdusikkunana. Tämän jälkeen siirrytään takaisin alkuperäisen selaimen aiemmalle sivulle, joka on kirjauskansioluettelon sisältävä sivu.</span><span class="sxs-lookup"><span data-stu-id="a4202-111">Clicking the **Open in new window** icon opens the lines page in a new, pop-up browser, and then navigates the original browser back in history to the page that displayed the list of journals.</span></span> <span data-ttu-id="a4202-112">Tämän jälkeen sivut voidaan näyttää rinnakkain.</span><span class="sxs-lookup"><span data-stu-id="a4202-112">You can then display both pages side by side.</span></span> <span data-ttu-id="a4202-113">Kirjauskansion tarkastelemisen jälkeen voit muuttaa kirjauskansiovalinnan kirjauskansioluettelosivulla. Tällöin ponnahdusikkunan rivien sivulla näytetään automaattisesti juuri valitun kirjauskansion rivit.</span><span class="sxs-lookup"><span data-stu-id="a4202-113">After viewing a journal, you can change the selected journal on the journal list page, and the lines page in the pop-up window will automatically display the lines of the newly selected journal.</span></span>

<span data-ttu-id="a4202-114">[![Sivut voidaan näyttää rinnakkain.](./media/pages-show-side-by-side.png)](./media/pages-show-side-by-side.png)</span><span class="sxs-lookup"><span data-stu-id="a4202-114">[![You can display pages side-by-side.](./media/pages-show-side-by-side.png)](./media/pages-show-side-by-side.png)</span></span>

<span data-ttu-id="a4202-115">Dynaaminen linkitys ja päivittäminen tapahtuu näiden sivujen taustalla olevien tietojen suhteiden vuoksi.</span><span class="sxs-lookup"><span data-stu-id="a4202-115">The dynamic linking and refreshing happens due to the relations that exist between the data that is backing these pages.</span></span> <span data-ttu-id="a4202-116">Jos järjestelmä ei ota huomioon tietojen välisiä suhteita, ponnahdusikkunaa ei päivitetä automaattisesti alkuperäisessä ikkunassa tehtävien muutosten perusteella.</span><span class="sxs-lookup"><span data-stu-id="a4202-116">If the system is not aware of the relation between the data, the pop-up window will not refresh automatically in response to a change in the window it originated from.</span></span>

<span data-ttu-id="a4202-117">Joillakin sivuilla on useita näkymiä, kuten ruudukko-, otsikko- ja tietonäkymä.</span><span class="sxs-lookup"><span data-stu-id="a4202-117">Some pages have multiple views such as the Grid view, Header view, and Details view.</span></span> <span data-ttu-id="a4202-118">**Avaa uudessa ikkunassa** -kuvake avaa koko sivun uudessa selainikkunassa.</span><span class="sxs-lookup"><span data-stu-id="a4202-118">The **Open in new window** icon causes the entire page to open in the new browser window.</span></span> <span data-ttu-id="a4202-119">Tämän vuoksi **Avaa uudessa ikkunassa** -toiminnon avulla kahta saman sivun näkymää ei voi näyttää rinnakkain.</span><span class="sxs-lookup"><span data-stu-id="a4202-119">Therefore, you cannot keep two views of the same page side by side using the **Open in new window** feature.</span></span> <span data-ttu-id="a4202-120">Melkein kaikilla tällaisilla sivuilla on siirtymisluettelo, jonka avulla tietueiden välillä voi siirtyä. Tällöin käyttökokemus on samanlainen.</span><span class="sxs-lookup"><span data-stu-id="a4202-120">Almost all such pages have a navigation list that you can use to switch between records and achieve a similar experience.</span></span>

<span data-ttu-id="a4202-121">Määritä selaimen ponnahdusikkunoiden estotoiminto sallimaan ponnahdusikkunat sivuston URL-osoitteesta, ennen kuin käytät **Avaa uudessa ikkunassa** -toimintoa.</span><span class="sxs-lookup"><span data-stu-id="a4202-121">Before using the **Open in new window** feature, you should configure your browser's pop-up blocker to allow pop-ups from the URL of the site.</span></span> <span data-ttu-id="a4202-122">Ponnahdusikkunat voidaan sallia esimerkiksi osoitteesta \*.dynamics.com.</span><span class="sxs-lookup"><span data-stu-id="a4202-122">As an example, you could allow pop-ups from "\*.dynamics.com".</span></span>

<span data-ttu-id="a4202-123">**Avaa uudessa ikkunassa** -toiminto on käytettävissä vain, kun ikkunassa on avoinna useita sivuja.</span><span class="sxs-lookup"><span data-stu-id="a4202-123">The **Open in new window** feature is only available when there is more than one page open in the window.</span></span> <span data-ttu-id="a4202-124">Ponnahdusikkuna suljetaan automaattisesti, kun sivuja ei ole enää avoinna (eli sitten, kun ikkunan viimeinen sivu suljetaan).</span><span class="sxs-lookup"><span data-stu-id="a4202-124">Also, the pop-up window automatically closes when there are no more pages open (that is, when you close the last page in that window).</span></span> <span data-ttu-id="a4202-125">Järjestelmä sulkee sivut myös silloin, kun siirryt sovelluksen toiselle alueelle.</span><span class="sxs-lookup"><span data-stu-id="a4202-125">The system also closes open pages when you navigate to a different area in the application.</span></span> <span data-ttu-id="a4202-126">Niinpä jos ponnahdusikkunoita on avoinna silloin, kun siirryt sovelluksen toiselle alueelle, ponnahdusikkunat sulkeutuvat automaattisesti, koska järjestelmä sulki ikkunoissa olevat sivut.</span><span class="sxs-lookup"><span data-stu-id="a4202-126">Therefore, if you have pop-up windows open and navigate to a different area in the application, the pop-up windows close automatically because the system closed the pages in those windows.</span></span>

<span data-ttu-id="a4202-127">Ponnahdusikkunoiden yläpalkeissa on tietoja yrityksestä, jolle sivu on avattu. Tiedot ovat vain luku -muodossa.</span><span class="sxs-lookup"><span data-stu-id="a4202-127">The top bar in the pop-up windows displays information about the company the page was opened in and is read-only.</span></span> <span data-ttu-id="a4202-128">Ponnahdusikkunat ovat riippuvaisia myös selainikkunasta.</span><span class="sxs-lookup"><span data-stu-id="a4202-128">The pop-up windows also rely on the main browser window.</span></span> <span data-ttu-id="a4202-129">Jos pääikkuna suljetaan tai päivitetään, kaikki avoimet ponnahdusikkunat siirtyvät vain luku -tilaan.</span><span class="sxs-lookup"><span data-stu-id="a4202-129">If the main window is closed or refreshed, all open pop-up windows will become read only.</span></span> <span data-ttu-id="a4202-130">Jos näin tapahtuu, voit yhä tarkastella ikkunoiden tietoja, mutta et voi käsitellä niitä.</span><span class="sxs-lookup"><span data-stu-id="a4202-130">If this situation occurs, you can still view the information in these windows, but you will not be able to interact with it.</span></span>
