<span data-ttu-id="efa39-104">Voit luoda työtilausraportin, jossa näkyy yksityiskohtaisia tietoja yhdestä tai useammasta valitsemastasi työtilauksesta.</span><span class="sxs-lookup"><span data-stu-id="efa39-104">You can generate a work order report that shows detailed information about one or more work orders that you select.</span></span>
