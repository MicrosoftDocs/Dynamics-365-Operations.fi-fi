<span data-ttu-id="807ba-104">Tässä aiheessa on yleiskatsaus toiminnallisista sijainneista resurssien hallinnassa.</span><span class="sxs-lookup"><span data-stu-id="807ba-104">This topic provides an overview of functional locations in Asset Management.</span></span> <span data-ttu-id="807ba-105">Toiminnalliset sijainnit ovat teknisen rakenteen elementtejä, kuten järjestelmän toiminnallisia yksiköitä.</span><span class="sxs-lookup"><span data-stu-id="807ba-105">Functional locations are elements of a technical structure, such as the functional units in a system.</span></span> <span data-ttu-id="807ba-106">Toiminnalliset sijainnit luodaan hierarkkisesti ja niihin asennetaan resursseja.</span><span class="sxs-lookup"><span data-stu-id="807ba-106">Functional locations are created hierarchically, and you install assets on them.</span></span> <span data-ttu-id="807ba-107">Yrityksen toiminnallisten sijaintien asetukset määräytyvät yrityksen tarpeiden mukaan.</span><span class="sxs-lookup"><span data-stu-id="807ba-107">The setup of functional locations in your company depends on the company's requirements.</span></span>

<span data-ttu-id="807ba-108">Seuraavassa on esimerkkejä siitä, miten voit käyttää toiminnallisia sijainteja:</span><span class="sxs-lookup"><span data-stu-id="807ba-108">Here are some examples of how you can use functional locations:</span></span>

- <span data-ttu-id="807ba-109">**Toiminnallinen** – toiminnalliset sijainnit voivat olla käyttäjälähtöisiä ja käyttää hallitsemaan resursseja, joilla on samanlainen käyttäytyminen.</span><span class="sxs-lookup"><span data-stu-id="807ba-109">**Functional** – The functional locations can be user-oriented and used to manage assets that have similar behavior.</span></span>

<span data-ttu-id="807ba-112">Jokaista toiminnallista sijaintia hallitaan itsenäisesti resurssien hallinnassa.</span><span class="sxs-lookup"><span data-stu-id="807ba-112">Each functional location is managed independently in Asset Management.</span></span> <span data-ttu-id="807ba-113">Tässä on muutamia toiminnallisten siajintien hyödyllisiä ominaisuuksia:</span><span class="sxs-lookup"><span data-stu-id="807ba-113">Here are some of the useful features of functional locations:</span></span>

- <span data-ttu-id="807ba-115">Määritä resurssin määrittelyvaatimukset.</span><span class="sxs-lookup"><span data-stu-id="807ba-115">Set up asset specification requirements.</span></span>
- <span data-ttu-id="807ba-116">Määritä ennaltaehkäisevän ja reaktiivisen kunnossapidon huoltojaksot.</span><span class="sxs-lookup"><span data-stu-id="807ba-116">Set up maintenance sequences for preventive and reactive maintenance.</span></span>
- <span data-ttu-id="807ba-117">Hallitse asennettuja resursseja.</span><span class="sxs-lookup"><span data-stu-id="807ba-117">Manage installed assets.</span></span>
- <span data-ttu-id="807ba-118">Voit seurata asennettuun omaisuuteen liittyviä aktiivisia pyyntöjä ja työtilauksia.</span><span class="sxs-lookup"><span data-stu-id="807ba-118">Track active requests and work orders that are related to installed assets.</span></span>

<span data-ttu-id="807ba-121">Toiminnalliset sijainnit tarjoavat resurssien jäljitettävyyden pyyntöjen, työtilausten, vikarekisteröintien, kunnon arviointien, tuotannon keskeytyksen rekisteröintien ja resurssien laskurien rekisteröintien suhteen.</span><span class="sxs-lookup"><span data-stu-id="807ba-121">Functional locations provide traceability of assets in relation to requests, work orders, fault registrations, condition assessments, production stop registrations, and asset counter registrations.</span></span>

> <span data-ttu-id="807ba-122">Vaikka resurssi asennetaan eri toiminnallisiin paikkoihin sen elinkaaren aikana, kustannukset voivat liittyä kuhunkin sijaintiin.</span><span class="sxs-lookup"><span data-stu-id="807ba-122">Even if an asset is installed on various functional locations during its lifetime, the costs can be related to each location.</span></span> <span data-ttu-id="807ba-123">Toisin sanoen resurssien kustannukset liittyvät aina siihen toiminnalliseen sijaintiin, johon resurssi on kullakin hetkellä asennettu.</span><span class="sxs-lookup"><span data-stu-id="807ba-123">In other words, asset costs are always related to the functional location that the asset was installed on at a given time.</span></span>

<span data-ttu-id="807ba-124">Toiminnalliset sijainnit **eivät** ole joustavia.</span><span class="sxs-lookup"><span data-stu-id="807ba-124">Functional locations are **not** flexible.</span></span> <span data-ttu-id="807ba-125">Siksi, kun olet määrittänyt toiminnallisen sijainnin hierarkian, et voi siirtää sijainteja sen ympärillä.</span><span class="sxs-lookup"><span data-stu-id="807ba-125">Therefore, after you set up a functional location hierarchy, you can't move locations around in it.</span></span>
