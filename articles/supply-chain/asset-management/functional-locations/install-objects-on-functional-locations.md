    <span data-ttu-id="cb205-121">**Määritteet**-osassa on luettelo resurssimääritteen vaatimuksista, jotka on määritetty toiminnalliselle sijainnille valitulle toiminnalliselle sijaintityypille.</span><span class="sxs-lookup"><span data-stu-id="cb205-121">The **Attributes** section shows a list of the asset attribute requirements that are set up on the functional location type that is selected for the functional location.</span></span> <span data-ttu-id="cb205-122">Määritteet on tarkoitettu vain tiedoksi.</span><span class="sxs-lookup"><span data-stu-id="cb205-122">The attributes are for informational purposes only.</span></span> <span data-ttu-id="cb205-123">Järjestelmä ei vahvista määritteitä niiden resurssimääritteiden perusteella, jotka on määritetty resurssille asennettaessa.</span><span class="sxs-lookup"><span data-stu-id="cb205-123">The system doesn't validate the attributes against the asset attributes that are set up on the asset that you're installing.</span></span> <span data-ttu-id="cb205-124">Sinun on tehtävä oikeellisuustarkistus, kun olet valinnut resurssin **Resurssi**-kentässä.</span><span class="sxs-lookup"><span data-stu-id="cb205-124">You must do that validation after you select an asset in the **Asset** field.</span></span>

> [!NOTE]
> <span data-ttu-id="cb205-142">Kun resurssiin liittyviä tietoja muutetaan, se päivitetään automaattisesti toiminnalliseen sijaintiin, johon resurssi on asennettu.</span><span class="sxs-lookup"><span data-stu-id="cb205-142">When asset-related data is changed, it's automatically updated on the functional location that the asset is installed on.</span></span> <span data-ttu-id="cb205-143">Tämä automaattinen päivitys koskee kunnossapitopyyntöjen, työtilausten, resurssin vikarekisteröintien, kunnossapitokatkoksien rekisteröintien ja resurssien mittausrekisteröintien muutoksia.</span><span class="sxs-lookup"><span data-stu-id="cb205-143">This automatic update pertains to changes to maintenance requests, work orders, asset fault registrations, maintenance downtime registrations, and asset measure registrations.</span></span>
