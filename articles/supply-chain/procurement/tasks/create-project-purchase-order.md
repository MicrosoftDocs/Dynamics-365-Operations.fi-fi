<span data-ttu-id="088ec-104">Tämä menettely osoittaa, miten projektin ostotilaus luodaan.</span><span class="sxs-lookup"><span data-stu-id="088ec-104">This procedure shows you how to create a project purchase order.</span></span> <span data-ttu-id="088ec-105">Tässä tehtävässä käytetään USSI-tietojoukkoa.</span><span class="sxs-lookup"><span data-stu-id="088ec-105">This task uses the USSI data set.</span></span>

1. <span data-ttu-id="088ec-106">Valitse Projektinhallinta ja kirjanpito > Projektit > Kaikki projektit.</span><span class="sxs-lookup"><span data-stu-id="088ec-106">Go to Project management and accounting > Projects > All projects.</span></span>
2. <span data-ttu-id="088ec-107">Napsauta luettelossa valitulla rivillä olevaa linkkiä.</span><span class="sxs-lookup"><span data-stu-id="088ec-107">In the list, click the link in the selected row.</span></span>
3. <span data-ttu-id="088ec-108">Valitse toimintoruudussa Hallitse.</span><span class="sxs-lookup"><span data-stu-id="088ec-108">On the Action Pane, click Manage.</span></span>
4. <span data-ttu-id="088ec-109">Valitse Nimiketehtävä.</span><span class="sxs-lookup"><span data-stu-id="088ec-109">Click Item task.</span></span>
5. <span data-ttu-id="088ec-110">Valitse Ostotilaus.</span><span class="sxs-lookup"><span data-stu-id="088ec-110">Click Purchase order.</span></span>
6. <span data-ttu-id="088ec-111">Anna tai valitse arvo Toimittajatili-kentässä.</span><span class="sxs-lookup"><span data-stu-id="088ec-111">In the Vendor account field, enter or select a value.</span></span>

    * <span data-ttu-id="088ec-113">Nämä vaiheet eivät ole pakollisia, mutta ne yksinkertaistavat ostotilausta määrittämällä ostotilausriveille oletustoimipaikan ja -varaston.</span><span class="sxs-lookup"><span data-stu-id="088ec-113">These steps aren't required, but they do simplify the purchase order by setting up a default site and warehouse for the purchase order lines.</span></span>  
8. <span data-ttu-id="088ec-114">Anna tai valitse Varasto-kentässä arvo.</span><span class="sxs-lookup"><span data-stu-id="088ec-114">In the Warehouse field, enter or select a value.</span></span>
9. <span data-ttu-id="088ec-115">Valitse OK.</span><span class="sxs-lookup"><span data-stu-id="088ec-115">Click OK.</span></span>
10. <span data-ttu-id="088ec-116">Merkitse valittu rivi luettelossa.</span><span class="sxs-lookup"><span data-stu-id="088ec-116">In the list, mark the selected row.</span></span>
11. <span data-ttu-id="088ec-117">Syötä tai valitse arvo Nimiketunnus-kentässä.</span><span class="sxs-lookup"><span data-stu-id="088ec-117">In the Item number field, enter or select a value.</span></span>
    * <span data-ttu-id="088ec-118">Tämä voi olla nimiketunnus tai hankintaluokka.</span><span class="sxs-lookup"><span data-stu-id="088ec-118">This can be the item number or a procurement category.</span></span>  
12. <span data-ttu-id="088ec-119">Laajenna Rivin erittely -osa.</span><span class="sxs-lookup"><span data-stu-id="088ec-119">Expand the Line details section.</span></span>
13. <span data-ttu-id="088ec-120">Valitse Projekti-välilehti.</span><span class="sxs-lookup"><span data-stu-id="088ec-120">Click the Project tab.</span></span>
    * <span data-ttu-id="088ec-121">Tarkista, että myynti- ja kustannushinnat ovat käytettävissä.</span><span class="sxs-lookup"><span data-stu-id="088ec-121">Verify that the sales and cost prices are available.</span></span> <span data-ttu-id="088ec-122">Jos ne eivät ole käytettävissä mutta niitä tarvitaan, anna tiedot.</span><span class="sxs-lookup"><span data-stu-id="088ec-122">If they are not available but needed, enter the information.</span></span>  
14. <span data-ttu-id="088ec-123">Valitse Tallenna.</span><span class="sxs-lookup"><span data-stu-id="088ec-123">Click Save.</span></span>
