<span data-ttu-id="2fbaa-105">Kun luot projektibudjettia, voit antaa projektin arvioidut tuotot ja kustannukset ja hallita sitten toteutuneita projektitapahtumia niiden avulla.</span><span class="sxs-lookup"><span data-stu-id="2fbaa-105">When you create a project budget, you can enter estimated revenues and costs for a project, and then use those to control actual project transactions.</span></span> <span data-ttu-id="2fbaa-106">Projektin budjetoinnissa kaikki alkuperäiset budjetit ja muutokset on lähettävä projektityönkulkuun hyväksyttäväksi.</span><span class="sxs-lookup"><span data-stu-id="2fbaa-106">In project budgeting, all original budgets and revisions must be sent to project workflow for approval.</span></span> <span data-ttu-id="2fbaa-107">Työnkulku parantaa prosessin hallintaa ja luo muutoshistoriatietueen.</span><span class="sxs-lookup"><span data-stu-id="2fbaa-107">Workflow gives you increased control over the process and creates a change history record.</span></span>

5. <span data-ttu-id="2fbaa-113">Valitse Projektibudjetti.</span><span class="sxs-lookup"><span data-stu-id="2fbaa-113">Click Project budget.</span></span>
6. <span data-ttu-id="2fbaa-114">Kirjoita arvo Kuvaus-kenttään.</span><span class="sxs-lookup"><span data-stu-id="2fbaa-114">In the Description field, type a value.</span></span>

14. <span data-ttu-id="2fbaa-122">Merkitse valittu rivi luettelossa.</span><span class="sxs-lookup"><span data-stu-id="2fbaa-122">In the list, mark the selected row.</span></span>

16. <span data-ttu-id="2fbaa-124">Anna tai valitse Luokka-kentässä arvo.</span><span class="sxs-lookup"><span data-stu-id="2fbaa-124">In the Category field, enter or select a value.</span></span>
