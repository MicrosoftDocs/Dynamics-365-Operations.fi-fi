## <a name="registration"></a><span data-ttu-id="423fb-117">Rekisteröinti</span><span class="sxs-lookup"><span data-stu-id="423fb-117">Registration</span></span>
<span data-ttu-id="423fb-118">Tuotteen vastaanoton rekisteröinti tehdään varaston saapuvien laiturilla.</span><span class="sxs-lookup"><span data-stu-id="423fb-118">Product receipt registration often occurs at the inbound docks in a warehouse.</span></span> <span data-ttu-id="423fb-119">Se tehdään joko kannettavalla laitteella tai saapumisen kirjauskansioiden avulla.</span><span class="sxs-lookup"><span data-stu-id="423fb-119">It’s performed either by using a hand-held device or through arrival journals.</span></span> <span data-ttu-id="423fb-120">Voit vaihtoehtoisesti rekisteröidä tuotteen vastaanoton manuaalisesti **Rekisteröinti**-toiminnolla **Ostotilaus**-sivulla.</span><span class="sxs-lookup"><span data-stu-id="423fb-120">Alternatively, you can manually register product receipt by using the **Registration** action on the **Purchase order** page.</span></span> <span data-ttu-id="423fb-121">Kummassakin tapauksessa tuotteiden tilaksi on merkitty **Rekisteröity**.</span><span class="sxs-lookup"><span data-stu-id="423fb-121">In both cases, the products are marked as **Registered**.</span></span> <span data-ttu-id="423fb-122">Huomaa, että tuotteiden tilaksi ei ole vielä merkitty **Vastaanotettu**.</span><span class="sxs-lookup"><span data-stu-id="423fb-122">Note that the products aren’t yet marked as **Received**.</span></span>

<span data-ttu-id="423fb-130">Useimmiten ostotilauksen tuotteet merkitään **Vastaanotettu**-tilaan käyttämällä **Ostotilaukset**-sivun **Tuotteen vastaanotto** -toimintoa.</span><span class="sxs-lookup"><span data-stu-id="423fb-130">Most often, the **Product receipt** action on the **Purchase orders** page is used to mark products as **Received** on the PO.</span></span> <span data-ttu-id="423fb-131">**Tuotteen vastaanoton kirjaus** -sivulla on useita asetuksia määrän kirjaamiseen vastaanotetuksi.</span><span class="sxs-lookup"><span data-stu-id="423fb-131">The **Posting product receipt** page has various options for the quantity that is accounted as received.</span></span> <span data-ttu-id="423fb-132">Voit esimerkiksi määrittää **Määrä**-kentän arvoksi **Tilattu määrä** tai **Vastaanottomäärä nyt**.</span><span class="sxs-lookup"><span data-stu-id="423fb-132">For example, you can set the **Quantity** field to **Ordered quantity** or **Receive now quantity**.</span></span> <span data-ttu-id="423fb-133">Jos käytössä on varastoon saapumisprosessi, voit vaihtoehtoisesti määrittää tämän kentän arvoksi **Rekisteröity määrä**.</span><span class="sxs-lookup"><span data-stu-id="423fb-133">Alternatively, if a warehouse arrival process has been used, you will often set this field to **Registered quantity**.</span></span> <span data-ttu-id="423fb-134">Voit muokata kunkin **Vastaanotettu**-tilaan merkittävän tilausrivien määriä mahdollisia poikkeavuuksia vastaaviksi.
