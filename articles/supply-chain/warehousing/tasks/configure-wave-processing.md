# <a name="configure-wave-processing"></a><span data-ttu-id="36d81-103">Aallon käsittelyn kokoonpano</span><span class="sxs-lookup"><span data-stu-id="36d81-103">Configure wave processing</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="36d81-104">Tässä ohjeessa kuvataan ehdot, jotka määrittävät kuinka aallon käsittelyn yhteydessä varastolle luotu työ käsitellään, ja käsitelläänkö aallot manuaalisesti vai automaattisesti.</span><span class="sxs-lookup"><span data-stu-id="36d81-104">This guide describes how to set up the criteria that determine what work is generated for a warehouse when a wave is processed, and whether waves are processed manually or automatically.</span></span> <span data-ttu-id="36d81-105">Voit määrittää ehdot asettamalla aaltomallit ja kyselyt, jotka vastaavat aaltoa myyntitilausten, tuotantotilausten ja kanban-tilausten vapautettujen rivien kanssa.</span><span class="sxs-lookup"><span data-stu-id="36d81-105">You specify the criteria by setting up wave templates and queries that match a wave with released lines in sales orders, production orders, or kanban orders.</span></span> <span data-ttu-id="36d81-106">Aallon käsittelyä käytetään varastoissa, jotka käyttävät varastonhallintamoduulin toimintoja, ei varastoissa, joissa käytetään inventoinninhallintamoduulin toimintoja.</span><span class="sxs-lookup"><span data-stu-id="36d81-106">Wave processing is used in warehouses that use the functionality in the Warehouse management module, and not those that use the functionality in the Inventory management module.</span></span> <span data-ttu-id="36d81-107">Voit suorittaa tämän menettelyn esittely-yrityksessä USMF.</span><span class="sxs-lookup"><span data-stu-id="36d81-107">You can run this procedure in demo data company USMF.</span></span>

    * <span data-ttu-id="36d81-111">Aaltomallia määrittäessäsi voit määrittää järjestyksen, jossa mallit sovitetaan vapautettuihin myyntitilausten, tuotantotilausten tai kanbanien riveihin.</span><span class="sxs-lookup"><span data-stu-id="36d81-111">When you set up a wave template, you specify the sequence in which the templates will be matched to released lines on sales orders, production orders, or kanbans.</span></span> <span data-ttu-id="36d81-112">Kun rivi on vapautettu varastoon tai tuotantoon, siinä käytetään ensimmäistä ehtoja vastaavaa aaltomallia.</span><span class="sxs-lookup"><span data-stu-id="36d81-112">When a line is released to the warehouse or to production, it uses the first wave template that it meets the criteria for.</span></span> <span data-ttu-id="36d81-113">Suosittelemme, että sijoitat tarkimmin määritellyt kriteerit sisältävän mallit luettelon alkuun.</span><span class="sxs-lookup"><span data-stu-id="36d81-113">We recommend that you put templates with the most specific criteria at the top of the list.</span></span> <span data-ttu-id="36d81-114">Mitä laajempi ehto on, sitä todennäköisemmin rivi täyttää ehdon, ja tämä saattaa johtaa rivien määrittämisen väärään aaltoon.</span><span class="sxs-lookup"><span data-stu-id="36d81-114">The broader the criteria, the more likely it is for a line to meet the criteria, and this could lead to lines being assigned to the wrong wave.</span></span>

    * <span data-ttu-id="36d81-128">Rivit määritetään aaltoihin aaltomallin kyselysuodattimen perusteella.</span><span class="sxs-lookup"><span data-stu-id="36d81-128">Lines are assigned to waves based on the query filter for the wave template.</span></span>  
11. <span data-ttu-id="36d81-129">Aseta Käsittele aalto automaattisesti raja-arvossa -asetuksen arvoksi Kyllä.</span><span class="sxs-lookup"><span data-stu-id="36d81-129">Set the Process wave automatically at threshold option to Yes.</span></span>
