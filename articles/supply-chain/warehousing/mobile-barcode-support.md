# <a name="mobile-barcode-support"></a><span data-ttu-id="fcba9-103">Mobiililaitteiden viivakoodituki</span><span class="sxs-lookup"><span data-stu-id="fcba9-103">Mobile barcode support</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="fcba9-104">Koska Android on avoimen lähdekoodin projekti, kuka tahansa varaston viivakoodilukijoiden laitteistovalmistaja voi tehdä Android-käyttöjärjestelmää käyttävän laitteen.</span><span class="sxs-lookup"><span data-stu-id="fcba9-104">Because Android is an open-source project, any manufacturer of hardware for warehouse bar code scanners can build a device to run the Android operating system.</span></span> <span data-ttu-id="fcba9-105">Laite on Android-yhteensopiva vain, jos siinä voidaan suorittaa Android-suoritusympäristölle luotuja sovelluksia.</span><span class="sxs-lookup"><span data-stu-id="fcba9-105">A device is only Android-compatible if it can run apps that are written for the Android execution environment.</span></span>

<span data-ttu-id="fcba9-106">Laitteiston toimittaja voi kuitenkin muokata Android-versiota ja luoda siihen kerroksia niin, että versio toimii laitteistossa.</span><span class="sxs-lookup"><span data-stu-id="fcba9-106">However, a hardware vendor can modify and create overlays for the Android version that runs on their hardware.</span></span> <span data-ttu-id="fcba9-107">Microsoft ei vastaa siitä, että Android-laitteen mobiililaitteen viivakoodin skannaussovellus on yhteensopiva valmistajan viivakoodilukulaitteiston kanssa tai että Android-versio toimii laitteessa.</span><span class="sxs-lookup"><span data-stu-id="fcba9-107">Microsoft can't take any responsibility to ensure that a mobile bar code scanning app for Android is compatible with a manufacturer's bar code scanning hardware and the Android version that runs on it.</span></span>

<span data-ttu-id="fcba9-108">Varastonhallinnan mobiilisovellusta on testattu erilaisissa Android-laitteissa viivakoodinlukuominaisuuksien varalta.</span><span class="sxs-lookup"><span data-stu-id="fcba9-108">The Warehouse Management mobile app has been tested with a selection of Android powered devices for bar code scanning.</span></span> <span data-ttu-id="fcba9-109">Testeissä oli mukana vain osa markkinoilla olevista laitteista.</span><span class="sxs-lookup"><span data-stu-id="fcba9-109">These tests only cover a sample of the devices that are available on the market.</span></span>

<span data-ttu-id="fcba9-110">Suosittelemme varaston mobiililaitteen skannaussovelluksen testaamista laitteistossa, ennen kuin laitteiston ostopäätös tehdään.</span><span class="sxs-lookup"><span data-stu-id="fcba9-110">As a customer, we recommend that you test the Warehouse mobile scanning app on selected hardware before you decide on the hardware that you want to buy.</span></span>
