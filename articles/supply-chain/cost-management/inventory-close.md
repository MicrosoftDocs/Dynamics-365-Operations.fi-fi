# <a name="inventory-close"></a><span data-ttu-id="93a2f-103">Varaston sulkeminen</span><span class="sxs-lookup"><span data-stu-id="93a2f-103">Inventory close</span></span>

<span data-ttu-id="93a2f-105">Varaston sulkemisprosessi selvittää varasto-ottotapahtumat vastaanottotapahtumiksi nimikkeen nimikemalliryhmässä valitun varastonarvostusmenetelmän perusteella.</span><span class="sxs-lookup"><span data-stu-id="93a2f-105">The inventory close process settles issue transactions to receipt transactions, based on the inventory valuation method that is selected in the item’s item model group.</span></span> <span data-ttu-id="93a2f-106">Selvitysprosessin yhteydessä voit määrittää, että kirjanpito on päivitettävä, jotta se vastaa tehtyjä oikaisuja.</span><span class="sxs-lookup"><span data-stu-id="93a2f-106">As part of the settlement process, you can specify that the general ledger should be updated, so that it reflects the adjustments that have been made.</span></span> <span data-ttu-id="93a2f-107">Varaston sulkemiseen tai uudelleenlaskentaan saakka tapahtumat kirjataan kuitenkin lasketun keskimääräisen kustannushinnan mukaan.</span><span class="sxs-lookup"><span data-stu-id="93a2f-107">However, until inventory close or recalculation has been run, issue transactions are posted at the calculated running average cost price.</span></span>
