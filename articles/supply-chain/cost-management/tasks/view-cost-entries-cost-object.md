# <a name="view-cost-entries-for-a-cost-object"></a>Näytä kustannusobjektin kustannusmerkinnät

[!include [banner](../../includes/banner.md)]

Tässä menettelyssä käsitellään, miten kustannusobjektin kustannusmerkintöjä tarkastellaan. Tämän menettelyn luomisessa käytetty esittely-yritys on USMF. Tämä menettely on tarkoitettu kustannusten vastuuhenkilölle.

1. Valitse Kustannuslaskenta.
2. Valitse Vapautetut tuotteet.
3. Käytä pikasuodatinta tietueiden etsimiseen. Voit esimerkiksi suodattaa Nimiketunnus-kenttää arvolla m0004.
4. Valitse toimintoruudussa Hallitse kustannuksia.
5. Valitse Kustannusobjektit.
6. Valitse Kustannusmerkinnät.
7. Suodata pikasuodattimella Nimiketunnus-kenttää arvon p000031 mukaan.
    * Jos kustannusmerkinnät ovat tyhjiä, määritä Päivämäärästä-arvoksi 31. tammikuuta 2012 ja Päivämäärään-arvoksi 31. joulukuuta 2012.
