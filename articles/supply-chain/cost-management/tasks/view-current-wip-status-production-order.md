# <a name="view-current-wip-status-on-a-production-order"></a>Näytä tuotantotilauksen nykyinen KET-tila

[!include [task guide banner](../../includes/task-guide-banner.md)]

Tässä menettelyssä näytetään, miten tuotantotilauksen KET-raporttia tarkastellaan. Tämän menettelyn luomisessa käytetty esittely-yritys on USMF. Tämä menettely on tarkoitettu kustannusten vastuuhenkilölle.

1. Valitse Kustannuslaskenta.
2. Valitse Tuotantotilaukset.
3. Pikasuodattimen avulla voit suodattaa Tuotanto-kentän arvolla p000153.
4. Valitse toimintoruudussa Hallitse kustannuksia.
5. Valitse Tuotannon KET-raportti.
6. Lisää Päivämäärästä-kenttään päivämäärä 1.1.2012.
7. Lisää Päivämäärään-kenttään päivämäärä 31.12.2012.
