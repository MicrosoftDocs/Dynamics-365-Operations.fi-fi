# <a name="define-discrete-manufacturing-resource-group"></a>Määritä erillisen valmistuksen resurssiryhmä

[!include [banner](../../includes/banner.md)]

Resurssiryhmä on operatiivisten resurssien joukko, joka vastaa yleensä tuotannon työnohjauksen keltaisten viivojen määrittämää työsolujen fyysistä organisaatiota. Tässä kuvataan, miten erillisessä tuotannossa käytettävä resurssiryhmä määritetään. Voit käydä tämän menettelyn läpi emotietojen yrityksen USMF avulla tai käyttää omia tietojasi.

1. Siirry Resurssiryhmät-kohtaan.
2. Valitse Uusi.
3. Kirjoita arvo Resurssiryhmä-kenttään.
4. Kirjoita arvo Kuvaus-kenttään.
5. Syötä tai valitse arvo Toimipaikka-kenttään.
6. Syötä tai valitse arvo Tuotantoyksikkö-kenttään.

## <a name="define-default-operational-parameters"></a>Oletustoimintaparametrien määrittäminen
1. Laajenna Työvaihe-osa.
2. Syötä numero Hävikkiprosentti-kenttään.
3. Syötä tai valitse arvo Asetusluokka-kenttään.
4. Syötä tai valitse arvo Ajoaikaluokka-kenttään.
5. Syötä numero Työvaiheen karkeasuunnittelun prosenttiosuus -kenttään.

## <a name="define-operating-hours"></a>Työtuntien määrittäminen
1. Laajenna Kalenterit-osa.
2. ValitseLisää.
3. Syötä tai valitse arvo Kalenteri-kenttään.

## <a name="add-operations-resources"></a>Operatiivisten resurssien lisääminen
1. Laajenna Resurssit-osa.
2. ValitseLisää.
3. Syötä tai valitse arvo Resurssi-kenttään.
4. ValitseLisää.
5. Syötä tai valitse arvo Resurssi-kenttään.
6. Etsi haluamasi tietue luettelosta ja valitse se.
7. Napsauta luettelossa valitulla rivillä olevaa linkkiä.
