# <a name="create-a-value-stream"></a>Luo arvovirta

[!include [banner](../../includes/banner.md)]

Arvovirrat ovat toimintayksiköitä, joilla ryhmitetään vähintään yksi tuotantovirta. Tuotantovirran määritettävän edellyttää arvovirran määrittämistä ensin.


## <a name="open-the-form"></a>Lomakkeen avaaminen
1. Valitse Tuotannonhallinta > Asetukset > Lean-tuotantovirta > Arvovirrat.

## <a name="create-a-new-value-stream"></a>Luo uusi arvovirta
1. Avaa valintaikkuna valitsemalla Uusi.
2. Kirjoita arvo Nimi-kenttään.

## <a name="select-a-value-stream-manager"></a>Valitse arvovirran hallinta
1. Avaa haku napsauttamalla Esimies-kentässä avattavan valikon painiketta.
    * Arvovirtaa hallitsevalle työntekijä on määritetty dokumentoimaan organisaatio.  
2. Etsi haluamasi tietue luettelosta ja valitse se.
3. Napsauta luettelossa valitulla rivillä olevaa linkkiä.

## <a name="save-and-close"></a>Tallenna ja sulje
1. Valitse Tallenna.
2. Sulje sivu.
3. Päivitä sivu.
