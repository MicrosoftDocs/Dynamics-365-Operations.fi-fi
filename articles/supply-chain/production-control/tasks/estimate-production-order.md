# <a name="estimate-a-production-order"></a>Arvioi tuotantotilaus

[!include [banner](../../includes/banner.md)]

Voit suorittaa tämän menettelyn USMF-demoyrityksen tiedoilla tai käyttää omaa tietojoukkoa. Molemmissa tapauksissa sinulla on oltava avoin tuotantotilaus, jonka tila on Luotu. Tämä on toinen seitsemästä tuotantotilauksen elinkaaresta kertovasta menettelystä.


## <a name="estimate-a-production-order"></a>Arvioi tuotantotilaus
1. Siirry kohtaan Tuotannonhallinta > Tuotantotilaukset > Kaikki tuotantotilaukset.
2. Valitse ruudukossa tuotantotilaus, jonka tila on Luotu
3. Valitse toimintoruudussa Tuotantotilaus.
4. Valitse Arvio.
    * Tässä vaiheessa lasketaan yhden tuotantotilauksen arvioidut kustannukset.   
5. Valitse OK.

## <a name="view-the-calculation-details"></a>Näytä laskennan tiedot
1. Valitse toimintoruudussa Hallitse kustannuksia.
2. Valitse Näytä laskennan tiedot.
    * Kustannuserittely näkyy tällä sivulla. Voit esimerkiksi tarkastella ensimmäisellä rivillä valmiin tuotteen yksikkökohtaista kokonaiskustannushintaa. Seuraavilla riveillä on tuoterakenteen ja tuotantoreitin kustannukset sekä välilliset kustannukset.
