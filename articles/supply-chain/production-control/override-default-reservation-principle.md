# <a name="override-the-default-reservation-principle-for-materials-in-production"></a><span data-ttu-id="643bd-103">Tuotannon materiaalien oletusvarausperiaatteen ohittaminen</span><span class="sxs-lookup"><span data-stu-id="643bd-103">Override the default reservation principle for materials in production</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="643bd-104">*Ohita tuotannon oletusvaraus* -toiminnolla voidaan määrittää kunkin nimikemalliryhmän oletusvarausperiaate.</span><span class="sxs-lookup"><span data-stu-id="643bd-104">The *Override default production reservation* feature lets you set a default reservation principle for each item model group.</span></span> <span data-ttu-id="643bd-105">Niinpä kussakin tuotannon tuoterakenteen tai erätilauskaavan nimikkeessä käytetään automaattisesti eri varausperiaatetta.</span><span class="sxs-lookup"><span data-stu-id="643bd-105">Therefore, different reservation principles can automatically be applied for each item that is part of a production bill of materials (BOM) or batch order formula.</span></span> <span data-ttu-id="643bd-106">Voit valita, ohittaako jokainen nimikemalliryhmä tilaukselle määritetyn oletusvarausperiaatteen ja mitä varausperiaatetta käytetään sen tilalla (*manuaalinen*, *arvio*, *aikataulutus*, *vapautus* tai *aloitus*).</span><span class="sxs-lookup"><span data-stu-id="643bd-106">You can select whether each item model group should override the default reservation principle that is set for an order, and what reservation principle should be used instead (*manual*, *estimation*, *scheduling*, *release*, or *start*).</span></span>

<span data-ttu-id="643bd-107">Uutta tuotantotilausta tai erätilausta luotaessa pyydetään valitsemaan varausperiaate, jota on käytettävä kyseisessä tilauksessa ja sen kaikilla tuoterakenneriveillä tai kaavariveillä.</span><span class="sxs-lookup"><span data-stu-id="643bd-107">When you create a new production order or batch order, you're prompted to select the reservation principle that should be applied to that order and all its BOM lines or formula lines.</span></span> <span data-ttu-id="643bd-108">*Ohita tuotannon oletusvaraus* -toimintoa käytettäessä osa tuoterakenne- tai kaavariveistä tai kaikki kyseiset rivit voivat ohittaa kyseisen varausperiaatteen ja käyttää sen sijaan soveltuvalle nimikemalliryhmälle määritettyä varausperiaatetta.</span><span class="sxs-lookup"><span data-stu-id="643bd-108">When the *Override default production reservation* feature is used, some or all of the BOM or formula lines can override that reservation principle and instead use the reservation principle that is set for the relevant item model group.</span></span>

<span data-ttu-id="643bd-109">Jos esimerkiksi raaka-aineet tai ainesosat edellyttävät keräilytyötä, kyseisille tuotteille luodut tuoterakenne- tai kaavarivit edellyttävät fyysistä varausta, koska fyysinen varaus on varastotyön luonnin edellytys.</span><span class="sxs-lookup"><span data-stu-id="643bd-109">For example, if you have raw materials or ingredients that require pick work, BOM or formula lines that are created for those products require a physical reservation, because physical reservation is a prerequisite for the generation of warehouse work.</span></span> <span data-ttu-id="643bd-110">Jos varauksen halutaan tapahtuvan automaattisesti, yleensä valitaan jokin seuraavista varausperiaatteista: *arvio*, *aikataulutus*, *vapautus* tai *aloitus*.</span><span class="sxs-lookup"><span data-stu-id="643bd-110">Typically, if you want the reservation to occur automatically, you select one of the following reservation principles: *estimation*, *scheduling*, *release*, or *start*.</span></span> <span data-ttu-id="643bd-111">Jos materiaalit tai ainesosat eivät kuitenkaan edellytä keräilytyötä, koska ne kulutetaan suojaan sijainnista, *manuaalinen* varausperiaate valitaan yleensä sen vuoksi, että mitään varauksia ei tehdä eikä keräilytöitä luoda.</span><span class="sxs-lookup"><span data-stu-id="643bd-111">On the other hand, if you have materials or ingredients that don't require pick work, because they are consumed directly from a location, you typically select the *manual* reservation principle, which doesn't make any physical reservations or generate any pick work.</span></span>

## <a name="assign-a-production-reservation-policy-to-an-item-model-group"></a><span data-ttu-id="643bd-118">Tuotannon varauskäytännön määrittäminen nimikemalliryhmään</span><span class="sxs-lookup"><span data-stu-id="643bd-118">Assign a production reservation policy to an item model group</span></span>

<span data-ttu-id="643bd-135">Tässä skenaariossa luodaan erätilaus tuottamaan 1 000 litraa irtotavaraa.</span><span class="sxs-lookup"><span data-stu-id="643bd-135">In this scenario, a batch order to make 1,000 liters of bulk material is created.</span></span> <span data-ttu-id="643bd-136">(Tämä tilaus on irtotavaratilaus.)
