## <a name="description-of-the-flushing-principles"></a><span data-ttu-id="4692f-125">Materiaalinottosääntöjen kuvaus</span><span class="sxs-lookup"><span data-stu-id="4692f-125">Description of the flushing principles</span></span>

### <a name="manual"></a><span data-ttu-id="4692f-126">Manuaalinen</span><span class="sxs-lookup"><span data-stu-id="4692f-126">Manual</span></span>
<span data-ttu-id="4692f-127">Manuaalinen materiaalinottosääntö ilmaisee, että materiaalikulutuksen rekisteröinti on manuaalinen toiminto.</span><span class="sxs-lookup"><span data-stu-id="4692f-127">The Manual flushing principle indicates that the registration of material consumption is a manual operation.</span></span> <span data-ttu-id="4692f-128">Tällä säännöllä on merkitystä, jos haluat esimerkiksi seurata aikaa, ja kulutettujen eränumeroiden tai sarjanumeroiden määrästä on oltava selvillä seurantaa varten.</span><span class="sxs-lookup"><span data-stu-id="4692f-128">This principle is relevant if, for example, you want to be able to track time, and the quantity of consumed batch numbers or serial numbers must be accounted for, for tracking purposes.</span></span> <span data-ttu-id="4692f-129">Manuaalinen kulutus rekisteröidään tuotannon keräysluettelon kirjauskansioon.</span><span class="sxs-lookup"><span data-stu-id="4692f-129">Manual consumption is registered in a production picking list journal.</span></span> <span data-ttu-id="4692f-130">Jos edistykselliset varastoprosessit on otettu käyttöön nimikkeissä, voidaan käyttää käsikäyttöistä työnkulkua.</span><span class="sxs-lookup"><span data-stu-id="4692f-130">For items that are enabled for advanced warehouse processes, a hand-held flow can be applied.</span></span>

### <a name="start"></a><span data-ttu-id="4692f-131">Alku</span><span class="sxs-lookup"><span data-stu-id="4692f-131">Start</span></span>
<span data-ttu-id="4692f-132">Alku-materiaaliottosääntö ilmaisee, että materiaali kulutetaan automaattisesti, kun tuotantotilaus käynnistetään.</span><span class="sxs-lookup"><span data-stu-id="4692f-132">The Start flushing principle indicates that material will be automatically consumed when the production order is started.</span></span> <span data-ttu-id="4692f-133">Kulutetun materiaalin määrä on suhteessa käynnistettyyn määrään.</span><span class="sxs-lookup"><span data-stu-id="4692f-133">The amount of material that is consumed is proportional to the quantity that is started.</span></span> <span data-ttu-id="4692f-134">Kun Alku-materiaaliottosääntöä käytetään yhdessä tuotannonohjausjärjestelmän kanssa, sitä voidaan käyttää materiaaliottoihin, kun toiminto tai prosessityö on aloitettu.</span><span class="sxs-lookup"><span data-stu-id="4692f-134">When the Start flushing principle is used together with the manufacturing execution system, it can also be used to flush materials when an operation or a process job is started.</span></span> <span data-ttu-id="4692f-135">Säännöllä on merkitystä, jos esimerkiksi kulutuksen varianssi on pieni, materiaalit ovat arvoiltaan vähäisiä, seurantaa ei tarvita tai toimenpiteen suoritusaika on lyhyt.</span><span class="sxs-lookup"><span data-stu-id="4692f-135">This principle is relevant if, for example, the variance in the consumption is low, the materials are low-value materials, there are no tracking requirements, or there is a short run time on operations.</span></span> 

### <a name="finish"></a><span data-ttu-id="4692f-136">Valmis</span><span class="sxs-lookup"><span data-stu-id="4692f-136">Finish</span></span>
<span data-ttu-id="4692f-137">Loppu-materiaaliottosääntö ilmaisee, että materiaali kulutetaan automaattisesti, kun tuotantotilaus on ilmoitettu valmiiksi tai kun materiaalia kuluttavaksi määritetty toiminto on rekisteröity valmiiksi.</span><span class="sxs-lookup"><span data-stu-id="4692f-137">The Finish flushing principle indicates that material will be automatically consumed when the production order is reported as finished, or when an operation that is set up to consume the materials is registered as completed.</span></span> <span data-ttu-id="4692f-138">Kulutetun materiaalin määrä on suhteessa valmiiksi ilmoitettuun määrään.</span><span class="sxs-lookup"><span data-stu-id="4692f-138">The amount of material that is consumed is proportional to the quantity that is reported as finished.</span></span> <span data-ttu-id="4692f-139">Kun Loppu-materiaaliottosääntöä käytetään yhdessä tuotannonohjausjärjestelmän kanssa, sitä voidaan käyttää materiaaliottoihin, kun toiminto tai prosessityö on valmis.</span><span class="sxs-lookup"><span data-stu-id="4692f-139">When the Finish flushing principle is used together with the manufacturing execution system, it can also be used to flush materials when an operation or a process job is completed.</span></span> <span data-ttu-id="4692f-140">Tällä säännöllä on merkitystä samoissa tilanteissa kuin Alku-säännöllä.</span><span class="sxs-lookup"><span data-stu-id="4692f-140">This principle is relevant in the same situations as the Start principle.</span></span> <span data-ttu-id="4692f-141">Loppu-sääntö on kuitenkin tarkoitettu toiminnoille, joiden ajoaika pidempi ja joissa materiaaleja ei määritetä keskeneräisiksi töiksi ennen toiminnon valmistumista.</span><span class="sxs-lookup"><span data-stu-id="4692f-141">However, the Finish principle is for operations that have a longer run time, where materials should not be set to WIP before the operation is completed.</span></span> 

### <a name="available-at-location"></a><span data-ttu-id="4692f-142">Käytettävissä sijainnissa</span><span class="sxs-lookup"><span data-stu-id="4692f-142">Available at location</span></span>
<span data-ttu-id="4692f-143">Käytettävissä sijainnissa -materiaaliottosääntö ilmaisee, että materiaali kulutetaan automaattisesti, kun se on rekisteröity tuotantoon kerätyksi.</span><span class="sxs-lookup"><span data-stu-id="4692f-143">The Available at location flushing principle indicates that the material will be automatically consumed when it's registered as picked for production.</span></span> <span data-ttu-id="4692f-144">Materiaali on rekisteröity sijainnista kerätyksi, kun raaka-aineen keräilytyö on valmis tai kun materiaali on käytettävissä tuotannon varastosijainnissa ja materiaalirivi on vapautettu varastoon.</span><span class="sxs-lookup"><span data-stu-id="4692f-144">The material is registered as picked from location when work for the raw material picking is completed, or when material is available on the production input location and the material line is released to the warehouse.</span></span> <span data-ttu-id="4692f-145">Prosessin aikana luotu keräystyö kirjataan erätyöhön.</span><span class="sxs-lookup"><span data-stu-id="4692f-145">The picking list that is generated during the process is posted in a batch job.</span></span> <span data-ttu-id="4692f-146">Tällä säännöllä on merkitystä, jos sinulla on esimerkiksi tuotantotilaus, jossa on useita keräystoimintoja.</span><span class="sxs-lookup"><span data-stu-id="4692f-146">This principle is relevant if, for example, you have many picking activities against one production order.</span></span> <span data-ttu-id="4692f-147">Tässä tapauksessa keräyslistaa ei tarvitse päivittää manuaalisesti ja saat KET-saldon ajantasaisen näkymän.</span><span class="sxs-lookup"><span data-stu-id="4692f-147">In this case, you don't have to update the picking list manually, and you can get a current view of the WIP balance.</span></span>
