## <a name="registrations-in-manufacturing-execution"></a><span data-ttu-id="e2692-131">Rekisteröinnit valmistustoteutuksessa</span><span class="sxs-lookup"><span data-stu-id="e2692-131">Registrations in manufacturing execution</span></span>
<span data-ttu-id="e2692-132">Työntekijät voivat tehdä erityyppisiä rekisteröintejä tuotannonohjauksessa työlle, joka on suoritettu tuotantotöistä.</span><span class="sxs-lookup"><span data-stu-id="e2692-132">Workers can make various types of registrations in manufacturing execution for work that is performed on production jobs.</span></span> <span data-ttu-id="e2692-133">Järjestelmän asetuksesta riippuen saattaa olla myös mahdollista, että työntekijät voivat suorittaa rekisteröintejä projektitehtäville ja tuottamattomille tehtäville, kuten tauoille, poissaololle ja epäsuorille tehtäville.</span><span class="sxs-lookup"><span data-stu-id="e2692-133">Depending on the system setup, workers might also be able to make registrations on project activities and nonproductive tasks, such as breaks, absences, and indirect activities.</span></span> <span data-ttu-id="e2692-134">Rekisteröintityypit ovat seuraavat:</span><span class="sxs-lookup"><span data-stu-id="e2692-134">Here are the registration types:</span></span>

-   <span data-ttu-id="e2692-143">**Rekisteröi projektimaksut ja projektinimikkeet** (käytettävissä työajan seurannassa): Työntekijät voivat rekisteröidä maksuja (kustannukset), jotka liittyvät projektiin sen maksukirjauskansiossa, kuten kilometrikorvauksia ja siltamaksuja.</span><span class="sxs-lookup"><span data-stu-id="e2692-143">**Register project fees and project items** (available with time and attendance) – Workers can register fees (expenses) that are associated with a project in a project fee journal, such as mileage and bridge toll.</span></span> <span data-ttu-id="e2692-144">Työntekijät voivat myös rekisteröidä nimikkeen kulutuksen projekteissa.</span><span class="sxs-lookup"><span data-stu-id="e2692-144">Workers can also register item consumption on projects.</span></span> <span data-ttu-id="e2692-145">Tämä suoritetaan projektinimikkeiden kirjauskansiossa.</span><span class="sxs-lookup"><span data-stu-id="e2692-145">This is done in a project item journal.</span></span>

-   <span data-ttu-id="e2692-155">**Rekisteröi ylityö** (saatavilla työajan seurannassa) – Ylitöihin pyydetyt työntekijät voivat merkitä ylityötuntinsa joustavaksi työajaksi tai ylitöiksi.</span><span class="sxs-lookup"><span data-stu-id="e2692-155">**Register overtime** (available with time and attendance) – Workers who have been asked to work longer hours can select whether the extra hours should be registered as flextime or overtime.</span></span>
