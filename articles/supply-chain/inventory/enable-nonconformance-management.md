[!include[banner](../includes/banner.md)]

<span data-ttu-id="2138a-107">Voit ottaa määrityksistä poikkeamisen käyttöön seuraavien vaiheiden avulla:</span><span class="sxs-lookup"><span data-stu-id="2138a-107">To enable nonconformance management, follow these steps:</span></span>

    -   <span data-ttu-id="2138a-114">Käytä **Laadunhallinta** -välilehteä **Raportin asetukset** -sivulla, jotta voit määrittää tulostettavan asiakirjan tyypin.</span><span class="sxs-lookup"><span notes.</span></span> <span data-ttu-id="2138a-117">On hyödyllistä käyttää **Asiakirjatyyppi**-sivua määritettäessä yksilöivä asiakirjatyyppi määrityksistä poikkeamiselle ja korjauksille.</span><span class="sxs-lookup"><span data-stu-id="2138a-117">You might find it helpful to use the **Document type** page to define a unique document type for nonconformances and a unique document type for     -   <span data-ttu-id="2138a-120">Syötä määrityksistä poikkeamisten ja korjausviitteiden numerosarjat.</span><span class="sxs-lookup"><span data-stu-id="2138a-120">Enable number sequences for nonconformance and correction references.</span></span>

3.  <span data-ttu-id="2138a-125">Määritä määrityksistä poikkeamisen liitettävät ongelmatyypit.</span><span class="sxs-lookup"><span data-stu-id="2138a-125">Define the problem types that will be assigned to nonconformances.</span></span> <span data-ttu-id="2138a-126">Määritä **Ongelmatyypit**-sivulla luokitus laatuongelmille, joita kohdataan erityyppisissä määrityksistä poikkeamisen tyypeissä.</span><span class="sxs-lookup"><span data-stu-id="2138a-126">Use the **Problem types** page to define a classification of quality problems that are encountered for the various nonconformance types.</span></span> <span data-ttu-id="2138a-127">Voit määrittää seuraavat määrityksistä poikkeamisen tyypit: **Sisäinen**, **Asiakas**, **Toimittaja**, **Palvelupyyntö**, **Tuotanto** **Määrityksistä poikkeamisen tyypit** -sivulla.</span><span class="sxs-lookup"><span data-stu-id="2138a-128">Use the **Non conformance types** page to authorize a problem type to be used in one or 4.  <span data-ttu-id="2138a-130">Määritä karanteenivyöhykkeet, jotka ohjaavat käsittelyä odottavaa viallista materiaalia.</span><span class="sxs-lookup"><span data-stu-id="2138a-130">Define quarantine zones to provide guidance about defective material should be handled.</span></span> <span data-ttu-id="2138a-131">Määritä **Karanteenivyöhykkeet**-sivulla vyöhykkeet, jotka voidaan liittää määrityksistä poikkeamiseen.</span><span class="sxs-lookup"><span data-stu-id="2138a-131">Use the **Quarantine zones** page to define zones that can be assigned to a nonconformance.</span></span> <span data-ttu-id="2138a-132">Tulostetussa määrityksistä poikkeamisen tunnuksessa näkyy määritetty karanteenivyöhyke ja käyttötiedot, jotka ohjaavat viallisen materiaalin käsittelyssä.</span><span class="sxs-lookup"><span data-stu-id="2138a-132">The printed nonconformance tag will show the assigned quarantine zone and information about usage, to provide guidance about how to handle defective material.</span></span> <span data-ttu-id="2138a-133">Vyöhykkeet voivat vastata varastopaikkoja tai operatiivisia resursseja.</span><span class="sxs-lookup"><span data-stu-id="2138a-133">The zones might correspond to inventory locations or operations resources.</span></span>
5.  <span data-ttu-id="2138a-134">Määritä korjauksiin liitettävät vianmääritystyypit.</span><span class="sxs-lookup"><span data-stu-id="2138a-134">Define the diagnostic types that will be assigned to corrections.</span></span> <span data-ttu-id="2138a-135">Määritä vianmääritystoimenpiteiden luokitus **Vianmääritystyypit**-sivulla.</span><span class="sxs-lookup"><span data-stu-id="2138a-135">Use the **Diagnostic types** page to define a classification of diagnostic actions.</span></span> <span data-ttu-id="2138a-136">Korjaus ilmaisee hyväksytyn poikkeaman vianmääritystoimenpiteen ja sen tekijän.</span><span class="sxs-lookup"><span data-stu-id="2138a-136">A correction specifies what type of diagnostic action should be taken for an approved nonconformance, and who should take that action.</span></span> <span data-ttu-id="2138a-137">Se määrittää myös halutun ja suunnittelun valmistumispäivämäärän.</span><span class="sxs-lookup"><span data-stu-id="2138a-137">It also specifies the requested completion date and the planned completion date.</span></span>
6.  <span data-ttu-id="2138a-138">Määritä muut määrityksistä poikkemisiin liitettävät toiminnot.</span><span class="sxs-lookup"><span data-stu-id="2138a-138">Define the related operations that will be assigned to nonconformances.</span></span> <span data-ttu-id="2138a-139">Määritä **Toiminnot**-sivulla avulla luokitus hyväksytyn määrityksistä poikkeamisen käsittelyä varten.</span><span class="sxs-lookup"><span data-stu-id="2138a-139">Use the **Operations** page to define a classification of the work that can be performed for an approved nonconformance.</span></span> <span data-ttu-id="2138a-140">Kun liität määrityksistä poikkeamiseen toiminnon, voit myös määrittää yksityiskohtia tietoja, kuten tietoja liittyvästä materiaalista, työtunneista ja erilaisista toiminnon tekemiseen tarvittavista kuluista.</span><span class="sxs-lookup"><span data-stu-id="2138a-140">When you assign a related operation to a nonconformance, you can provide detailed information, such as information about the associated material, labor hours, and miscellaneous charges that are required in order to perform the operation.</span></span> <span data-ttu-id="2138a-141">Näiden tietojen avulla lasketaan toiminnon arvioitu kustannus.</span><span class="sxs-lookup"><span data-stu-id="2138a-141">This information is used to calculate an estimated cost for the operation.</span></span> <span data-ttu-id="2138a-142">Tarkat tiedot ja arvioidut kustannukset on tarkoitettu viitetiedoiksi.</span><span class="sxs-lookup"><span data-stu-id="2138a-142">The detailed information and estimated costs are for reference.</span></span> <span data-ttu-id="2138a-143">Laatuun liittyvät toiminnot poikkeavat tuotantoreititystä varten määritettävistä toiminnoista.</span><span class="sxs-lookup"><span data-stu-id="2138a-143">The related operations for quality differ from the operations that can be defined for a production route.</span></span>
