# <a name="set-up-a-transportation-tender"></a>Määritä kuljetuksen maksuväline

[!include [banner](../../includes/banner.md)]

Tässä menettelyssä näytetään, miten kuljetuksen maksuväline määritetään. Kuljetuskoordinaattori tekee tämän yleensä. Voit käyttää tätä menettelyä USMF:n esittely-yrityksessä.


## <a name="select-a-route"></a>Valitse reitti
1. Valitse Kuljetustenhallinta > Suunnittelu > Kuormasuunnittelun työtila.
2. Poista Piilota toimitettu ja vastaanotettu -valintaruudun valinta.
3. Valitse rivi, jolla on kuormatunnus 00006.
4. Valitse Hinnoittelu ja reititys.
5. Valitse Reitit.

## <a name="create-the-transportation-tender"></a>Luo kuljetuksen maksuväline
1. Valitse Kuljetuksen maksuvälineet.
2. Valitse Uusi.
3. Laajenna Yleiset-osa.
4. Kirjoita numero Pyydetyt hinnat -kenttään.
5. Valitse Tallenna.
6. Valitse Päivitä tila.
7. Valitse Lähetä.
8. Valitse reitti.
