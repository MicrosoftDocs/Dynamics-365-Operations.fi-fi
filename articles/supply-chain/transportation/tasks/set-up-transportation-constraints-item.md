# <a name="set-up-transportation-constraints-for-an-item"></a>Määritä nimikkeen kuljetusrajoitteet

[!include [banner](../../includes/banner.md)]

Tässä menettelyssä asetetaan kuljetusrajoitus, jolla estetään valitun nimikkeen kuljetus valitun keskuksen kautta. Yleensä kuljetuskoordinaattori tekee tämän tehtävän. Voit käyttää tätä menettelyä USMF-demoyrityksen tiedoilla tai omilla tiedoillasi.


## <a name="create-an-item-constaint"></a>Nimikerajoituksen luominen
1. Siirry kohtaan Rajoitukset.
2. Valitse Uusi.
3. Kirjoita arvo Nimikerajoitus-kenttään.
4. Kirjoita arvo Nimi-kenttään.
5. Syötä tai valitse arvo Toimipaikka-kenttään.
6. Anna tai valitse Varasto-kentässä arvo.
7. Syötä tai valitse arvo Nimiketunnus-kentässä.
8. Syötä tai valitse Keskus-kentän arvo.
9. Valitse Rajoitustoiminto-kentästä haluamasi vaihtoehto.
10. Valitse Tallenna.
11. Sulje sivu.
