# <a name="create-a-product"></a>Tuotteen luominen

[!include [banner](../../includes/banner.md)]

Tässä menettelyssä esitellään uuden tuotteen luominen Commerce-tuotehierarkiassa. Tämän tehtävän luomisessa käytetään esittely-tietojen yritystä USRT. Tehtävä on tarkoitettu myynninedistämispäällikölle ja tuotesuunnittelijalle.

1. Valitse Retail ja Commerce > .. > Tuotteet luokittain.
2. Valitse Uusi.
3. Syötä Tuotteen nimi -kenttään tuotteen nimi.
4. Avaa haku napsauttamalla luokkakentässä avattavan valikon painiketta.
5. Laajenna puussa solmu ALL (ALL)\Fashion (Fashion).
6. Valitse puussa solmu ALL (ALL)\Fashion (Fashion)\Fashion Accessories (Fashion Accessories).
7. Valitse OK.
8. Kirjoita arvo Tuotenumero-kenttään.
9. Valitse OK.
