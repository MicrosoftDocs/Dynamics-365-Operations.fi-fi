# <a name="create-a-product"></a>Luo tuote

[!include [task guide banner](../../includes/task-guide-banner.md)]

Tässä menettelyssä esitellään vähittäismyynnin tuotehierarkian uuden tuotteen luominen. Tämän tehtävän luomisessa käytetään esittely-tietojen yritystä USRT. Tehtävä on tarkoitettu myynninedistämispäällikölle ja tuotesuunnittelijalle.

1. Valitse Vähittäismyynti ja kauppa > .. > Tuotteet luokittain.
2. Valitse Uusi.
3. Syötä Tuotteen nimi -kenttään tuotteen nimi.
4. Avaa haku valitsemalla Vähittäismyyntiluokka-kentässä avattavan valikon painike.
5. Laajenna puussa solmu ALL (ALL)\Fashion (Fashion).
6. Valitse puussa solmu ALL (ALL)\Fashion (Fashion)\Fashion Accessories (Fashion Accessories).
7. Valitse OK.
8. Kirjoita arvo Tuotenumero-kenttään.
9. Valitse OK.
