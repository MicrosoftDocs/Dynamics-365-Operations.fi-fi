# <a name="set-up-disposition-codes"></a><span data-ttu-id="34f8a-103">Määritä käsittelykoodit</span><span class="sxs-lookup"><span data-stu-id="34f8a-103">Set up disposition codes</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="34f8a-104">Voit määrittää käsittelykoodit, jotka määrittävät, miten asiakkaan palauttamaa nimikettä käsitellään.</span><span class="sxs-lookup"><span data-stu-id="34f8a-104">You can set up disposition codes to specify how to process an item that is returned by a customer.</span></span> <span data-ttu-id="34f8a-105">Luo esimerkiksi käsittelykoodi nimeltä **Korjaa ja palauta**, kun haluat osoittaa, että palautettu nimike korjataan ja palautetaan asiakkaalle.</span><span class="sxs-lookup"><span data-stu-id="34f8a-105">For example, create a disposition code named **Repair and return** to indicate that the returned item will be repaired and then returned to the customer.</span></span> <span data-ttu-id="34f8a-106">Lisää esimerkkejä käsittelykoodeista, joita käytetään yleisesti asiakkaiden palauttamissa nimikkeissä, on kohdassa [Palautettujen nimikkeiden poistotavan määrittäminen](specify-how-to-dispose-of-returned-items.md).</span><span class="sxs-lookup"><span data-stu-id="34f8a-106">For more examples of disposition codes that are typically used for items that are returned by customers, see [Specify how to dispose of returned items](specify-how-to-dispose-of-returned-items.md).</span></span>

<span data-ttu-id="34f8a-107">Voit määrittää myös syykoodin, kun haluat kertoa nimikkeen palautuksen syyn.</span><span class="sxs-lookup"><span data-stu-id="34f8a-107">You can also set up a reason code to help explain why an item was returned.</span></span> <span data-ttu-id="34f8a-108">Lisätietoja syykoodeista on kohdassa [Palautusten syykoodien määrittäminen](set-up-return-reason-code.md).</span><span class="sxs-lookup"><span data-stu-id="34f8a-108">For more information about reason codes, see [Set up return reason codes](set-up-return-reason-code.md).</span></span>

1.  <span data-ttu-id="34f8a-109">Valitse **Myynti ja markkinointi** \> **Asetukset** \> **Myyntitilaukset** \> **Palautukset** \> **Käsittelykoodit**.</span><span class="sxs-lookup"><span data-stu-id="34f8a-109">Go to **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Disposition codes**.</span></span>

2.  <span data-ttu-id="34f8a-110">Valitse **Uusi** luodaksesi uuden käsittelykoodin.</span><span class="sxs-lookup"><span data-stu-id="34f8a-110">Select **New** to create a new disposition code.</span></span>

3.  <span data-ttu-id="34f8a-111">Syötä yksilöivä kuvaava nimi, valitse toiminto ja syötä käsittelykoodille kuvaus.</span><span class="sxs-lookup"><span data-stu-id="34f8a-111">Enter a unique, descriptive name, select an action, and enter a description for the disposition code.</span></span>

4.  <span data-ttu-id="34f8a-112">Jos haluat liittää tähän käsittelykoodiin asiakasmaksuja, valitse **Kulut**-painike avataksesi **Määritä kulut** -lomakkeen.</span><span class="sxs-lookup"><span data-stu-id="34f8a-112">If you want to associate any customer charges with this disposition code, select the **Charges** button to open the **Set up charges** form.</span></span>

5.  <span data-ttu-id="34f8a-113">Jos haluat määrittää yrityksen omia käsittelykoodeja vastaavia ulkoisia koodeja, valitse **Ulkoiset koodit** -painike avataksesi **Ulkoiset koodit** -lomakkeen.</span><span class="sxs-lookup"><span data-stu-id="34f8a-113">If you want to define any external codes to match with the company's own disposition codes, select the **External codes** button to open the **External codes** form.</span></span>
