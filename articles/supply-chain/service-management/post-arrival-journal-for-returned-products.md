[!include [banner](../includes/banner.md)]


<span data-ttu-id="78712-104">Palautuksen käsittely tapahtuu vahvistamalla ensin palautusmäärä ja päivittämällä määräkenttä nimikkeen saapumisen kirjauskansiossa.</span><span class="sxs-lookup"><span data-stu-id="78712-104">To process a return, first validate the return quantity, update the quantity field in the item arrival journal.</span></span> <span data-ttu-id="78712-105">Valitse sitten käsittelykoodi tai määritä, että palautetut nimikkeet on tarkastettava.</span><span class="sxs-lookup"><span data-stu-id="78712-105">Then select a disposition code or indicate that the returned items have to be inspected.</span></span> <span data-ttu-id="78712-106">Näiden vaiheiden jälkeen voit kirjata palautustilauksen nimikkeen saapumisen kirjauskansioon.</span><span class="sxs-lookup"><span data-stu-id="78712-106">After completing these steps, you can post the item arrival journal for the return order.</span></span>

1.  <span data-ttu-id="78712-107">Napsauta **Varastonhallinta** \> **Kausittainen** \> **Saapumisten yhteenveto**.</span><span class="sxs-lookup"><span data-stu-id="78712-107">Click **Inventory management** \> **Periodic** \> **Arrival overview**.</span></span>

2.  <span data-ttu-id="78712-108">Valitse **Asetusten nimi** -suodattimessa **Palautustilaus**.</span><span class="sxs-lookup"><span data-stu-id="78712-108">In the **Setup name** filter, select **Return order**.</span></span>

3.  <span data-ttu-id="78712-109">Jos vastaanottajaluettelo on liian pitkä, lyhennä luetteloa **Alue**-alueen kenttien avulla.</span><span class="sxs-lookup"><span data-stu-id="78712-109">If the list of receipts is long, use the fields in the **Range** area to narrow the list.</span></span>

4.  <span data-ttu-id="78712-110">Etsi palautustilauksen kirjattava rivi, valitse rivin **Valitse saapumiseen** -ruutu ja valitse sitten **Aloita saapuminen**.</span><span class="sxs-lookup"><span data-stu-id="78712-110">Locate the line of the return order that you want to post, select its **Select for arrival** box, and then click **Start arrival**.</span></span>

5.  <span data-ttu-id="78712-111">Napsauta **Kirjauskansiot** \> **Näytä vastaanottojen saapumiset** avataksesi **Sijainnin kirjauskansio** -lomakkeen.</span><span class="sxs-lookup"><span data-stu-id="78712-111">Click **Journals** \> **Show arrivals from receipts** to open the **Location journal** form.</span></span>
    

    > [!TIP]
    > <P><span data-ttu-id="78712-112">Voit katsella yksityiskohtaisia tietoja valitsemalla kirjauskansion ja napsauttamalla sitten <STRONG>Rivit</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="78712-112">To view detailed information, select a journal, and then click <STRONG>Lines</STRONG>.</span></span></P>

<span data-ttu-id="78712-114">Palautetut nimikkeet rekisteröidään varastoon kirjauskansion kirjaamisen jälkeen. **Palautustilaukset**-lomake osoittaa, että nimikkeet ovat saapuneet varastoon.</span><span class="sxs-lookup"><span data-stu-id="78712-114">After the journal is posted, the returned items are registered in inventory, and the **Return orders** form indicates that the items have arrived at the warehouse.</span></span>
