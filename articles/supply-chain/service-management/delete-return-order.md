# <a name="delete-a-return-order"></a>Palautustilauksen poistaminen 

[!include [banner](../includes/banner.md)]


**Poista palautustilaus** -lomakkeella voit poistaa useita palautustilauksia.

1.  Napsauta **Myynti ja markkinointi** \> **Kausittainen** \> **Puhdista** \> **Poista palautustilaukset**.

2.  Napsauta **Valitse** avataksesi **Myyntipäivitys**-lomakkeen.

3.  Määritä ehdot ja valitse **OK** palataksesi **Poista palautustilaus**-lomakkeelle. Oikeushenkilössä luodut palautustilaukset näytetään määritettyjen ehtojen perusteella.

4.  Valitse poistettavan palautustilauksen rivi ja napsauta sitten **OK**.

5.  Poista palautustilaukset napsauttamalla **Kyllä**.
