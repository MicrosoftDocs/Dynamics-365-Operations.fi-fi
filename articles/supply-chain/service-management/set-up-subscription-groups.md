# <a name="set-up-subscription-groups"></a><span data-ttu-id="3836e-103">Määritä ylläpitosopimusryhmät</span><span class="sxs-lookup"><span data-stu-id="3836e-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="3836e-104">Valitse **Huoltohallinta** \> **Asetukset** \> **Huoltotilaukset** \> **Ylläpitosopimusryhmät**.</span><span class="sxs-lookup"><span data-stu-id="3836e-104">Go to **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="3836e-105">Valitse **Uusi** luodaksesi uuden ylläpitosopimusryhmän.</span><span class="sxs-lookup"><span data-stu-id="3836e-105">Select **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="3836e-106">Kirjoita **Ylläpitosopimusryhmä**-kenttään uutta ylläpitosopimusryhmää kuvaava nimi.</span><span class="sxs-lookup"><span data-stu-id="3836e-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="3836e-107">Valitse **Laskutus**-kentässä ylläpitosopimuksen laskutustiheys.</span><span class="sxs-lookup"><span data-stu-id="3836e-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="3836e-108">Laskutustiheys ilmaistaan kausikoodina, joka määritetään **Kausityypit**-lomakkeessa.</span><span class="sxs-lookup"><span data-stu-id="3836e-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>

5.  <span data-ttu-id="3836e-109">Jaksota ylläpitosopimustapahtumien tuotot valitsemalla **Jaksota tuotot** -valintaruutu ja valitsemalla sitten tiheys **Jaksotus**-kentässä.</span><span class="sxs-lookup"><span data-stu-id="3836e-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="3836e-110">Jaksotustiheys perustuu kausikoodiin, joka määritetään **Kaudet**-lomakkeessa.</span><span class="sxs-lookup"><span data-stu-id="3836e-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
