# <a name="set-up-return-reason-codes"></a>Palautusten syykoodien määrittäminen   

[!include [banner](../includes/banner.md)]


## <a name="set-up-return-reason-code-groups"></a>Palautusten syykoodiryhmien määrittäminen 

1.  Valitse **Myynti ja markkinointi** \> **Asetukset** \> **Myyntitilaukset** \> **Palautukset** \> **Palautuksen syykoodiryhmät**.

2.  Luo uusi rivi palautuksen syykoodiryhmälle painamalla CTRL+N-näppäinyhdistelmää.

3.  Syötä **Palautuksen syykoodiryhmä** -kenttään tämän ryhmän tunnus.

4.  Syötä **Kuvaus**-kenttään lyhyt kuvaus siitä, miten tätä ryhmää käytetään.

## <a name="set-up-return-reason-codes"></a>Palautusten syykoodien määrittäminen 

Tämän toiminnon avulla voit määrittää palautusten syykoodit, joiden avulla ilmaistaan syy, miksi asiakas palautti tuotteen.

1.  Valitse **Myynti ja markkinointi** \> **Asetukset** \> **Myyntitilaukset** \> **Palautukset** \> **Palautuksen syykoodit**.

2.  Luo uusi palautuksen syykoodirivi painamalla näppäinyhdistelmää CTRL + N.

3.  Määritä **Palautuksen syykoodi** -kenttään tätä koodia kuvaava teksti.

4.  Määritä **Kuvaus**-kenttään palautuksen syykoodia kuvaava teksti.

5.  Lisää **Palautuksen syykoodiryhmä** -pudotusvalikosta koodi palautuksen syykoodiryhmään.
