<span data-ttu-id="8f27a-110">Tämän toiminnon avulla voit määrittää palautusten syykoodit, joiden avulla ilmaistaan syy, miksi asiakas palautti tuotteen.</span><span class="sxs-lookup"><span data-stu-id="8f27a-110">This procedure helps you set up return reason codes that you can use to indicate why a product was returned by the customer.</span></span>
