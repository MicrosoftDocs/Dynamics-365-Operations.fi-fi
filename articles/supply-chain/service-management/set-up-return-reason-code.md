## <a name="set-up-return-reason-codes"></a><span data-ttu-id="30763-109">Palautusten syykoodien määrittäminen</span><span class="sxs-lookup"><span data-stu-id="30763-109">Set up return reason codes</span></span> 

<span data-ttu-id="30763-110">Tämän toiminnon avulla voit määrittää palautusten syykoodit, joiden avulla ilmaistaan syy, miksi asiakas palautti tuotteen.</span><span class="sxs-lookup"><span data-stu-id="30763-110">This procedure helps you set up return reason codes that you can use to indicate why a product was returned by the customer.</span></span>
