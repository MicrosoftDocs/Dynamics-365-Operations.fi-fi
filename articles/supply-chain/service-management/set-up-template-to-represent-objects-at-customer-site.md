# <a name="set-up-template-to-represent-objects-at-customer-site"></a><span data-ttu-id="340d5-103">Mallin määrittäminen objektien esittämiseksi asiakkaan sivulla</span><span class="sxs-lookup"><span data-stu-id="340d5-103">Set up template to represent objects at customer site</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="340d5-104">Tämän menettelyn avulla voit määrittää mallin, joka kuvaa asiakkaan toimipaikan objekteja.</span><span class="sxs-lookup"><span data-stu-id="340d5-104">Use this procedure to set up a template that represents objects at a customer site.</span></span>

1.  <span data-ttu-id="340d5-105">Valitse **Palvelunhallinta** \> **Yleinen** \> **Palvelusopimukset** \> **Palvelusopimukset**.</span><span class="sxs-lookup"><span data-stu-id="340d5-105">Click **Service management** \> **Common** \> **Service agreements** \> **Service agreements**.</span></span>

2.  <span data-ttu-id="340d5-106">Kaksoisnapsauta huoltosopimusta tai luo uusi huoltosopimus valitsemalla **Huoltosopimus**.</span><span class="sxs-lookup"><span data-stu-id="340d5-106">Double-click a service agreement, or click **Service agreement** to create a new service agreement.</span></span>

3.  <span data-ttu-id="340d5-107">Valitse **Asetukset**-välilehti.</span><span class="sxs-lookup"><span data-stu-id="340d5-107">Click the **Setup** tab.</span></span>

4.  <span data-ttu-id="340d5-108">Liitä mallituoterakenne huoltosopimukseen napsauttamalla **Huoltokohteet**.</span><span class="sxs-lookup"><span data-stu-id="340d5-108">Click **Service objects** to attach a template bill of materials (BOM) to the service agreement.</span></span>

5.  <span data-ttu-id="340d5-109">Valitse **Huoltokohteet**-lomakkeessa huoltokohde, jonka haluat liittää mallituoterakenteeseen, ja napsauta sitten **Toiminnot** \> **Liitä mallituoterakenne**.</span><span class="sxs-lookup"><span data-stu-id="340d5-109">In the **Service objects** form, select a service object, and then click **Functions** \> **Attach template BOM**.</span></span>

6.  <span data-ttu-id="340d5-110">Valitse **Valitse mallituoterakenne**-lomakkeessa mallituoterakenne **Mallituoterakenne**-kentässä ja valitse sitten **OK**.</span><span class="sxs-lookup"><span data-stu-id="340d5-110">In the **Select template BOM** form, select a template BOM in the **Template BOM** field, and then click **OK**.</span></span>

7.  <span data-ttu-id="340d5-111">Valitse **Huoltokohteet**-lomakkeesta **Suunnittelija**, jos haluat muokata mallituoterakennetta niin, että tuoterakenne vastaa haluamiasi kohteita.</span><span class="sxs-lookup"><span data-stu-id="340d5-111">In the **Service objects** form, click **Designer** to modify the template BOM so that the BOM represents the objects that you want.</span></span>

8.  <span data-ttu-id="340d5-112">Tallenna muutokset sulkemalla lomakkeet.</span><span class="sxs-lookup"><span data-stu-id="340d5-112">Close the forms to save your changes.</span></span>
