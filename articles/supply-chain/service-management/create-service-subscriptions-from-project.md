[!include [banner](../includes/banner.md)]


<span data-ttu-id="34a50-104">Voit luoda projektista huollon ylläpitosopimuksen kohdassa **Projektinhallinta ja kirjanpito**.</span><span class="sxs-lookup"><span data-stu-id="34a50-104">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="34a50-105">Valitse **Projektinhallinta ja kirjanpito** \> **Yleinen** \> **Projektit** \> **Kaikki projektit**.</span><span class="sxs-lookup"><span data-stu-id="34a50-105">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="34a50-106">Avaa projekti, jolle haluat luoda tilauksen.</span><span class="sxs-lookup"><span data-stu-id="34a50-106">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="34a50-107">**Projektit**-lomakkeen **toimintoruudussa** **Hallinta**-välilehdessä, napsauta **Huolto** \> **Huollon ylläpitosopimus**.</span><span class="sxs-lookup"><span data-stu-id="34a50-107">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

<span data-ttu-id="34a50-111">Voit nyt luoda ylläpitosopimustapahtumia ylläpitosopimukselle.</span><span class="sxs-lookup"><span data-stu-id="34a50-111">You can now create subscription transactions for the subscription.</span></span>


> [!NOTE]
> <P><span data-ttu-id="34a50-112">Jos olet määrittänyt ylläpitosopimusryhmälle numerosarjan, ylläpitosopimukselle luodaan automaattisesti ylläpitosopimuksen tunnus määrittäessäsi ylläpitosopimusryhmän.</span><span class="sxs-lookup"><span data-stu-id="34a50-112">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
