[!include [banner](../includes/banner.md)]


<span data-ttu-id="1eeb7-104">Voit luoda projektista huollon ylläpitosopimuksen kohdassa **Projektinhallinta ja kirjanpito**.</span><span class="sxs-lookup"><span data-stu-id="1eeb7-104">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="1eeb7-105">Valitse **Projektinhallinta ja kirjanpito** \> **Yleinen** \> **Projektit** \> **Kaikki projektit**.</span><span class="sxs-lookup"><span data-stu-id="1eeb7-105">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="1eeb7-106">Avaa projekti, jolle haluat luoda tilauksen.</span><span class="sxs-lookup"><span data-stu-id="1eeb7-106">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="1eeb7-107">**Projektit**-lomakkeen **toimintoruudussa** **Hallinta**-välilehdessä, napsauta **Huolto** \> **Huollon ylläpitosopimus**.</span><span class="sxs-lookup"><span data-stu-id="1eeb7-107">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

3.  <span data-ttu-id="1eeb7-108">Kirjoita **Ylläpitosopimus**-lomakkeen **Huollon ylläpitosopimus** -kenttään yksilöivä tunnus.</span><span class="sxs-lookup"><span data-stu-id="1eeb7-108">On the **Subscription** form, in the **Service subscription** field, enter a unique identification number.</span></span> <span data-ttu-id="1eeb7-109">Projektin numero lisätään automaattisesti valitun projektin ylläpitosopimukseen.</span><span class="sxs-lookup"><span data-stu-id="1eeb7-109">The project number is entered automatically into the subscription for the selected project.</span></span>

<span data-ttu-id="1eeb7-111">Voit nyt luoda ylläpitosopimustapahtumia ylläpitosopimukselle.</span><span class="sxs-lookup"><span data-stu-id="1eeb7-111">You can now create subscription transactions for the subscription.</span></span>


> [!NOTE]
> <P><span data-ttu-id="1eeb7-112">Jos olet määrittänyt ylläpitosopimusryhmälle numerosarjan, ylläpitosopimukselle luodaan automaattisesti ylläpitosopimuksen tunnus määrittäessäsi ylläpitosopimusryhmän.</span><span class="sxs-lookup"><span data-stu-id="1eeb7-112">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
