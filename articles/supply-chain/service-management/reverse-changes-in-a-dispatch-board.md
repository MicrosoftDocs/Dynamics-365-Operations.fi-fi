# <a name="reverse-changes-in-a-dispatch-board"></a><span data-ttu-id="e9cbf-103">Muutosten peruuttaminen resursointitaulussa</span><span class="sxs-lookup"><span data-stu-id="e9cbf-103">Reverse changes in a dispatch board</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="e9cbf-104">Tässä ohjeaiheessa kuvataan, kuinka resursointitaululle tehdyt tallentamattomat muutokset peruutetaan.</span><span class="sxs-lookup"><span data-stu-id="e9cbf-104">This topic describes how to reverse unsaved modifications that you make in a dispatch board.</span></span> <span data-ttu-id="e9cbf-105">Esimerkiksi määrität työntekijän palvelutehtävään, tallennat tiedot ja päätät myöhemmin määrittää toisen työntekijän palvelutehtävään.</span><span class="sxs-lookup"><span data-stu-id="e9cbf-105">For example, you assign a worker to a service activity, save the information, and then later decide to assign a different worker to the service activity.</span></span> <span data-ttu-id="e9cbf-106">Muokkaat työntekijää resursointitaulussa ja havaitset ennen muutoksen tallentamista, että juuri määritetty työntekijä ei ole käytettävissä.</span><span class="sxs-lookup"><span data-stu-id="e9cbf-106">You modify the worker in the dispatch board, and then, before saving the change, learn that the worker just assigned is not available.</span></span> <span data-ttu-id="e9cbf-107">Voit palauttaa tallentamattomat muutokset siten, että alkuperäinen työntekijä liitetään uudelleen huoltotilaukseen.</span><span class="sxs-lookup"><span data-stu-id="e9cbf-107">You can reverse the unsaved modification so that the original worker is reassigned to the service order.</span></span>

<span data-ttu-id="e9cbf-108">Seuraavien vaiheiden avulla voit perua tallentamattomat muutokset resursointitaulussa:</span><span class="sxs-lookup"><span data-stu-id="e9cbf-108">Use the following steps to reverse unsaved changes in a dispatch board:</span></span>

1.  <span data-ttu-id="e9cbf-109">Valitse **Huollon hallinta** \> **Kausittainen** \> **Resursointitaulu**.</span><span class="sxs-lookup"><span data-stu-id="e9cbf-109">Click **Service management** \> **Periodic** \> **Dispatch board**.</span></span>

2.  <span data-ttu-id="e9cbf-110">**Resursointitaulu**-lomakkeessa kirjoita tarpeelliset tiedot kenttiin ja valitse sitten **OK**.</span><span class="sxs-lookup"><span data-stu-id="e9cbf-110">In the **Dispatch board** form, enter relevant information in the fields, and then click **OK**.</span></span> 

3.  <span data-ttu-id="e9cbf-111">Voit palauttaa viimeisimmän tallentamattoman muutoksen napsauttamalla **Kumoa**.</span><span class="sxs-lookup"><span data-stu-id="e9cbf-111">To reverse the most recent change that is not saved, click **Undo**.</span></span>

4.  <span data-ttu-id="e9cbf-112">Jos haluat peruuttaa joukon tallentamattomia muutoksia, jatka **Kumoa**-painikkeen napsauttamista, kunnes haluamasi muutokset on peruutettu.</span><span class="sxs-lookup"><span data-stu-id="e9cbf-112">To reverse a series of changes that are not saved, continue clicking **Undo** until each change that you want to discard is reversed.</span></span>
