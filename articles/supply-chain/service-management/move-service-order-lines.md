# <a name="move-service-order-lines"></a><span data-ttu-id="f6f56-103">Siirrä huoltotilausrivejä</span><span class="sxs-lookup"><span data-stu-id="f6f56-103">Move service order lines</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="f6f56-104">Valitse **Huoltohallinta** \> **Yleinen** \> **Huoltotilaukset** \> **Huoltotilaukset**.</span><span class="sxs-lookup"><span data-stu-id="f6f56-104">Click **Service management** \> **Common** \> **Service orders** \> **Service orders**.</span></span>

2.  <span data-ttu-id="f6f56-105">Kaksoisnapsauta huoltotilausta, joka sisältää toiseen huoltotilaukseen siirrettävät rivit.</span><span class="sxs-lookup"><span data-stu-id="f6f56-105">Double-click the service order that contains the lines that you want to move to another service order.</span></span>

3.  <span data-ttu-id="f6f56-106">Napsauta **Huoltotilaukset**-lomakkeen toimintoruudussa olevaa **Siirrä rivit** -painiketta.</span><span class="sxs-lookup"><span data-stu-id="f6f56-106">In the **Service orders** form, on the Action Pane, click the **Move lines** button.</span></span>

4.  <span data-ttu-id="f6f56-107">Valitse **Siirrä huoltotilausrivejä** -lomakkeen **Huoltotilaus**-kentässä huoltotilaus, johon haluat siirtää rivit.</span><span class="sxs-lookup"><span data-stu-id="f6f56-107">In the **Move service order lines** form, in the **Service order** field, select the service order that you want to move the lines to.</span></span>

5.  <span data-ttu-id="f6f56-108">Valitse **Poista tyhjä huoltotilaus** -valintaruutu, jos haluat poistaa huoltotilauksen, josta siirsit huoltotilauksen rivit.</span><span class="sxs-lookup"><span data-stu-id="f6f56-108">Select the **Delete empty service order** check box to delete the service order that you moved the service order lines from.</span></span>

6.  <span data-ttu-id="f6f56-109">Napsauta **OK**.</span><span class="sxs-lookup"><span data-stu-id="f6f56-109">Click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="f6f56-110">Voit siirtää huoltotilauksen rivejä vain sellaisten huoltotilausten välillä, jotka on liitetty samaan huoltosopimukseen.</span><span class="sxs-lookup"><span data-stu-id="f6f56-110">You can move service order lines only between service orders that are attached to the same service agreement.</span></span></P>
