## <a name="credit-subscription-transactions"></a><span data-ttu-id="9e5e4-104">Ylläpitosopimustapahtumien hyvittäminen</span><span class="sxs-lookup"><span data-stu-id="9e5e4-104">Credit subscription transactions</span></span>

1.  <span data-ttu-id="9e5e4-105">Valitse **Huoltohallinta** \> **Yleinen** \> **Huoltotilaukset** \> **Kaikki huoltotilaukset**.</span><span class="sxs-lookup"><span data-stu-id="9e5e4-105">Click **Service management** \> **Common** \> **Service subscriptions** \> **All service subscriptions**.</span></span>

2.  <span data-ttu-id="9e5e4-106">Valitse siihen ylläpitosopimustapahtumaan liittyvä ylläpitosopimus, jolle hyvityslasku luodaan.</span><span class="sxs-lookup"><span data-stu-id="9e5e4-106">Select the subscription attached to the subscription transaction for which you want to create a credit note.</span></span>

3.  <span data-ttu-id="9e5e4-107">Valitse Toimintoruudussa **Analysoi**-välilehti ja valitse sitten **Maksutapahtumat**-painike.</span><span class="sxs-lookup"><span data-stu-id="9e5e4-107">Select the **Analyze** tab, and then click the **Fee transactions** button on the Action Pane.</span></span>

4.  <span data-ttu-id="9e5e4-108">Valitse **Maksutapahtumat**-lomakkeessa tapahtuma, jolle hyvityslasku luodaan.</span><span class="sxs-lookup"><span data-stu-id="9e5e4-108">From the **Fee transactions** form, select the transaction for which you want to create a credit note.</span></span>

5.  <span data-ttu-id="9e5e4-109">Napsauta **Toiminnot** \> **Valitse hyvityslaskua varten**.</span><span class="sxs-lookup"><span data-stu-id="9e5e4-109">Click **Functions** \> **Select for credit note**.</span></span>

6.  <span data-ttu-id="9e5e4-110">Valitse **Valitse hyvityslaskua varten** -lomakkeesta tapahtuma hyvitystä varten ja napsauta **OK**.</span><span class="sxs-lookup"><span data-stu-id="9e5e4-110">From the **Select for credit note** form, select the transaction that you want to credit and then click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="9e5e4-111">Kun luot hyvityslaskun, varmista, että valitset <STRONG>Hyvityslaskut</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="9e5e4-111">When you create the credit note, make sure that you select <STRONG>Credit notes</STRONG>.</span></span> <span data-ttu-id="9e5e4-112">Se on <STRONG>Laskutusmenetelmä</STRONG>-luettelossa <STRONG>Luo lasku</STRONG> -valintaikkunassa.</span><span class="sxs-lookup"><span data-stu-id="9e5e4-112">This is found in the <STRONG>Invoicing method</STRONG> list in the <STRONG>Create invoice</STRONG> dialog box.</span></span></P>

<span data-ttu-id="9e5e4-113">Jos **Palauta jaksotukset hyvitettäessä** -kentän arvo **Huoltohallinnan parametrit** -lomakkeessa on **Manuaalinen**, jokainen jaksotetun tuoton tapahtuma on palautettava erikseen, ennen kuin voit luoda tapahtuman hyvityslaskuehdotuksen.</span><span class="sxs-lookup"><span data-stu-id="9e5e4-113">If the **Reverse accruals on crediting** field in the **Service management parameters** form is set to **Manual**, you have to reverse each accrued revenue transaction individually before you create a credit note proposal for the transaction.</span></span>
