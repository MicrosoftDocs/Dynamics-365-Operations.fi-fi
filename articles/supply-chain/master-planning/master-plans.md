# <a name="master-plans-overview"></a><span data-ttu-id="49bdb-103">Pääsuunnitelmat – yleiskatsaus</span><span class="sxs-lookup"><span data-stu-id="49bdb-103">Master plans overview</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="49bdb-104">Voit käyttää useita pääsuunnitelmia. Pääsuunnitelmilla voidaan tukea yrityksen päivittäistä toimintaa, simuloida eri suunnittelustrategioita, joita halutaan seurata, sekä ottaa käyttöön esimerkiksi yrityksen sisäistä suorituskykyä tai asiakastyytyväisyyttä koskevia käytäntöjä.</span><span class="sxs-lookup"><span data-stu-id="49bdb-104">Use various master plans to support your company's daily working operations, simulate different planning strategies that you want to monitor, and implement a company policy, such as a policy about internal performance or customer satisfaction.</span></span> 

<span data-ttu-id="49bdb-105">Voit määrittää pääsuunnitelmia **Pääsuunnitelmat**-sivulla.</span><span class="sxs-lookup"><span data-stu-id="49bdb-105">You can configure master plans on the **Master plans** page.</span></span>

<span data-ttu-id="49bdb-106">Suunnitelmia on kahdentyyppisiä:</span><span class="sxs-lookup"><span data-stu-id="49bdb-106">There are two types of plans:</span></span>
-   <span data-ttu-id="49bdb-107">**Staattinen suunnitelma** - Pääsuunnittelua laskettaessa luodaan nettotarvesuunnitelma käyttämällä tämänhetkisiä tietoja.</span><span class="sxs-lookup"><span data-stu-id="49bdb-107">**Static plan** – The master planning calculation uses the current data to generate a net requirements plan.</span></span> <span data-ttu-id="49bdb-108">Tämä suunnitelma ei muutu, ennen kuin pääsuunnittelu suoritetaan seuraavan kerran tai muutat suunnitelmaa manuaalisesti.</span><span class="sxs-lookup"><span data-stu-id="49bdb-108">This plan remains unchanged until the next time that you run master planning or manually change the plan.</span></span> <span data-ttu-id="49bdb-109">Tämä on toimintasuunnitelma, jota yrityksen henkilöstö, kuten ostojen ja tuotannon suunnittelija, voi käyttää päätöstensä pohjana ja päivittäisten tehtäviensä tukena.</span><span class="sxs-lookup"><span data-stu-id="49bdb-109">This is an operating plan that various company personnel, such as a purchaser or production planner, can use to base their decisions on and perform their daily activities.</span></span>
-   <span data-ttu-id="49bdb-110">**Dynaaminen suunnitelma** – Tämä suunnitelma lähtee samasta nettotarvesuunnitelmasta, joka luotiin pääsuunnittelussa.</span><span class="sxs-lookup"><span data-stu-id="49bdb-110">**Dynamic plan** – This plan starts with the same net requirements plan that was generated by master planning.</span></span> <span data-ttu-id="49bdb-111">Voit kuitenkin päivittää dynaamisen suunnitelman aina, kun perustiedot muuttuvat.</span><span class="sxs-lookup"><span data-stu-id="49bdb-111">However, you can update the dynamic plan every time that the master data changes.</span></span> <span data-ttu-id="49bdb-112">Voit tehdä päivityksen esimerkiksi luodessasi uutta myyntitilausta.</span><span class="sxs-lookup"><span data-stu-id="49bdb-112">This could be when you create a new sales order, for example.</span></span> <span data-ttu-id="49bdb-113">Näin voit seurata muuttuvaa tilauksen verkostoa ja nimikkeiden saatavuutta sen vaikuttamatta staattiseen suunnitelmaan, jota muut käyttävät työprosesseissaan.</span><span class="sxs-lookup"><span data-stu-id="49bdb-113">This enables you to monitor the changing order network and item availability without disturbing the static plan that others are using for their work processes.</span></span>

<span data-ttu-id="49bdb-114">Yritys voi halutessaan käyttää pelkkää dynaamista suunnitelmaa tai sekä staattisia että dynaamisia suunnitelmia.</span><span class="sxs-lookup"><span data-stu-id="49bdb-114">A company may choose to work with just a dynamic plan or it may use both static and dynamic plans.</span></span> <span data-ttu-id="49bdb-115">Lisäksi mikä tahansa pääsuunnitelma voidaan konfiguroida heijastamaan tiettyä strategiaa tai ongelmakohtaa.</span><span class="sxs-lookup"><span data-stu-id="49bdb-115">In addition, you can configure any master plan to reflect a particular strategy or address an issue.</span></span> <span data-ttu-id="49bdb-116">Esimerkkejä ovat seuraavat:</span><span class="sxs-lookup"><span data-stu-id="49bdb-116">Examples are as follows:</span></span>
-   <span data-ttu-id="49bdb-117">Varastotasot asetetaan suuremmiksi varaston loppumisen välttämiseksi.</span><span class="sxs-lookup"><span data-stu-id="49bdb-117">Set higher inventory levels to guarantee against stockouts.</span></span>
-   <span data-ttu-id="49bdb-118">Varmuusmarginaalit asetetaan pidemmiksi epäluotettavien toimittajien varalta.</span><span class="sxs-lookup"><span data-stu-id="49bdb-118">Set longer safety margins to protect against unreliable vendors.</span></span>

<span data-ttu-id="49bdb-119">Alkuperäinen dynaaminen suunnitelma voidaan myös asettaa päivittymään uuden tarvesuunnitelman mukaisesti aina, kun pääsuunnittelu suoritetaan.</span><span class="sxs-lookup"><span data-stu-id="49bdb-119">You can also set up the starting dynamic plan so that it is updated with the new requirements plan every time that you run master planning.</span></span> <span data-ttu-id="49bdb-120">Voit määrittää asetukset **Pääsuunnittelun parametrit** -sivulla.</span><span class="sxs-lookup"><span data-stu-id="49bdb-120">You can specify these settings on the **Master planning parameters** page.</span></span>
