# <a name="resource-name-isnt-updated-after-changing-a-worker-name-in-the-address-book"></a>Resurssin nimeä ei päivitetä sen jälkeen, kun työntekijän nimi vaihdetaan osoitekirjassa

## <a name="symptoms"></a>Oireet

Jos työntekijän nimeä muutetaan yleisessä osoitekirjassa, vastaavaa resurssin nimeä ei päivitetä pääresurssiryhmässä.

## <a name="resolution"></a>Ratkaisu

Vastaavaa resurssin nimeä ei tällä hetkellä päivitetä automaattisesti. Ongelma korjataan päivittämällä resurssin nimi manuaalisesti.
