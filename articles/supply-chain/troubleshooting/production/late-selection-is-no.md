# <a name="late-selection-isnt-respected-when-production-orders-are-reset-via-a-batch-job"></a>Myöhäistä valintaa ei ole noudateta, kun tuotantotilaukset nollataan erätyön avulla

Tietopankin numero: 4614634

## <a name="symptoms"></a>Oireet

Kun palautat tuotantotilauksen tilan toistuvan erätyön avulla, myöhäisiä valintoja ei noudateta.

## <a name="resolution"></a>Ratkaisu

Nykyinen rakenne ei tue *Palauta tila* -prosessin myöhäisten valintojen käyttöä.
