# <a name="cant-trace-serial-numbers-of-finished-goods-against-those-of-consumed-goods"></a>Valmiiden tuotteiden sarjanumeroita ei voi jäljittää kulutettujen tuotteiden perusteella

## <a name="symptoms"></a>Oireet

Valmiiden tuotteiden sarjanumeroita ei voi jäljittää käyttämällä niiden materiaalien sarjanumeroita, joita tuotantotilaus käyttää kyseisten valmiiden tuotteiden valmistamiseen.

## <a name="workaround"></a>Ongelman kiertäminen

Ongelman voi kiertää luomalla tuotantotilauksia, joiden määrä on 1.
