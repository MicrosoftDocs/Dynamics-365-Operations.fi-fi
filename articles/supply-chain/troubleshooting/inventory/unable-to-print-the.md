# <a name="picking-list-report-doesnt-show-new-notes-that-are-attached-to-a-journal"></a>Keräysluetteloraportti ei näytä uusia kirjauskansioon liitettyjä huomautuksia

Tietopankin numero: 4611660

## <a name="symptoms"></a>Oireet

Keräysluetteloraportti ei näytä uusia keräysluettelon kirjauskansioon liitettyjä huomautuksia.

## <a name="resolution"></a>Ratkaisu

Keräysluetteloraportti sisältää vain myyntitilausriviin ja myyntitilauksen otsikkoon liitetyt huomautukset. Siinä ei näy laskukirjauskansioon lisättyjä uusia huomautuksia.
