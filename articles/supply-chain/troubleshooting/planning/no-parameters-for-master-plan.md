# <a name="parameters-for-the-master-plan-dont-exist"></a>Pääsuunnitelman parametreja ei ole

Virhekoodi: SYS25368

## <a name="symptoms"></a>Oireet

Kun yrität vahvistaa suunnitellun tilauksen, näyttöön tulee seuraava virhesanoma:

> Pääsuunnitelman %1 parametreja ei ole.

## <a name="cause"></a>Syy

Konfigurointiongelmien takia järjestelmä ei löydä määritetyn suunnitelman asetuksia.

## <a name="resolution"></a>Ratkaisu

- Siirry kohtaan **Pääsuunnittelu \> Asetukset \> Suunnitelmat \> Pääsuunnitelmat** ja varmista, että suunnitelmalla on määritetty nimi.
