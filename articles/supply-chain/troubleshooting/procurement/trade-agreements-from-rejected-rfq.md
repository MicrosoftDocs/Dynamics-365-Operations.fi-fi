# <a name="trade-agreements-can-be-created-from-rejected-rfqs"></a>Kauppasopimuksia voi luoda hylätyistä tarjouspyynnöistä

## <a name="symptoms"></a>Oireet

Kauppasopimuksia voi luoda hylätyistä tarjouspyynnöistä. Siksi järjestelmä ei estä kauppasopimusten kirjauskansioiden luomista, jos tarjouspyyntöriviä ei ole hyväksytty.

## <a name="resolution"></a>Ratkaisu

Tämä toiminta on odotettua. Voit luoda kauppasopimuksia mille tahansa tarjouspyynnön vastaukselle riippumatta siitä, hyväksyttiinkö vai hylättiinkö vastaus. Lisätietoja: [Tarjouspyyntöjen yleiskatsaus](/dynamics365/supply-chain/procurement/request-quotations).
