# <a name="you-cant-add-a-line-to-a-purchase-requisition-after-you-request-a-change"></a>Et voi lisätä riviä ostoehdotukseen muutoksen pyytämisen jälkeen

Tietopankin numero: 4611211

## <a name="symptoms"></a>Oireet

Järjestelmä ei salli rivin lisäämistä ostoehdotukseen muutoksen pyytämisen jälkeen.

## <a name="resolution"></a>Ratkaisu

Peruuta työnkulku. Kun ostoehdotuksen tilana on *Luonnos*, voit jatkaa sen muokkaamista tai lisätä siihen rivin.
