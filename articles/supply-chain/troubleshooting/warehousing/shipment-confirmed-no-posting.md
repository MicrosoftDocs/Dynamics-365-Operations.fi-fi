# <a name="shipment-for-load-has-been-confirmed-but-no-lines-are-posted"></a>Kuorman lähetys on vahvistettu, mutta rivejä ei ole kirjattu

## <a name="symptoms"></a>Oireet

Kun käytät lähteviä varastotoimintoja, voit saada seuraavan sanoman:

> Kuorman 1% lähetys on vahvistettu.

Uusia kirjauksia ei kuitenkaan ole tapahtunut.

## <a name="resolution"></a>Ratkaisu

Tämä on suunniteltu ominaisuus. Lähetyksen vahvistus ei vaikuta kirjaukseen. Se vain päivittää lähetyksen ja kuorman tilan. Kirjaus on tehtävä erillisessä prosessissa.
