# <a name="packing-station-doesnt-show-product-notes"></a>Pakkausasema ei näytä tuotehuomautuksia

Tietopankin numero: 4614615

## <a name="symptoms"></a>Oireet

Pakkaushuomautuksia ei näytetä pakkauslomakkeessa, kun pakkausohjeet lisätään tuotteen pää- tai tuotevariantin liitteeksi.

## <a name="resolution"></a>Ratkaisu

Järjestelmä käyttäytyy suunnitellulla tavalla.

Pakkaushuomautusten nykyinen näyttämisen logiikka pakkauslomakkeessa edellyttää, että huomautukset on liitetty lähetyksiin.
