# <a name="quantity-is-not-valid-when-performing-a-split-pick-across-multiple-batches"></a>Määrä ei kelpaa, kun suoritetaan jaettu keräily useissa erissä

## <a name="symptoms"></a>Oireet

Kun yrität suorittaa *jaetun keräilyn*, johon liittyy useita eriä, näkyviin tulee seuraava virhesanoma:

> Yksikön %1 määrä ei kelpaa.

## <a name="resolution"></a>Ratkaisu

Varastotyöntekijän on käytettävä *lyhyt keräily* -prosessia Warehouse Managementin mobiilisovelluksessa. Jos yrität kerätä useita eriä samasta sijainnista, voit käyttää myös sovelluksen **Täysi**-vaihtoehdolla.
