# <a name="invalid-license-plate-error-when-scanning-license-plate-id-in-the-mobile-app"></a>Virheellinen rekisterikilpi -virhe luettaessa rekisterikilven tunnusta mobiilisovelluksessa

## <a name="symptoms"></a>Oireet

Kun luet rekisterikilven tunnuksen Warehouse Managementin mobiilisovelluksessa, seuraava virhesanoma saattaa avautua:

> Virheellinen rekisterikilpi.

## <a name="resolution"></a>Ratkaisu

Varmista, että rekisterikilven tunnus on rekisterikilpitaulukossa ja että rekisterikilven nimikkeet ja määrät ovat käytettävissä (eli niitä ei ole estetty).
