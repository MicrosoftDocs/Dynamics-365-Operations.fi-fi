# <a name="cant-move-inventory-to-a-location-that-is-license-plate-controlled"></a>Varastoa ei voi siirtää rekisterikilpiohjattuun sijaintiin

## <a name="symptoms"></a>Oireet

Kuormituksen kerättyjä määriä ei voi vähentää.

## <a name="resolution"></a>Ratkaisu

Aiemmissa versioissa kuormituksen kerättyjä määriä ei voinut vähentää. Voit nyt kumota keräyksen rekisterikilpiohjattuun sijaintiin. Sekä **Sijainti**- että **Rekisterikilpi**-arvo on määritettävä kuormariville **Vähennä keräiltyä määrää** -sivulla.
