# <a name="scale-unit-capability-for-supply-chain-management-is-currently-paused-for-new-customers"></a>Supply Chain Managementin mittayksikön skaalaustoiminto on tällä hetkellä keskeytetty uusien asiakkaiden osalta

[!include [banner](../includes/banner.md)]

> [!IMPORTANT]
> Microsoft Dynamics 365 Supply Chain Managementin scale unit -ominaisuuden keskeytetään uusien asiakkaiden osalta, sillä arkkitehtuuri parannetaan. Vaikka tähän ratkaisuun ei oteta tällä hetkellä uusia asiakkaita, toimitusketjutoimialan arkkitehtuuria kehitetään laadinnan, joustavuuden ja helppokäyttöisyyden osalta. Tämä parantaa mahdollisuutta tuottaa huippuluokan scale unit -ratkaisu asiakkaille.
>
> Dynamics 365 Supply Chain Managementia parannetaan siten, että tiettyjen moduulien, kuten varastonhallinnan, laadittavuus paranee. Asiakkaiden tarpeita vastaavia pilvipalvelujen ja reunan scale unit -tuotteiden suunnittelua ja kehittämistä jatketaan. Lisätietoja voi pyytää osoitteesta [d365scmcneext@microsoft.com](mailto:d365scmcneext@microsoft.com).
>
> Microsoft Dynamics 365 Commercen Commerce Scale Unit (CSU) Core -ympäristö ei liity millään tavalla Dynamics 365 Supply Chain Managementin scale unit -ominaisuuteen. Tämä ilmoitus ei vaikuta CSU Core -ympäristöön, ja sitä tuetaan edelleen Dynamics 365 Commerce -sovellukselle. Lisätietoja tästä ominaisuudesta on kohdassa [Commerce Scale Unit (CSU) Core -opas](../../commerce/dev-itpro/CSU-core.md).
