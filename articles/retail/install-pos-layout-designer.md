2.  <span data-ttu-id="d1d73-111">Valitse asettelu, jonka sovellustyyppi on **Modern POS Windowsille** tai **Vähittäismyynnin pilvimyyntipiste**, ja valitse sitten **asettelun suunnittelutoiminto**.</span><span class="sxs-lookup"><span data-stu-id="d1d73-111">Select any layout that has an application type of **Modern POS for Windows** or **Cloud POS**, and then click **Layout designer**.</span></span>
