[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="75efe-104">Kustannuslaskennan kirjanpito edustaa koko raportointiyksikköä.</span><span class="sxs-lookup"><span data-stu-id="75efe-104">A cost accounting ledger represents the overall reporting unit.</span></span> <span data-ttu-id="75efe-105">Se määritetään kustannustason dimension, tilastodimension, kirjanpidon vuosikalenterin ja valuutan mukaan.</span><span class="sxs-lookup"><span data-stu-id="75efe-105">It is defined by a cost element dimension, statistical dimension, fiscal calendar, and currency.</span></span> <span data-ttu-id="75efe-106">Se ei riipu yrityksen konseptista.</span><span class="sxs-lookup"><span data-stu-id="75efe-106">It is agnostic to the concept of legal entities.</span></span> <span data-ttu-id="75efe-107">Yritys ja sen tiedot voidaan liittää useisiin kustannuslaskennan kirjanpitoihin.</span><span class="sxs-lookup"><span data-stu-id="75efe-107">A legal entity and its data can be associated with many cost accounting ledgers.</span></span> <span data-ttu-id="75efe-108">Tämä tallenne käyttää esittelytietojen USP2-yritystä.</span><span class="sxs-lookup"><span data-stu-id="75efe-108">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="75efe-109">Valitse Kustannuslaskenta > Kirjanpidon asetukset > Kustannuslaskennan kirjanpidot.</span><span class="sxs-lookup"><span data-stu-id="75efe-109">Go to Cost accounting > Ledger setup > Cost accounting ledgers.</span></span>
2. <span data-ttu-id="75efe-110">Valitse Uusi.</span><span class="sxs-lookup"><span data-stu-id="75efe-110">Click New.</span></span>
3. <span data-ttu-id="75efe-111">Kirjoita arvo Nimi-kenttään.</span><span class="sxs-lookup"><span data-stu-id="75efe-111">In the Name field, type a value.</span></span>

5. <span data-ttu-id="75efe-113">Syötä tai valitse arvo Kirjanpidon vuosikalenteri -kenttään.</span><span class="sxs-lookup"><span data-stu-id="75efe-113">In the Fiscal calendar field, enter or select a value.</span></span>
6. <span data-ttu-id="75efe-114">Syötä tai valitse arvo Kirjanpitovaluutta-kenttään.</span><span class="sxs-lookup"><span data-stu-id="75efe-114">In the Accounting currency field, enter or select a value.</span></span>
7. <span data-ttu-id="75efe-115">Syötä tai valitse arvo Vaihtokurssin tyyppi -kenttään.</span><span class="sxs-lookup"><span data-stu-id="75efe-115">In the Exchange rate type field, enter or select a value.</span></span>
8. <span data-ttu-id="75efe-116">Syötä tai valitse arvo Dimensio-kenttään.</span><span class="sxs-lookup"><span data-stu-id="75efe-116">In the Statistical dimension field, enter or select a value.</span></span>
9. <span data-ttu-id="75efe-117">Valitse Tallenna.</span><span class="sxs-lookup"><span data-stu-id="75efe-117">Click Save.</span></span>
