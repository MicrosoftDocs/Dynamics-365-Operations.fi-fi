[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="4b491-104">Kustannuslaskennan kirjanpito edustaa koko raportointiyksikköä.</span><span class="sxs-lookup"><span data-stu-id="4b491-104">A cost accounting ledger represents the overall reporting unit.</span></span> <span data-ttu-id="4b491-105">Se määritetään kustannustason dimension, tilastodimension, kirjanpidon vuosikalenterin ja valuutan mukaan.</span><span class="sxs-lookup"><span data-stu-id="4b491-105">It is defined by a cost element dimension, statistical dimension, fiscal calendar, and currency.</span></span> <span data-ttu-id="4b491-106">Se ei riipu yrityksen konseptista.</span><span class="sxs-lookup"><span data-stu-id="4b491-106">It is agnostic to the concept of legal entities.</span></span> <span data-ttu-id="4b491-107">Yritys ja sen tiedot voidaan liittää useisiin kustannuslaskennan kirjanpitoihin.</span><span class="sxs-lookup"><span data-stu-id="4b491-107">A legal entity and its data can be associated with many cost accounting ledgers.</span></span> <span data-ttu-id="4b491-108">Tämä tallenne käyttää esittelytietojen USP2-yritystä.</span><span class="sxs-lookup"><span data-stu-id="4b491-108">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="4b491-109">Valitse Kustannuslaskenta > Kirjanpidon asetukset > Kustannuslaskennan kirjanpidot.</span><span class="sxs-lookup"><span data-stu-id="4b491-109">Go to Cost accounting > Ledger setup > Cost accounting ledgers.</span></span>
2. <span data-ttu-id="4b491-110">Valitse Uusi.</span><span class="sxs-lookup"><span data-stu-id="4b491-110">Click New.</span></span>
3. <span data-ttu-id="4b491-111">Kirjoita arvo Nimi-kenttään.</span><span class="sxs-lookup"><span data-stu-id="4b491-111">In the Name field, type a value.</span></span>
4. <span data-ttu-id="4b491-112">Syötä tai valitse arvo Kustannustason dimensio -kenttään.</span><span class="sxs-lookup"><span data-stu-id="4b491-112">In the Cost element dimension field, enter or select a value.</span></span>
5. <span data-ttu-id="4b491-113">Syötä tai valitse arvo Kirjanpidon vuosikalenteri -kenttään.</span><span class="sxs-lookup"><span data-stu-id="4b491-113">In the Fiscal calendar field, enter or select a value.</span></span>
6. <span data-ttu-id="4b491-114">Syötä tai valitse arvo Kirjanpitovaluutta-kenttään.</span><span class="sxs-lookup"><span data-stu-id="4b491-114">In the Accounting currency field, enter or select a value.</span></span>
7. <span data-ttu-id="4b491-115">Syötä tai valitse arvo Vaihtokurssin tyyppi -kenttään.</span><span class="sxs-lookup"><span data-stu-id="4b491-115">In the Exchange rate type field, enter or select a value.</span></span>
8. <span data-ttu-id="4b491-116">Syötä tai valitse arvo Dimensio-kenttään.</span><span class="sxs-lookup"><span data-stu-id="4b491-116">In the Statistical dimension field, enter or select a value.</span></span>
9. <span data-ttu-id="4b491-117">Valitse Tallenna.</span><span class="sxs-lookup"><span data-stu-id="4b491-117">Click Save.</span></span>
