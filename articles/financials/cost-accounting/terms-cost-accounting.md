<span data-ttu-id="be525-105">**Kohdistusperuste**</span><span class="sxs-lookup"><span data-stu-id="be525-105">**Allocation base**</span></span>

<span data-ttu-id="be525-106">Kohdistusperustetta käytetään tehtävien mittaamiseen esimerkiksi koneen käyttötunteina, kulutettuina kilowattitunteina tai käytössä olevina neliömetreinä.</span><span class="sxs-lookup"><span data-stu-id="be525-106">The allocation base is used to measure and quantify activities, such as machine hours that are used, kilowatt hours that are consumed, or square footage that is occupied.</span></span> <span data-ttu-id="be525-107">Sitä käytetään perusteena kohdistettaessa kustannuksia yhdelle tai useammalle kustannusobjektille.</span><span class="sxs-lookup"><span data-stu-id="be525-107">It's used as basis for allocating costs to one or more cost objects</span></span>

<span data-ttu-id="be525-108">**Kustannuslaskenta**</span><span class="sxs-lookup"><span data-stu-id="be525-108">**Cost accounting**</span></span>

<span data-ttu-id="be525-109">Kustannuslaskentaan voidaan kerätä tietoja useista eri lähteistä, kuten kirjanpidosta, alareskontrista, budjeteista ja tilastotiedoista.</span><span information.</span></span> <span data-ttu-id="be525-110">Kustannustietoja voidaan analysoida, arvioida ja tehdä yhteenvetoja, jotta yritysjohto voi tehdä tarvittavat päätökset esimerkiksi kustannuslaskennassa.</span><span class="sxs-lookup"><span data-stu-id="be525-113">However, when you collect cost data from various sources, and especially when you import the main accounts from <span data-ttu-id="be525-114">Toistoa tarvitaan, koska taloushallintoa käytetään ulkoiseen raportointiin ja kustannuslaskentaa sisäistä raportointia varten.</span><span class="sxs-lookup"><span <span data-ttu-id="be525-115">**Kustannuslaskennan kirjanpito**</span><span class="sxs-lookup"><span data-stu-id="be525-115">**Cost accounting ledger**</span></span>

<span data-ttu-id="be525-116">Se on määritetty kalenteri-, valuutta- ja kustannuselementin dimensioperustaisesti, ja se hallitsee kustannuslaskennan prosesseja ja käytäntöjä.</span><span class="sxs-lookup"><span data-stu-id="be525-116">Defined by calendar, currency, and cost element dimension, it controls processes and policies for measuring costs.</span></span> 

<span data-ttu-id="be525-117">**Kustannusmerkintä**</span><span class="sxs-lookup"><span data-stu-id="be525-117">**Cost entry**</span></span>

<span data-ttu-id="be525-118">Kustannustapahtumat muodostuvat siirroista, jotka tehdään tietoyhdistimillä kirjanpidon merkinnöistä, kustannusten kohdistuksista ja kustannusten kirjauskansioihin kirjatuista kustannustapahtumista.</span><span class="sxs-lookup"><span data-stu-id="be525-118">Cost entries are the result of a transfer via data connectors from general ledger entries, cost allocations, and posted cost entries in cost journals.</span></span>

<span data-ttu-id="be525-119">**Kustannusobjekti**</span><span class="sxs-lookup"><span data-stu-id="be525-119">**Cost object**</span></span>

<span data-ttu-id="be525-120">Kaikki kustannusseurantaan valitut objektit.</span><span class="sxs-lookup"><span data-stu-id="be525-120">Any type of object that is selected for cost control.</span></span> <span data-ttu-id="be525-121">Kustannukset tai tuotot kirjataan joko suoraan tai kohdistetaan kustannusobjekteihin.</span><span class="sxs-lookup"><span data-stu-id="be525-121">Costs or revenues are either directly posted on or allocated to cost objects.</span></span> <span data-ttu-id="be525-122">Tyypillisiä kustannusobjekteja ovat:</span><span class="sxs-lookup"><span data-stu-id="be525-122">Some typical cost objects are:</span></span>

-  <span data-ttu-id="be525-123">Tuotteet</span><span class="sxs-lookup"><span data-stu-id="be525-123">Products</span></span>
-  <span data-ttu-id="be525-124">Projektit</span><span class="sxs-lookup"><span data-stu-id="be525-124">Projects</span></span>
-  <span data-ttu-id="be525-125">Osastot</span><span class="sxs-lookup"><span data-stu-id="be525-125">Departments</span></span>
-  <span data-ttu-id="be525-126">Kustannuspaikat</span><span class="sxs-lookup"><span data-stu-id="be525-126">Cost centers</span></span>

<span data-ttu-id="be525-127">Yritysjohto käyttää kustannusobjekteja kustannusten määrittämiseen ja kannattavuusanalyysin tekemiseen.</span><span class="sxs-lookup"><span data-stu-id="be525-127">Management uses cost objects to quantify costs, but also to drive profitability analysis.</span></span>

<span data-ttu-id="be525-128">**Kustannustaso**</span><span class="sxs-lookup"><span data-stu-id="be525-128">**Cost element**</span></span>

<span data-ttu-id="be525-129">Käytetään toimintona kustannusten seurannassa ja luokittelussa.</span><span class="sxs-lookup"><span data-stu-id="be525-129">Used as a function to track and categorize costs.</span></span> <span data-ttu-id="be525-130">Kustannustasoja on kahdenlaisia: ensisijaiset ja toissijaiset kustannukset.</span><span class="sxs-lookup"><span data-stu-id="be525-130">There are two types of cost elements: primary and secondary.</span></span>

<span data-ttu-id="be525-131">Ensisijaiset kustannustasot edustavat kustannusvirtaa kirjanpidosta kustannuslaskentaan.</span><span class="sxs-lookup"><span data-stu-id="be525-131">Primary cost elements represent the cost flow from financial accounting to Cost accounting.</span></span> <span data-ttu-id="be525-132">Rakenne vastaa yleensä kirjanpidon tuloksen tilirakennetta, jossa kustannustaso voi vastata päätiliä.</span><span class="sxs-lookup"><span data-stu-id="be525-132">The structure typically corresponds to the profit and loss account structure in the general ledger where a cost element can correspond to a main account.</span></span> <span data-ttu-id="be525-133">Liiketoiminnan vaatimusten mukaisesti kaikkia päätilejä ei voida merkitä kustannustasoiksi.</span><span class="sxs-lookup"><span data-stu-id="be525-133">Not all main accounts must be represented as cost elements, depending on business requirements.</span></span> 

<span data-ttu-id="be525-134">Toissijaiset kustannustasot edustavat sisäistä kustannusvirtaa, koska näitä kustannuksia käytetään ainoastaan kustannuslaskennassa.</span><span class="sxs-lookup"><span data-stu-id="be525-134">Secondary cost elements represent the internal cost flow because these costs are only used in Cost accounting.</span></span> <span data-ttu-id="be525-135">Niitä käytetään kustannusten koontisäännöissä niiden kokoamiseksi kuvaileviin jakaumiin, joita käytetään yleiskustannusten laskennassa.</span><span class="sxs-lookup"><span data-stu-id="be525-135">They are used in cost roll-up rules to aggregate costs into meaningful buckets used by overhead calculation.</span></span> 

<span data-ttu-id="be525-136">**Kustannusluokittelu**</span><span class="sxs-lookup"><span data-stu-id="be525-136">**Cost classification**</span></span>

-   <span data-ttu-id="be525-139">**Tasot**: materiaali, työ ja kulut.</span><span class="sxs-lookup"><span data-stu-id="be525-139">**By elements** – Materials, labor, and expenses.</span></span>
-   <span data-ttu-id="be525-140">**Jäljitettävyys**: välittömät ja välilliset kustannukset.</span><span class="sxs-lookup"><span data-stu-id="be525-140">**By traceability** – Direct costs and indirect costs.</span></span> <span data-ttu-id="be525-141">Välittömät kustannukset määritetään kustannusobjekteiksi.</span><span class="sxs-lookup"><span data-stu-id="be525-141">Direct costs are assigned directly to cost objects.</span></span> <span data-ttu-id="be525-142">Välilliset kustannukset eivät ole suoraan jäljitettävissä kustannusobjekteihin.</span><span class="sxs-lookup"><span data-stu-id="be525-142">Indirect costs aren't directly traceable to cost objects.</span></span> <span data-ttu-id="be525-143">Epäsuorat kustannukset kohdistetaan kustannusobjekteihin.</span><span class="sxs-lookup"><span data-stu-id="be525-143">Indirect costs are allocated to cost objects.</span></span>
-   <span data-ttu-id="be525-144">**Käyttäytyminen**: kiinteät, muuttuvat ja puolimuuttuvat.</span><span class="sxs-lookup"><span data-stu-id="be525-144">**By behavior** – Fixed, variable, and semi-variable.</span></span>

<span data-ttu-id="be525-145">**Kustannusten käyttäytyminen**</span><span class="sxs-lookup"><span data-stu-id="be525-145">**Cost behavior**</span></span>

<span data-ttu-id="be525-146">Kustannusten käyttäytyminen luokittelee kustannukset sen mukaan, miten ne käyttäytyvät suhteessa keskeisen liiketoiminnan muutoksiin.</span><span class="sxs-lookup"><span data-stu-id="be525-146">Cost behavior classifies costs according to their behavior in relation to changes in key business activities.</span></span> <span data-ttu-id="be525-147">Tehokas kustannusten hallinta edellyttää yrityksen johdolta kustannusten käyttäytymisen ymmärtämistä.</span><span class="sxs-lookup"><span data-stu-id="be525-147">To control costs effectively, management must understand the cost behavior.</span></span> <span data-ttu-id="be525-148">Kustannusten käyttäytyminen jaetaan kolmeen tyyppiin: kiinteät, muuttuvat ja puolimuuttuvat kustannukset.</span><span class="sxs-lookup"><span data-stu-id="be525-148">There are three types of cost behavior pattern: fixed, variable, and semi-variable.</span></span>

- <span data-ttu-id="be525-149">**Kiinteä kustannus** - Kiinteä kustannus tarkoittaa kustannuksia, jotka eivät vaihtele lyhyellä aikavälillä riippumatta tuotannon muutoksista.</span><span class="sxs-lookup"><span data-stu-id="be525-149">**Fixed cost** - A fixed cost is a cost that doesn't vary in the short term, regardless of changes in activity level.</span></span> <span data-ttu-id="be525-150">Kiinteät kustannukset voivat olla esimerkiksi yrityksen perustoimintojen kuluja, kuten vuokra, johon tuotannon kasvu tai lasku ei vaikuta.</span><span class="sxs-lookup"><span data-stu-id="be525-150">For example, a fixed cost can be a basic operating expense of a business, such as rent, that won't be affected even if the activity level increases or decreases.</span></span>

- <span data-ttu-id="be525-151">**Muuttuva kustannut** - Muuttuva kustannus vaihtelee toiminnan muutosten mukaan.</span><span class="sxs-lookup"><span data-stu-id="be525-151">**Variable cost** - A variable cost changes according to changes in activity level.</span></span> <span data-ttu-id="be525-152">Esimerkiksi tietyt suorat materiaalikustannukset liittyvät jokaiseen myytävään tuotteeseen.</span><span class="sxs-lookup"><span data-stu-id="be525-152">For example, a specific direct materials cost is associated with each product that is sold.</span></span> <span data-ttu-id="be525-153">Materiaalikustannukset ovat sitä suuremmat, mitä enemmän tuotteita myydään.</span><span class="sxs-lookup"><span data-stu-id="be525-153">The more products that are sold, the more direct materials costs there are.</span></span>

- <span data-ttu-id="be525-154">**Puolimuuttuva kustannus** - Puolimuuttuvat kustannukset ovat osittain kiinteitä ja osittain muuttuvia kustannuksia.</span><span class="sxs-lookup"><span data-stu-id="be525-154">**Semi-variable cost** - Semi-variable costs are partly fixed and partly variable costs.</span></span> <span data-ttu-id="be525-155">Esimerkiksi Internetin käyttökustannukset sisältävät kuukausittaisen perusmaksun ja laajakaistan käyttömaksun.</span><span class="sxs-lookup"><span data-stu-id="be525-155">For example, an Internet access fee includes a standard monthly access fee and a broadband usage fee.</span></span> <span data-ttu-id="be525-156">Kuukausittainen perusmaksu on kiinteä kustannus, kun taas laajakaistan käyttömaksu on muuttuva kustannus.</span><span class="sxs-lookup"><span data-stu-id="be525-156">The standard monthly access fee is a fixed cost, whereas the broadband usage fee is a variable cost.</span></span>

<span data-ttu-id="be525-157">**Kustannusseurantayksikkö**</span><span class="sxs-lookup"><span data-stu-id="be525-157">**Cost control unit**</span></span>

<span data-ttu-id="be525-158">Kustannusseurantayksikkö edustaa kustannusrakennetta.</span><span class="sxs-lookup"><span data-stu-id="be525-158">The cost control unit represents the cost structure.</span></span> <span data-ttu-id="be525-159">Rakenne määrää, miten kustannukset jakautuvat hierarkkisessa järjestyksessä kustannusobjektin dimensioiden ja niitä vastaavien kustannusobjektien välillä.</span><span class="sxs-lookup"><span data-stu-id="be525-159">The structure determines how cost flows in a hierarchical order between cost object dimensions and their respective cost objects.</span></span> 

<span data-ttu-id="be525-160">**Kustannusten jako**</span><span class="sxs-lookup"><span data-stu-id="be525-160">**Cost distribution**</span></span>

<span data-ttu-id="be525-161">Sitä käytetään kustannusten jakamiseen yhdestä kustannusobjektista toiseen (tai useampaan kustannusobjektiin) määrittämällä asiaankuuluvan kohdistusperusteen.</span><span class="sxs-lookup"><span data-stu-id="be525-161">Is used to distribute cost from one cost object to one or more other cost objects by applying a relevant allocation base.</span></span> <span data-ttu-id="be525-162">Kustannusten jako ja kustannusten kohdistus eroavat toisistaan sillä, että kustannusten jako tapahtuu aina alkuperäisen kustannuksen ensisijaisen kustannuselementin tasolla.</span><span class="sxs-lookup"><span data-stu-id="be525-162">Cost distribution and cost allocation differ in that cost distribution always occurs at the level of the primary cost element of the original cost and no reciprocal processing.</span></span>

<span data-ttu-id="be525-163">**Kustannusten kohdistus**</span><span class="sxs-lookup"><span data-stu-id="be525-163">**Cost allocation**</span></span>

<span data-ttu-id="be525-171">**Kustannusten kohdistuskäytäntö**</span><span class="sxs-lookup"><span data-stu-id="be525-171">**Cost allocation policy**</span></span>

<span data-ttu-id="be525-175">**Kustannusten koonti**</span><span class="sxs-lookup"><span data-stu-id="be525-175">**Cost roll-up**</span></span>

<span data-ttu-id="be525-176">Kustannusten koontilaskelmia käytetään kustannusten kokoamiseksi kuvaileviin jakaumiin.</span><span class="sxs-lookup"><span data-stu-id="be525-176">The purpose of cost roll-up rules is to aggregate costs into meaningful buckets.</span></span> <span data-ttu-id="be525-177">Koosteen taso on käyttäjän määrittämä ja siihen liittyy toissijaisen kustannus elementin määrittäminen.</span><span class="sxs-lookup"><span data-stu-id="be525-177">The level of aggregation is user-defined and involves the assignment of a secondary cost element.</span></span> <span data-ttu-id="be525-178">Jos kustannusten koontilaskelmia ei käytetä, jokainen kustannustaso kohdistetaan kustannusobjektista toiseen.</span><span class="sxs-lookup"><span data-stu-id="be525-178">If cost roll-up is not used, every element of a cost is allocated from one cost object to another</span></span>

<span data-ttu-id="be525-179">**Kustannushintakäytäntö**</span><span class="sxs-lookup"><span data-stu-id="be525-179">**Cost rate policy**</span></span>
