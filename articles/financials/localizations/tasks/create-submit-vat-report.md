# <a name="create-and-submit-vat-report"></a>ALV-raportin luominen ja lähettäminen

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

Tämä menettely selittää, miten ALV-raportti luodaan ja lähetetään Saksan viranomaisille.

Tämä menettelyn luomisessa käytettiin DEMF-yrityksen demotietoja. 

Toiminto on niiden yritysten käytettävissä, joiden ensisijainen osoite on Saksassa.



1. Valitse Vero > Ilmoitukset > Arvonlisävero > Ilmoita arvonlisävero tilityskaudelle.
2. Syötä tai valitse arvo kentässä Tilityskausi.
3. Syötä päivämäärä Päivämäärästä-kenttään.
4. Valitse OK.
5. Valitse Luo sähköinen verotiedosto -kentässä Kyllä.
6. Valitse OK.
