# <a name="create-a-credit-note-on-project-invoices"></a>Hyvityslasku luominen projektilaskuille

[!include [task guide banner](../../includes/task-guide-banner.md)]

1. Valitse Projektinhallinta ja kirjanpito > Projektit > Kaikki projektit. 
2. Etsi haluamasi tietue luettelosta ja valitse se. 
3. Valitse hyvityslaskussa käytettävä projektitunnus. Napsauta luettelossa valitulla rivillä olevaa linkkiä. 
4. Valitse toimintoruudussa Hallitse. 
5. Valitse Laskukirjauskansiot. 
6. Valitse hyvityslaskun tapahtuma. Merkitse valittu rivi luettelossa. 
7. Valitse Toiminnot. 
8. Napsauta Valitse hyvityslaskua varten -kohtaa. 
9. Valitse hyvityslaskun tapahtuma merkitsemällä valintaruutu.
10. Valitse OK. 

Tämän jälkeen hyvityslasku on käytettävissä laskun ehdotusta varten.
