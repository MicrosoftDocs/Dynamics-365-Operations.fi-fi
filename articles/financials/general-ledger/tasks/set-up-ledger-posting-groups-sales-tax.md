[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="19c1d-104">Arvonlisävero lasketaan ja kirjataan kirjanpidon kirjausryhmien päätileille.</span><span class="sxs-lookup"><span data-stu-id="19c1d-104">Sales tax is calculated and posted to main accounts that are specified in the Ledger posting groups.</span></span> <span data-ttu-id="19c1d-105">Kirjanpidon kirjausryhmät liitetään kuhunkin arvonlisäverokoodiin.</span><span class="sxs-lookup"><span data-stu-id="19c1d-105">Ledger posting groups are attached to each sales tax code.</span></span> <span data-ttu-id="19c1d-106">Voit määrittää kullekin arvonlisäverokoodille kirjanpidon kirjausryhmät käyttämällä kaikissa arvonlisäverokoodeissa yhtä kirjanpidon kirjausryhmää tai liittämällä arvonlisäverokoodeille useita kirjanpidon kirjausryhmiä.</span><span class="sxs-lookup"><span data-stu-id="19c1d-106">You can set up individual ledger posting groups for each sales tax code, use one ledger posting group for all sales tax codes or assign multiple ledger posting groups to the sales tax codes.</span></span> <span data-ttu-id="19c1d-107">Tässä tallenteessa käytetään esittely-yritystä DEMF.</span><span class="sxs-lookup"><span data-stu-id="19c1d-107">This recording uses the DEMF demo company.</span></span> 

1. <span data-ttu-id="19c1d-108">Siirry kohtaan Vero > Asetukset > Arvonlisävero > Kirjanpidon kirjausryhmät.</span><span class="sxs-lookup"><span data-stu-id="19c1d-108">Go to Tax > Setup > Sales tax > Ledger posting groups.</span></span>
2. <span data-ttu-id="19c1d-109">Valitse Uusi.</span><span class="sxs-lookup"><span data-stu-id="19c1d-109">Click New.</span></span>
3. <span data-ttu-id="19c1d-110">Kirjoita arvo Kirjanpidon kirjausryhmä -kenttään.</span><span class="sxs-lookup"><span data-stu-id="19c1d-110">In the Ledger posting group field, type a value.</span></span>
4. <span data-ttu-id="19c1d-111">Kirjoita arvo Kuvaus-kenttään.</span><span class="sxs-lookup"><span data-stu-id="19c1d-111">In the Description field, type a value.</span></span>
5. <span data-ttu-id="19c1d-112">Valitse Maksettava arvonlisävero -kentässä päätili lähtevälle arvonlisäverolle, joka maksetaan veroviranomaiselle.</span><span class="sxs-lookup"><span data-stu-id="19c1d-112">In the Sales tax payable field, select the main account for outgoing sales taxes that are payable to the tax authority.</span></span>
    * <span data-ttu-id="19c1d-113">Arvonlisäverot kerätään veroviranomaisen puolesta, kun verollisia tavaroita ja palveluja myydään.</span><span class="sxs-lookup"><span data-stu-id="19c1d-113">Sales taxes are collected on behalf of the tax authority when you sell taxable goods and services.</span></span>  
6. <span data-ttu-id="19c1d-114">Valitse Saatava arvonlisävero -kentässä päätili saapuvalle arvonlisäverolle, joka vastaanotetaan veroviranomaiselta.</span><span class="sxs-lookup"><span data-stu-id="19c1d-114">In the Sales tax receivable filed, select the main account for incoming taxes that are received from the tax authority.</span></span>
    * <span data-ttu-id="19c1d-115">Toimittajat keräävät verot veroviranomaisen puolesta, kun verollisia tavaroita ja palveluja ostetaan.</span><span class="sxs-lookup"><span data-stu-id="19c1d-115">Vendors collect taxes on behalf of the tax authority when you buy taxable goods and services.</span></span> <span data-ttu-id="19c1d-116">Tämä kenttä ei ole käytettävissä, jos Käytä arvonlisäverotuksen sääntöjä -vaihtoehto on valittu Kirjanpitoparametrit-sivulla.</span><span class="sxs-lookup"><span data-stu-id="19c1d-116">This field is not available if the Apply sales tax taxation rules option is selected in the General ledger parameters page.</span></span> <span data-ttu-id="19c1d-117">Sen sijaan toimittajille maksettavat arvonlisäverot hyvitetään samalta tililtä kuin ostot.</span><span class="sxs-lookup"><span data-stu-id="19c1d-117">Instead, sales taxes that are paid to vendors are debited to the same account as the purchase.</span></span>   
7. <span data-ttu-id="19c1d-118">Valitse Käyttöverokulu-kentässä päätili niille vähennettäville käyttöveroille, joita veroviranomainen ei peri tai joita toimittajat eivät raportoi veroviranomaiselle osana EU:n käännettyä GOT/HS-veroa.</span><span class="sxs-lookup"><span data-stu-id="19c1d-118">In the Use tax expense field, select  the main account for posting deductible Use taxes that are not claimed or reported to the tax authority by vendors as part of EU reverse charge GST/HST.</span></span>
    * <span data-ttu-id="19c1d-119">Tapahtumassa käytettävän arvonlisäveroryhmän arvonlisäverokoodin Käyttövero-vaihtoehto on valittava.</span><span class="sxs-lookup"><span data-stu-id="19c1d-119">The Use tax option needs to be selected for the Sales tax code in the Sales tax group that is used in the transaction.</span></span>  <span data-ttu-id="19c1d-120">Tämä kenttä ei ole käytettävissä, jos Käytä arvonlisäverotuksen sääntöjä -vaihtoehto on valittu Kirjanpitoparametrit-sivulla.</span><span class="sxs-lookup"><span data-stu-id="19c1d-120">This field is not be available if the Apply sales tax taxation rules option is selected in the General ledger parameters page.</span></span>   
8. <span data-ttu-id="19c1d-121">Valitse Maksettava käyttövero -kentässä päätili veroviranomaiselle maksettavien saapuvien käyttöverojen kirjaamista varten.</span><span class="sxs-lookup"><span data-stu-id="19c1d-121">In the Use tax payable field, select the main account for posting incoming Use taxes that are payable to tax authorities.</span></span>
    * <span data-ttu-id="19c1d-122">Arvonlisäveroryhmän arvonlisäverokoodin Käyttövero-vaihtoehto on valittava, kun käyttövero kirjataan.</span><span class="sxs-lookup"><span data-stu-id="19c1d-122">The Use tax option needs to be selected in the Sales tax code in the Sales tax group to post Use tax.</span></span> <span data-ttu-id="19c1d-123">Jos kirjanpidon parametrien Käytä arvonlisäverotuksen sääntöjä -vaihtoehto on valittu, vastakirjaus kirjataan tapahtuman kulutilille.</span><span class="sxs-lookup"><span data-stu-id="19c1d-123">If Apply sales tax taxation rules option is selected in General ledger parameters, the offset is posted to the transaction’s expense account.</span></span>   
9. <span data-ttu-id="19c1d-124">Valitse Tilitystili-kentässä päätili, jolle Maksettava käyttövero- ja Saatava arvonlisävero -kentässä määritettyjen kirjanpitotilien nettosaldo kirjataan.</span><span class="sxs-lookup"><span data-stu-id="19c1d-124">In the Settlement account field, select the main account  that the net balance of the ledger accounts specified in the Use tax payable and Sales tax receivable fields will be posted.</span></span>
    * <span data-ttu-id="19c1d-125">Saldo luodaan, kun arvonlisäveron tilitys ja työn kirjaaminen suoritetaan.</span><span class="sxs-lookup"><span data-stu-id="19c1d-125">The balance will be created when the Sales tax settle and post job is ran.</span></span>  <span data-ttu-id="19c1d-126">Jos tilityskauden veroviranomainen on liitetty toimittajan tiliin, saldo kirjataan sen sijaan toimittajan tiliin.</span><span class="sxs-lookup"><span data-stu-id="19c1d-126">If the Tax authority for the settlement period is associated with a vendor account, the balance is posted to the vendor account instead.</span></span>   
10. <span data-ttu-id="19c1d-127">Valitse Toimittajan käteisalennus -kentässä päätili, jonne tähän kirjanpidon kirjausryhmän liitetyn arvonlisäverokoodin käteisalennukset kirjataan.</span><span class="sxs-lookup"><span data-stu-id="19c1d-127">In the Vendor cash discount field, select the main account to post cash discount for Sales tax codes associated with this Ledger posting group.</span></span>
    * <span data-ttu-id="19c1d-128">Tämä kenttä on valinnainen. Jos tiliä ei ole syötetty, käytetään käteisalennuksen koodien päätiliä.</span><span class="sxs-lookup"><span data-stu-id="19c1d-128">This is optional and if no account is entered,  the main account on Cash discount codes will be used.</span></span> <span data-ttu-id="19c1d-129">Kirjanpidon kirjausryhmässä saattaa kannattaa käyttää eri tilejä, jos käytössä on arvonlisäveroryhmien Mitätöi käteisalennuksen arvonlisävero -kohta.</span><span class="sxs-lookup"><span data-stu-id="19c1d-129">It can be useful to use different accounts per Ledger posting group if using the reverse sales tax on cash discount option on Sales tax groups.</span></span>  
11. <span data-ttu-id="19c1d-130">Valitse Asiakastapauksen alennus -kentässä päätili, jonne tähän kirjanpidon kirjausryhmän liitetyn arvonlisäverokoodin käteisalennukset kirjataan.</span><span class="sxs-lookup"><span data-stu-id="19c1d-130">In the Customer case discount field, select the main account to post cash discount for Sales tax codes associated with this Ledger posting group.</span></span>
    * <span data-ttu-id="19c1d-131">Tämä kenttä on valinnainen. Jos tiliä ei ole syötetty, käytetään käteisalennuksen koodien päätiliä.</span><span class="sxs-lookup"><span data-stu-id="19c1d-131">This is optional and if no account is entered, the main account on the Cash discount codes will be used.</span></span> <span data-ttu-id="19c1d-132">Kirjanpidon kirjausryhmässä saattaa kannattaa käyttää eri tilejä, jos käytössä on arvonlisäveroryhmien Mitätöi käteisalennuksen arvonlisävero -kohta.</span><span class="sxs-lookup"><span data-stu-id="19c1d-132">It can be useful to use different accounts per Ledger posting group if using the reverse sales tax on cash discount option on Sales tax groups.</span></span>  
12. <span data-ttu-id="19c1d-133">Valitse Tallenna.</span><span class="sxs-lookup"><span data-stu-id="19c1d-133">Click Save.</span></span>
