[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="6f6af-104">Tässä tehtäväopastuksessa käsitellään jaksotusmalleihin perustuvien kirjanpidon jaksotustapahtumien luontia</span><span class="sxs-lookup"><span data-stu-id="6f6af-104">This task guide steps through generating ledger accrual transactions that are based on accrual schemes</span></span>

1. <span data-ttu-id="6f6af-105">Siirry kohtaan Kirjanpito > Kirjauskansioviennit > Yleiset kirjauskansiot.</span><span class="sxs-lookup"><span data-stu-id="6f6af-105">Go to General ledger > Journal entries > General journals.</span></span>
2. <span data-ttu-id="6f6af-106">Etsi ja valitse kirjauskansio luettelosta tai luo uusi kirjauskansio.</span><span class="sxs-lookup"><span data-stu-id="6f6af-106">In the list, find and select the desired journal or create a new one.</span></span>
3. <span data-ttu-id="6f6af-107">Avaa Kirjauskansion eränumero -kentän linkki napsauttamalla.</span><span class="sxs-lookup"><span data-stu-id="6f6af-107">Click to follow the link in the Journal batch number field.</span></span>
4. <span data-ttu-id="6f6af-108">Merkitse valittu rivi luettelossa.</span><span class="sxs-lookup"><span data-stu-id="6f6af-108">In the list, mark the selected row.</span></span>
5. <span data-ttu-id="6f6af-109">Määritä Tili-kenttään haluamasi arvot.</span><span class="sxs-lookup"><span data-stu-id="6f6af-109">In the Account field, specify the desired values.</span></span>
    * <span data-ttu-id="6f6af-110">Tässä esimerkissä on määritetään vakuutuksen kulu.</span><span class="sxs-lookup"><span data-stu-id="6f6af-110">In this example, we are defining the expense for the insurance.</span></span> <span data-ttu-id="6f6af-111">Siitä tulee kausittainen kulusumma.</span><span class="sxs-lookup"><span data-stu-id="6f6af-111">It will be come periodic expense amount.</span></span>  
6. <span data-ttu-id="6f6af-112">Kirjoita arvo Kuvaus-kenttään.</span><span class="sxs-lookup"><span data-stu-id="6f6af-112">In the Description field, type a value.</span></span>
7. <span data-ttu-id="6f6af-113">Syötä Debet-kenttään numero.</span><span class="sxs-lookup"><span data-stu-id="6f6af-113">In the Debit field, enter a number.</span></span>
8. <span data-ttu-id="6f6af-114">Määritä Vastatili-kenttään haluamasi arvot.</span><span class="sxs-lookup"><span data-stu-id="6f6af-114">In the Offset account field, specify the desired values.</span></span>
9. <span data-ttu-id="6f6af-115">Valitse Toiminnot.</span><span class="sxs-lookup"><span data-stu-id="6f6af-115">Click Functions.</span></span>
10. <span data-ttu-id="6f6af-116">Valitse Kirjanpidon jaksotukset.</span><span class="sxs-lookup"><span data-stu-id="6f6af-116">Click Ledger accruals.</span></span>

14. <span data-ttu-id="6f6af-120">Syötä päivämäärä Aloituspäivä-kenttään.</span><span class="sxs-lookup"><span data-stu-id="6f6af-120">In the Start date field, enter a date.</span></span>
15. <span data-ttu-id="6f6af-121">Valitse Tapahtumat.</span><span class="sxs-lookup"><span data-stu-id="6f6af-121">Click Transactions.</span></span>
16. <span data-ttu-id="6f6af-122">Sulje sivu.</span><span class="sxs-lookup"><span data-stu-id="6f6af-122">Close the page.</span></span>
17. <span data-ttu-id="6f6af-123">Valitse OK.</span><span class="sxs-lookup"><span data-stu-id="6f6af-123">Click OK.</span></span>
18. <span data-ttu-id="6f6af-124">Valitse Kirjaa.</span><span class="sxs-lookup"><span data-stu-id="6f6af-124">Click Post.</span></span>
