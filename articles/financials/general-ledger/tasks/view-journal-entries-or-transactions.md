# <a name="view-journal-entries-or-transactions"></a><span data-ttu-id="7f510-103">Kirjauskansiovientien tai tapahtumien näyttäminen</span><span class="sxs-lookup"><span data-stu-id="7f510-103">View journal entries or transactions</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="7f510-104">Näiden ohjeiden avulla voit käyttää tositetapahtumien kyselyä kirjauskansiovientien tai tapahtumien hakua varten.</span><span class="sxs-lookup"><span data-stu-id="7f510-104">This procedure shows how to use the Voucher transactions inquiry to search for journal entries or transactions.</span></span>

1. <span data-ttu-id="7f510-105">Siirry kohtaan Kirjanpito > Kyselyt ja raportit > Tositetapahtumat.</span><span class="sxs-lookup"><span data-stu-id="7f510-105">Go to General ledger > Inquiries and reports > Voucher transactions.</span></span>
2. <span data-ttu-id="7f510-106">Voit tehdä kenttä, jolle haluat määrittää suodatusehdon.</span><span class="sxs-lookup"><span data-stu-id="7f510-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="7f510-107">Kirjoita valitun kentän suodatusehdot.</span><span class="sxs-lookup"><span data-stu-id="7f510-107">Enter your filter critieria for the selected field.</span></span>
    * <span data-ttu-id="7f510-108">Voit suodattaa yksittäisellä arvolla tai alueella.</span><span class="sxs-lookup"><span data-stu-id="7f510-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="7f510-109">Kun määrität alueen, varmista, että käytät oikeaa syntaksia.</span><span class="sxs-lookup"><span data-stu-id="7f510-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="7f510-110">Arvot on erotettava toisistaan kahdella pisteellä (..).</span><span class="sxs-lookup"><span data-stu-id="7f510-110">The values should be separated by a double period (..).</span></span>  
4. <span data-ttu-id="7f510-111">Valitse Liitokset-välilehti ja lisää suodatettavia taulukoita.</span><span class="sxs-lookup"><span data-stu-id="7f510-111">Click the Joins tab to add additional tables from which to filter.</span></span>
5. <span data-ttu-id="7f510-112">Valitse puusta "Tables\General journal entry".</span><span class="sxs-lookup"><span data-stu-id="7f510-112">In the tree, select 'Tables\General journal entry'.</span></span>
6. <span data-ttu-id="7f510-113">Valitse Lisää taulukon liitos.</span><span class="sxs-lookup"><span data-stu-id="7f510-113">Click Add table join.</span></span>
7. <span data-ttu-id="7f510-114">Valitse Peruuta, jos et halua lisätä taulukkoa.</span><span class="sxs-lookup"><span data-stu-id="7f510-114">Click Cancel if you decide not to add an additional table.</span></span>
8. <span data-ttu-id="7f510-115">Valitse Alue-välilehti.</span><span class="sxs-lookup"><span data-stu-id="7f510-115">Click the Range tab.</span></span>
9. <span data-ttu-id="7f510-116">Suorita kysely valitsemalla OK.</span><span class="sxs-lookup"><span data-stu-id="7f510-116">Click OK to run the query.</span></span>
10. <span data-ttu-id="7f510-117">Valitse Tapahtuman alkuperä.</span><span class="sxs-lookup"><span data-stu-id="7f510-117">Click Transaction origin.</span></span>
    * <span data-ttu-id="7f510-118">Ruudukon eri painikkeita voidaan käyttää valitun tositetietueen lisätietojen tarkasteluun.</span><span class="sxs-lookup"><span data-stu-id="7f510-118">Various buttons about the grid can be used to research additional information about the selected record of the voucher.</span></span> <span data-ttu-id="7f510-119">Jotkin painikkeet eivät ole käytettävissä tapahtumatyypistä ja tapahtuman ominaisuuksista riippuen.</span><span class="sxs-lookup"><span data-stu-id="7f510-119">Some buttons may not be available, depending on the type of transaction and characteristics of the transaction.</span></span>  
11. <span data-ttu-id="7f510-120">Sulje sivu.</span><span class="sxs-lookup"><span data-stu-id="7f510-120">Close the page.</span></span>
12. <span data-ttu-id="7f510-121">Napsauta Alkuperäinen tiedosto.</span><span class="sxs-lookup"><span data-stu-id="7f510-121">Click Original document.</span></span>
13. <span data-ttu-id="7f510-122">Sulje sivu.</span><span class="sxs-lookup"><span data-stu-id="7f510-122">Close the page.</span></span>
