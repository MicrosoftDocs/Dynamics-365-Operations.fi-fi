# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="b41e6-103">Käyttöomaisuuden poisto vapaatekstilaskua käyttäen</span><span class="sxs-lookup"><span data-stu-id="b41e6-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="b41e6-104">Näiden ohjeiden avulla voit poistaa käyttöomaisuuden käyttämällä vapaatekstilaskua.</span><span class="sxs-lookup"><span data-stu-id="b41e6-104">This procedure shows how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="b41e6-105">Siirry kohtaan Myyntireskontra > Laskut > Kaikki vapaatekstilaskut.</span><span class="sxs-lookup"><span data-stu-id="b41e6-105">Go to Accounts receivable > Invoices > All free text invoices.</span></span>
2. <span data-ttu-id="b41e6-106">Valitse Uusi.</span><span class="sxs-lookup"><span data-stu-id="b41e6-106">Click New.</span></span>
3. <span data-ttu-id="b41e6-107">Syötä tai valitse arvo Asiakastili-kentässä.</span><span class="sxs-lookup"><span data-stu-id="b41e6-107">In the Customer account field, enter or select a value.</span></span>
4. <span data-ttu-id="b41e6-108">Tarkista laskun oletuspäivämäärä ja muokkaa sitä tarvittaessa.</span><span class="sxs-lookup"><span data-stu-id="b41e6-108">Validate the default Invoice date and edit if applicable.</span></span>
5. <span data-ttu-id="b41e6-109">Tarkista jäljellä olevat oletusarvoiset otsikkokentät, kuten Valuutta, ja muokkaa sitä tarvittaessa.</span><span class="sxs-lookup"><span data-stu-id="b41e6-109">Validate remaining default header fields, such as Currency and edit if applicable.</span></span>
6. <span data-ttu-id="b41e6-110">Kirjoita kuvaus laskuriville.</span><span class="sxs-lookup"><span data-stu-id="b41e6-110">Enter a Description into the invoice line.</span></span>
7. <span data-ttu-id="b41e6-111">Kirjoita tai valitse päätili laskuriville.</span><span class="sxs-lookup"><span data-stu-id="b41e6-111">Enter or select the Main account for the invoice line.</span></span>
8. <span data-ttu-id="b41e6-112">Tarkista oletusarvoinen myyntiveroryhmä ja nimikkeen myyntiveroryhmä.</span><span class="sxs-lookup"><span data-stu-id="b41e6-112">Validate the default Sales tax group and Item sales tax group.</span></span>
9. <span data-ttu-id="b41e6-113">Kirjoita käyttöomaisuuden yksikköhinta tai myyntisumma.</span><span class="sxs-lookup"><span data-stu-id="b41e6-113">Enter the Unit price or hte Amount of the sale of the fixed asset.</span></span>
10. <span data-ttu-id="b41e6-114">Napsauta Rivin tiedot.</span><span class="sxs-lookup"><span data-stu-id="b41e6-114">Click Line details.</span></span>  
11. <span data-ttu-id="b41e6-115">Valitse myytävän käyttöomaisuuden numero.</span><span class="sxs-lookup"><span data-stu-id="b41e6-115">Select the Fixed asset number to be sold.</span></span>
12. <span data-ttu-id="b41e6-116">Valitse Kirjaa.</span><span class="sxs-lookup"><span data-stu-id="b41e6-116">Click Post.</span></span>
