[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="7ecad-104">Tässä tehtävän ohjauksessa käytetään esittely-yritystä USMF.</span><span class="sxs-lookup"><span data-stu-id="7ecad-104">This task guide uses the USMF demo company.</span></span>  <span data-ttu-id="7ecad-105">Tehtävässä luodaan uusi käyttöomaisuus Käyttöomaisuus-luettelosivun avulla.</span><span class="sxs-lookup"><span data-stu-id="7ecad-105">It will create a new fixed asset using the Fixed asset list page.</span></span>

1. <span data-ttu-id="7ecad-106">Siirry kohtaan Käyttöomaisuudet > Käyttöomaisuudet > Käyttöomaisuudet.</span><span class="sxs-lookup"><span data-stu-id="7ecad-106">Go to Fixed assets > Fixed assets > Fixed assets.</span></span>
2. <span data-ttu-id="7ecad-107">Valitse Uusi.</span><span class="sxs-lookup"><span data-stu-id="7ecad-107">Click New.</span></span>
3. <span data-ttu-id="7ecad-108">Syötä tai valitse arvo Käyttöomaisuusryhmä-kentässä.</span><span class="sxs-lookup"><span data-stu-id="7ecad-108">In the Fixed asset group field, enter or select a value.</span></span>
    * <span data-ttu-id="7ecad-109">Numerolle määritetään oletusarvo, jos käyttöomaisuusparametrien ja käyttöomaisuusryhmän Käyttöomaisuuserien automaattinen numerointi -toiminto on käytössä.</span><span class="sxs-lookup"><span data-stu-id="7ecad-109">The Number will default if you have enabled Autonumber fixed assets functionality in the Fixed assets parameters and the Fixed asset group.</span></span>  <span data-ttu-id="7ecad-110">Jos se ei ole käytössä, syötä käyttöomaisuuden yksilöivä numero.</span><span class="sxs-lookup"><span data-stu-id="7ecad-110">If not, you must enter a unique number to identify the fixed asset.</span></span>

5. <span data-ttu-id="7ecad-113">Valitse Kirjat.</span><span class="sxs-lookup"><span data-stu-id="7ecad-113">Click Books.</span></span>
