# <a name="create-a-fixed-asset"></a>Luo käyttöomaisuuserä

[!include [task guide banner](../../includes/task-guide-banner.md)]

Tässä tehtävän ohjauksessa käytetään esittely-yritystä USMF.  Tehtävässä luodaan uusi käyttöomaisuus Käyttöomaisuus-luettelosivun avulla.

1. Siirry kohtaan Käyttöomaisuudet > Käyttöomaisuudet > Käyttöomaisuudet.
2. Valitse Uusi.
3. Syötä tai valitse arvo Käyttöomaisuusryhmä-kentässä.
    * Numerolle määritetään oletusarvo, jos käyttöomaisuusparametrien ja käyttöomaisuusryhmän Käyttöomaisuuserien automaattinen numerointi -toiminto on käytössä.  Jos se ei ole käytössä, syötä käyttöomaisuuden yksilöivä numero.  
4. Kirjoita arvo Nimi-kenttään.
    * Syötä käyttöomaisuuserää koskevat lisätiedot, joita yritys tarvitsee.  
5. Valitse Kirjat.
6. Kirjoita päivämäärä Hankintapäivämäärä-kenttään.
7. Syötä Hankintahinta-kenttään numero.
    * Anna käyttöomaisuuserää koskevat lisätiedot, joita kirja tarvitsee.  
    * Anna poistokirjoja koskevat lisätiedot, joita jäljellä olevat kirjat tarvitsevat.  
8. Sulje sivu.
