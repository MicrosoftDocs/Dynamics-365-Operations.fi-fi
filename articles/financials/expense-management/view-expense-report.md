<span data-ttu-id="2ada3-104">Kun haluat katsoa tapahtumia kirjanpidossa tai hankinnat ja suorita sitä varten kyselyn, kysely voi palauttaa liian paljon tietoja.</span><span class="sxs-lookup"><span data-stu-id="2ada3-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="2ada3-105">Tämän vuoksi haluat tarkentaa hakua.</span><span class="sxs-lookup"><span data-stu-id="2ada3-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="2ada3-106">Saat tarvitsemasi tiedot porautumalla tapahtumaan katsomaan kuluraporttia, jossa tapahtuma esiintyi.</span><span class="sxs-lookup"><span data-stu-id="2ada3-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>
