# <a name="whats-new-or-changed-in-dynamics-365-talent-core-hr-january-17-2019"></a>Dynamics 365 Talent: Core HRin uudet tai muuttuneet ominaisuudet (17. tammikuuta 2019)

[!include [banner](includes/banner.md)]

**Koontiversio 8.1.2107**

Tässä ohjeaiheessa käsitellään Core HR -ohjelman uusia tai muuttuneita ominaisuuksia.

## <a name="minor-changes-and-bug-fixes-included-in-this-release"></a>Tähän versioon sisältyvä vähäiset muutokset ohjelmavirhekorjaukset

### <a name="new-position-assignment-start-datetime-default-has-been-corrected"></a>Uuden toimimäärityksen aloituspäivämäärän ja -ajan oletusasetus on korjattu
Työntekijöitä siirrettäessä järjestelmä palauttaa päivämäärän ja ajan valittuun päivämäärään ja aikaan, joka on päivän alussa käyttäjän aikavyöhykkeellä.

### <a name="all-employees-have-the-same-position-description-in-the-exited-worker-task-management-list"></a>Kaikilla työntekijöillä on sama toimikuvaus poistuneen työntekijän tehtävänhallintaluettelossa
**Tehtävänhallinta**-työtila on päivitetty näyttämään oikeat kuvaukset kaikille yrityksestä äskettäin poistuneille työntekijöille.

### <a name="action-requested-by-field-is-now-populated-on-workers-action-page"></a>Toiminnon pyytäjä -kenttä on nyt täytetty työntekijöiden toimintosivulla
Tämän muutoksen ansiosta **Toiminnon pyytäjä** -kenttä on nyt oletusarvoisesti muutosta pyytävä käyttäjä.

### <a name="ideas-portal-updated"></a>Päivitetty ideaportaali
Sovelluksen linkit, jolla voi lähettää Talent-idean, on päivitetty uuteen ideaportaaliin. Kaikki vanhassa portaalissa annetut ideat on siirretty, joten kaikki hyvät ideat ovat tallella ja muut voivat äänestää niitä.
