# <a name="whats-new-or-changed-in-dynamics-365-for-talent-core-hr"></a>Dynamics 365 for Talent Core HR:n uudet tai muuttuneet ominaisuudet 

[!include [banner](includes/banner.md)]

Tällä sivulla on tietoja siitä, missä on lisätietoja Microsoft Dynamics Talent HR Coren uusista versioista.

Talentin uusia versioita julkaistaan viikoittain. Nämä versiot on mainittu docs.microsoft.com-sivuston sisällysluettelossa ([docs.microsoft.com/dynamics365/unified-operations](../index.md)). Voit joko selata sisällysluetteloa tai etsiä tarvitsemasi uusimmat aiheet käyttämällä sisällysluettelon yläpuolella olevaa suodatuksen ohjausobjektia. Ohjeaiheiden otsikossa on sanat Dynamics 365 for Talent... uudet tai muuttuneet ominaisuudet.

Sisällysluettelo päivitetään edelleen viikoittain, joten käy tarkistamassa sieltä Talentiin uusien versioiden uudet ja muuttuneet ominaisuudet.
