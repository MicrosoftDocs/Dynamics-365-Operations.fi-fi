# <a name="whats-new-or-changed-in-dynamics-365-talent-december-17-2019"></a>Dynamics 365 Talent:n uudet tai muuttuneet ominaisuudet (17. joulukuuta 2019)

Tässä artikkelissa käsitellään Dynamics 365 Talentin uusia tai muuttuneita toimintoja.

## <a name="changes-in-attract"></a>Attractin muutokset

Tässä julkaisussa on vähäisiä Dynamics 365 Talent: Attractin ohjelmakorjauksia.

## <a name="changes-in-onboard"></a>Onboardin muutokset

Tässä julkaisussa on vähäisiä Dynamics 365 Talent: Onboard:n ohjelmakorjauksia.

## <a name="changes-in-core-hr"></a>Core HR:n muutokset

Tässä osassa kuvatut muutokset koskevat koontiversiota 8.1.2674. Joissakin otsikoissa suluissa olevat luvut viittaavat tukinumeroihin Microsoft Dynamics Lifecycle Services (LCS) -palveluissa.

### <a name="print-performance-reviews"></a>Kehityskeskustelujen tulostaminen

Tulostuksen suorituskykyarviot ovat nyt saatavilla kaikissa ympäristöissä. Voit ottaa tämän toiminnon käyttöön käyttämällä **ominaisuuksienhallinta**-työtilaa. Lisätietoja Toimintojen hallinnasta on kohdassa [Toimintojen hallinnan yleiskuvaus](https://docs.microsoft.com/dynamics365/fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview). Lisätietoja suorituskyvyn arvostelusta on kohdassa [Suorituskykyarviot](https://docs.microsoft.com/dynamics365/talent/performance-management-overview#performance-reviews).

### <a name="retired-position-with-direct-reports-floats-in-org-diagram-as-an-open-position-385436"></a>Keskeytynyt toimi, jossa on suorat raportit "kelluu" organisaatiokaaviossa avoimena toimena (385436)

Tämän viikon julkaisu korjaa sen, kuinka keskeytyneenä olevat toimet suorien raporttien kanssa näkyvät raportointihierarkiassa.
 
### <a name="system-unable-to-generate-calendar-days-381195"></a>Järjestelmä ei voi luoda kalenteripäiviä (381195) 

Tämän viikon muutokset korjaavat ongelman, jonka vuoksi kalenteripäiviä ei voitu luoda.

### <a name="current-balance-report-in-leave-pbi-doubles-employee-balance-385581"></a>Nykyinen saldoraportti loma-PBI:ssä kaksinkertaistaa työntekijän saldon (385581)

Tämä muutos korjaa nykyisen saldoraportin tässä skenaariossa.
