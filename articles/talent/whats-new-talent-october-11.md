# <a name="whats-new-or-changed-in-dynamics-365-for-talent-core-hr-october-15-2018"></a>Dynamics 365 for Talent Core HR:n uudet tai muuttuneet ominaisuudet (15. lokakuuta 2018)

[!include [banner](includes/banner.md)]

**Koontiversio 8.1.1056**

Tässä ohjeaiheessa käsitellään Core HR -ohjelman uusia tai muuttuneita ominaisuuksia.


## <a name="changes"></a>Muutokset
Muiden korjausten lisäksi tähän versioon on tehty seuraavat päivitykset:
- Viimeinen työpäivä määrittään nyt työhönoton yhteydessä tai työsuhteen päättymispäivää määritettäessä.
- Viittaukset yhdysvaltalaiseen lainsäädäntöön poistettu, kun kyse Yhdysvaltojen ulkopuolisista yrityksistä (ACA, ADA ja I9).
- Virheelliset päivämäärät (1.1.1900) piilotetaan nyt analyysisivuilla.

## <a name="known-issue"></a>Tunnettu ongelma

**Ongelma:** Lisättäessä uuden liitteen työntekijään, **Uusi**- ja **Muokkaa**-painikkeet näkyvät harmaina. **Ratkaisuehdotus:** Varmista ennen liitesivun avaamista, että olet sulkenut **Työntekijä**-sivulla olevat tietoruudut. Jos tietoruudut ovat suljettuja **Työntekijä**-sivua ladattaessa, liitepainikkeet otetaan käyttöön. (Tämä ongelma korjataan seuraavassa ympäristöpäivityksessä.)
