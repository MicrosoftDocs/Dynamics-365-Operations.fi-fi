# <a name="get-started-with-onboard"></a>Onboardin käytön aloittaminen

[!include [banner](includes/banner.md)]

Microsoft Dynamics 365 Talent: Onboardin avulla voit työntekijöitä suoriutumaan tehokkaasti ja sisäistämään uudet roolit. Tehokas perehdytysprosessi auttaa uusia työntekijöitä ymmärtämään uudet roolinsa. Se myös helpottaa arvokkaiden suhteiden muodostamista, mikä auttaa heitä menestymään.

Onboardin avulla voit luoda nopeasti perehdytysoppaan, joka toivottaa uudet työntekijät tervetulleiksi ja saa heidät innostumaan organisaatioon kuulumisesta.
