# <a name="create-an-onboarding-template"></a>Perehdytysmallin luominen

[!include [banner](includes/banner.md)]

Microsoft Dynamics 365 Talent: Onboardissa on erilaisia malleja, joiden avulla voi luoda perehdytysoppaan mahdollisimman nopeasti. Voit käyttää yhtä tai useita näistä malleista tai voit sitten luoda omia malleja. Onboardissa on näyteteksti, jota voit käyttää, kun luot omia malleja. Tämän vuoksi prosessi ei ole hankala, vaikka aloittaisit alusta.

## <a name="create-an-onboarding-template-from-an-existing-template"></a>Perehdytysmallin luominen aiemmin luodusta mallista

1. Valitse vasemmassa valikossa **Mallit**.
2. Valitse malli **Valikoiman suositut mallit**- tai **Omat mallit** -kohdassa. Voit tarkastella lisää malleja valitsemalla **Lisää vaihtoehtoja mallivalikoimassa**.
3. Noudata seuraavia ohjeita:

    - Jos malli on valikoimasta, valitse **Tallenna omana mallina**, anna mallille uusi nimi ja valitse **Tallenna**.
    - Jos malli on **omista malleista**, valitse **Tallenna omana mallina**, anna mallille uusi nimi ja valitse **Tallenna**.

    [![Perehdytysmallin luominen aiemmin luodusta mallista](./media/onboard-save-template.png)](./media/onboard-save-template.png)

## <a name="create-a-new-onboarding-template"></a>Uuden perehdytysmallin luominen

1. Valitse vasemmassa valikossa **Mallit**.
2. Valitse **Omat mallit** -kohdassa **Lisää** (plus-merkki \[**+**\]) -ruutu.

    [![Uuden mallin luominen](./media/onboard-create-new-template.png)](./media/onboard-create-new-template.png)

3. Anna **Luo oppaan malli** -valintaikkunassa mallin nimi ja valitse sitten **Tallenna**.
