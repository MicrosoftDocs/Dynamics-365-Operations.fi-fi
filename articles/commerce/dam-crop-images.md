<span data-ttu-id="97ee9-105">Commerce-sivuston luontiohjelman mediakirjaston avulla voit rajata kuvat ja optimoida ne eri moduulityyppejä ja näyttöjä varten.</span><span class="sxs-lookup"><span data-stu-id="97ee9-105">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="97ee9-106">Kuvan rajaaminen</span><span class="sxs-lookup"><span data-stu-id="97ee9-106">Crop an image</span></span>

<span data-ttu-id="97ee9-107">Voit rajata kuvan sivuston luontiohjelmaan seuraavasti.</span><span class="sxs-lookup"><span data-stu-id="97ee9-107">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="97ee9-108">Valitse Commerce-sivuston luontiohjelman vasemmanpuoleisessa ruudussa **Mediakirjasto**.</span><span class="sxs-lookup"><span data-stu-id="97ee9-108">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="97ee9-109">Valitse pääikkunassa kuva, jota haluat muokata.</span><span class="sxs-lookup"><span data-stu-id="97ee9-109">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="97ee9-110">Valitse komentopalkissa **Muokkaa**.</span><span class="sxs-lookup"><span data-stu-id="97ee9-110">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="97ee9-111">Valitse kuva ja syötä **Muokkaustila**.</span><span class="sxs-lookup"><span data-stu-id="97ee9-111">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="97ee9-112">Valitse **Muokkaustila**-kohdassa **Muokkaa näkymää moduulin mukaan**.</span><span class="sxs-lookup"><span data-stu-id="97ee9-112">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="97ee9-113">Valitse avattavassa **Moduuli**-valikossa moduulityyppi.</span><span class="sxs-lookup"><span data-stu-id="97ee9-113">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="97ee9-114">Valitse avattavassa **Näkymätyyppi**-valikossa näkymätyyppi.</span><span class="sxs-lookup"><span data-stu-id="97ee9-114">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="97ee9-115">Valitse avattavassa **Sijoittelu**-luettelossa kuvan sijoittelu.</span><span class="sxs-lookup"><span data-stu-id="97ee9-115">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="97ee9-116">Valitse avattavassa **Näyttö**-valikossa näytön koko.</span><span class="sxs-lookup"><span data-stu-id="97ee9-116">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="97ee9-117">Kuva peittä alueen, joka edustaa rajausaluetta.</span><span class="sxs-lookup"><span data-stu-id="97ee9-117">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="97ee9-118">Siirrä rajausaluetta ja muuta sen kokoa tarvittaessa.</span><span class="sxs-lookup"><span data-stu-id="97ee9-118">Move and resize the crop region as needed.</span></span> <span data-ttu-id="97ee9-119">Kuvasuhdetta ylläpidetään automaattisesti.</span><span class="sxs-lookup"><span data-stu-id="97ee9-119">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="97ee9-120">Kun olet valmis, valitse komentopalkissa **Tallenna** ja valitse sitten **Lopeta muokkaus**.</span><span class="sxs-lookup"><span data-stu-id="97ee9-120">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="97ee9-121">Kun mukautettu rajaus on tehty, kuvien muutokset tulevat voimaan lähes välittömästi.</span><span class="sxs-lookup"><span data-stu-id="97ee9-121">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
