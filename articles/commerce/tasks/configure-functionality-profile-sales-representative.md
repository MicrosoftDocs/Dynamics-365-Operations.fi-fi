# <a name="configure-the-functionality-profile-for-a-sales-representative"></a>Myyntiedustajan toimintoprofiilin määrittäminen

[!include [banner](../includes/banner.md)]

Tässä menettelyssä esitellään myyntiedustajia koskevien myymälän toimintoprofiilin asetusten määrittäminen. Esimerkkinä käytetään USRT-esittelytietoyritystä.

1. Valitse Retail ja Commerce > Kanavan asetukset > POS-asetukset > POS-profiilit > Toimintoprofiilit.
2. Valitse Muokkaa.
3. Laajenna Toiminnot-osa.
    * Voit käyttää toimintoprofiilin asetuksia määrittääksesi myyntipisteen lisäämään automaattisesti kassan oletusmyyntiryhmän, kysymään myyntiryhmiä ja vaatimaan myyntiryhmiä.  
4. Valitse vaihtoehto Kysy myyntiedustajaa ‑kentästä.
5. Valitse Vaadi myyntiedustaja -kentässä Kyllä.
