# <a name="compliance-overview"></a>Yhteensopivuuden yleiskatsaus


[!include [banner](includes/banner.md)]

Tämä ohjeaihe on yleiskatsaus Microsoft Dynamics 365 Commercen kanssa yhteensopivista yhteensopivuusohjeista, joiden avulla voit tehdä yrityksesi sivustoista yhteensopivia. Yhteensopivuus on tärkeä tekijä kaikille yrityksille, koska se auttaa heidän käyttäjäkuntansa muodostamaan yhteyden sivuston sisältöön. Säilyttämällä yhteensopivuuden voit myös auttaa suojaamaan yrityksesi kalliilta oikeustoimilta tai mojovilta sakoilta.

Vaatimustenmukaisuuden dokumentointi sisältää tarkistuksen alueista, jotka kannattaa ottaa huomioon, kun käytät Dynamics 365 Commercea, jotta voit täyttää yrityksesi yhteensopivuusvaatimukset.

Olet vastuussa yrityksesi yhteensopivuusvaatimusten tarkistamisesta sekä niiden sivustojen sisällön muokkauksesta ja rakentamisesta, jotka täyttävät näiden vaatimusten standardit. Seuraavissa ohjeaiheissa on esimerkkejä, jotka näyttävät, miten voit ottaa yhteensopivuuden huomioon, kun käytät Commerce-luontityökaluja.

Saat lisätietoja perusperiaatteista, joita Microsoft käyttää, käymällä [Microsoft Trust Centerissä](https://www.microsoft.com/trust-center). Tällä sivustolla voit myös saada lisätietoja yhteensopivuudesta.
