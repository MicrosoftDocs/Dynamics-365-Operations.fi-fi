# <a name="troubleshoot-store-commerce-performance-issues"></a>Store Commercen suorituskykyongelmien vianmääritys

[!include [banner](../includes/banner.md)]
[!include [banner](../includes/preview-banner.md)]

Tässä artikkelissa on tietoja Microsoft Dynamics 365 Commerce Store Commerce -sovelluksen suorituskykyongelmien vianmäärityksestä.

- Kun teet Store Commercen suorituskykyongelmien vianmäärityksen, vältä etäkirjautumista. Kirjaudu sen sijaan suoraan Store Commerce -laitteen kautta.
- Jos Store Commerce -sovelluksen käyttö on hidasta, tarkista prosessorin, muistin, verkon ja levyjen käyttö Task Managerin tai resurssinvalvonnan avulla. Varmista, ettei laiteresurssit kulu merkittävästi.
- Tarkista verkkoyhteyden tila ja vahvista, että HTTPS-pyynnöt/-vastaukset vievät korkeintaan muutaman sekunnin.
