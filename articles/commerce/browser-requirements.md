# <a name="browser-requirements-for-dynamics-365-commerce-sites"></a>Dynamics 365 Commerce -sivustojen selainvaatimukset

[!include [banner](includes/banner.md)]

Tässä ohjeaiheessa käsitellään Microsoft Dynamics 365 Commerce -sivustojen selainvaatimuksia.

Dynamics 365 Commerce -sivustot ovat parhaiten tuettuja seuraavissa Internet-selaimissa. Ohjelmistovalmistajien sivustot sisältävät tietoja kunkin selaimen uusimmasta versiosta.

- [Microsoft Edge](https://www.microsoft.com/edge) (vanhin tuettu versio: uusin versionumero vähennettynä 1:llä)
- [Mozilla Firefox](https://www.mozilla.org/firefox/new) (vanhin tuettu versio: uusin versionumero vähennettynä 1:llä)
- [Google Chrome](https://www.google.com/chrome) (vanhin tuettu versio: uusin versionumero vähennettynä 1:llä)
- [Apple Safari](https://support.apple.com/downloads/safari) (vanhin tuettu versio: uusin versionumero vähennettynä 1:llä)

> [!NOTE]
> Internet Exploreria ei tueta. Jos käytät Internet Exploreria, jotkin toiminnot eivät ehkä toimi, ja käyttäjillä saattaa olla huonompi käyttökokemus.
