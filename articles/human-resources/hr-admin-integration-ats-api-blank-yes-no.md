# <a name="blank-yes-no"></a>Tyhjä Kyllä Ei

Tässä aiheessa kuvataan Tyhjä Kyllä Ei -asetusjoukkoa Dynamics 365 Human Resourcesissa.

Fyysinen nimi: mshr_hrmblankyesno

Tämä valintalista mahdollistaa asetusjoukon kyllä/ei-ominaisuuksille, jotka sallivat myös tyhjät arvot.

| Arvo | Nimiö | kuvaus |
| --- | --- | --- |
| 200000000 | Tyhjä | Arvoa ei ole valittu. |
| 200000001 | Kyllä | Kyllä. |
| 200000002 | Nro | Nro |
