# <a name="screening-frequency-generate-from"></a>Seulonnan tiheys luo kohteesta

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Tässä aiheessa kuvataan Seulonnan tiheys luo kohteesta -asetusjoukkoa Dynamics 365 Human Resourcesissa.

Fyysinen nimi: mshr_hcmfrequencygeneratefrom

Tämä valintalista määrittää asetusjoukon arvoille, jotka määrittävät seuraavan pakollisen seulonnan laskennan aloituspäivän.

| Arvo | Nimiö | kuvaus |
| --- | --- | --- |
| 200000000 Ei valittu | Arvoa ei ole valittu. |
| 200000001 Valmistumispäivämäärä | Laskenta tehdään edellisestä seulonnan valmistusmispäivämäärästä. |
| 200000002 Määräpäivä | Laskenta tehdään edellisestä seulonnan määräpäivästä. |
