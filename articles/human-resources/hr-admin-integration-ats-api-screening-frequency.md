# <a name="screening-frequency"></a>Seulonnan tiheys


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Tässä artikkelissa kuvataan Seulonnan tiheys -asetusjoukkoa Dynamics 365 Human Resourcesissa.

Fyysinen nimi: mshr_hcmfrequencyunit

Tämä valintalista määrittää seulontatiheydelle asetusjoukon. 

| Arvo | Nimiö | kuvaus |
| --- | --- | --- |
| 200000000 Vain kerran | Näyttö on pakollinen vain kerran. |
| 200000001 Päivinä | Seulontatiheys lasketaan päivinä. |
| 200000002 Viikkoina | Seulontatiheys lasketaan viikkoina. |
| 200000003 Kuukausina | Seulontatiheys lasketaan kuukausina. |
| 200000004 Vuosina | Seulontatiheys lasketaan vuosina. |
