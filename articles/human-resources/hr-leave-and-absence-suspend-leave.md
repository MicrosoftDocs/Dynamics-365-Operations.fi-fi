## <a name="suspend-leave-and-absence-for-an-employee"></a><span data-ttu-id="61cf0-105">Työntekijän keskeytetyt lomat ja poissaolot</span><span class="sxs-lookup"><span data-stu-id="61cf0-105">Suspend leave and absence for an employee</span></span>

1. <span data-ttu-id="61cf0-106">Valitse työntekijän tietueesta **Loma**.</span><span class="sxs-lookup"><span data-stu-id="61cf0-106">On the employee's record, select **Leave**.</span></span>

4. <span data-ttu-id="61cf0-109">Valitse **Keskeytä loman kertymä**-valintaikkunassa **Lomatyyppi** sekä keskeytyksen **alkamis-** ja **päättymis**-päivämäärä.</span><span class="sxs-lookup"><span data-stu-id="61cf0-109">In the **Suspend leave accrual** dialog box, select the **Leave type** along with the **Start date** and **End date** for the suspension.</span></span>
