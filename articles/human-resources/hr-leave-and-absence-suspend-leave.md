# <a name="suspend-leave"></a><span data-ttu-id="bfaf6-103">Keskeytä loma</span><span class="sxs-lookup"><span data-stu-id="bfaf6-103">Suspend leave</span></span>

[!include [banner](includes/preview-feature.md)]

<span data-ttu-id="bfaf6-104">Voit keskeyttää työntekijän loman, jos haluat lopettaa jaksotusten käsittelyn valituille lomatyypeille.</span><span class="sxs-lookup"><span data-stu-id="bfaf6-104">You can suspend leave for an employee to stop leave accruals from being processed for selected leave types.</span></span> 

## <a name="suspend-leave-and-absence-for-an-employee"></a><span data-ttu-id="bfaf6-105">Työntekijän keskeytetyt lomat ja poissaolot</span><span class="sxs-lookup"><span data-stu-id="bfaf6-105">Suspend leave and absence for an employee</span></span>

1. <span data-ttu-id="bfaf6-106">Valitse työntekijän tietueesta **Loma**.</span><span class="sxs-lookup"><span data-stu-id="bfaf6-106">On the employee's record, select **Leave**.</span></span>

2. <span data-ttu-id="bfaf6-107">Valitse **Loman keskeyttäminen**.</span><span class="sxs-lookup"><span data-stu-id="bfaf6-107">Select **Suspend leave**.</span></span>

3. <span data-ttu-id="bfaf6-108">Valitse **Uusi**.</span><span class="sxs-lookup"><span data-stu-id="bfaf6-108">Select **New**.</span></span>

4. <span data-ttu-id="bfaf6-109">Valitse **Keskeytä loman kertymä**-valintaikkunassa **Lomatyyppi** sekä keskeytyksen **alkamis-** ja **päättymis**-päivämäärä.</span><span class="sxs-lookup"><span data-stu-id="bfaf6-109">In the **Suspend leave accrual** dialog box, select the **Leave type** along with the **Start date** and **End date** for the suspension.</span></span>

5. <span data-ttu-id="bfaf6-110">Voit halutessasi lisätä keskeytykseen **Kommentin**.</span><span class="sxs-lookup"><span data-stu-id="bfaf6-110">Optionally, you can add a **Comment** for the suspension.</span></span> 

<span data-ttu-id="bfaf6-111">Jos jaksotukset käsitellään, kun työntekijän loma keskeytetään, keskeytettyjä lomatyyppejä varten ei tehdä jaksotusta.</span><span class="sxs-lookup"><span data-stu-id="bfaf6-111">If accruals are processed while the employee's leave is suspended, no accrual will be made for the suspended leave types.</span></span>
