# <a name="create-employment-categories"></a>Työsuhdeluokkien luominen

Määritä työntekijöitä vastaavat työluokat Dynamics 365 Human Resourcesissa.

1. Valitse **Etujen hallinta** -työtilassa **Asetukset**-kohdasta **Työsuhdeluokat**.

2. Valitse **Uusi**.

3. Määritä **Työntekijän tyypin**, **Työluokan** ja **Kuvauksen** arvot.

4. Valitse **Tallenna**.
