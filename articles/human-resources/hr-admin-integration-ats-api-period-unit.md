# <a name="period-unit"></a>Kausiyksikkö

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Tässä aiheessa kuvataan Kauden yksikkö -asetusjoukkoa Dynamics 365 Human Resourcesissa.

Fyysinen nimi: mshr_periodunit

Tämä valintalista tarjoaa mittayksiköiden arvojen asetusjoukon aikajaksoja varten.

| Arvo | Nimiö | kuvaus |
| --- | --- | --- |
| 200000000 | Päivää | Päivä. |
| 200000001 | Kuukautta | Kuukausi. |
| 200000002 | Vuosia | Vuosi. |
