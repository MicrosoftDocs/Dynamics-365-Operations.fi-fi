# <a name="contact-type"></a>Yhteyshenkilötyyppi

Tässä aiheessa kuvataan Yhteyshenkilötyyppi-asetusjoukkoa Dynamics 365 Human Resourcesissa.

Fyysinen nimi: mshr_logisticselectronicaddressmethodtype

Tämä valintalista määrittää hakijan yhteystietotyypeille asetusjoukon. 

| Arvo | Nimiö | kuvaus |
| --- | --- | --- |
| 200000000 | None | Tyyppiä ei valita. |
| 200000001 | Puhelin | Puhelinnumero |
| 200000002 | Sähköpostiosoite | Sähköpostiosoite. |
| 200000003 | URL | Sivuston URL-osoite. |
| 200000004 | Teleksi | Teleksinumero. |
| 200000005 | Faksi | Faksinumero. |
| 200000006 | Facebook | Facebook-tili. Tunnistetaan käyttäjätunnuksen avulla. |
| 200000007 | Twitter | Twitter-tili. Tunnistetaan @käyttäjänimen avulla. |
| 200000008 | LinkedIn | LinkedIn-tili. Tunnistetaan käyttäjänimen avulla. |
