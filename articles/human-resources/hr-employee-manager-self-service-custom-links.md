# <a name="create-custom-links-in-manager-self-service"></a>Mukautettujen linkkien luominen esimiehen itsepalvelussa

[!include [banner](includes/preview-feature.md)]

Voit lisätä mukautettuja linkkejä esimiehen itsepalvelun **Oma ryhmä** -välilehdessä. Tämän toiminnon avulla voi siirtyä nopeasti tärkeisiin tietoihin Se muistuttaa mukautettujen linkkien lisäämistä työntekijän itsepalvelun **Omat tiedot** -välilehdessä

## <a name="enable-the-preview-feature"></a>Esiversio-ominaisuuden ottaminen käyttöön

Tätä ominaisuutta voi käyttää ottamalla **(Esiversio) Esimiehen itsepalvelun mukautetut linkit** -ominaisuuden käyttöön **Ominaisuuksien hallinta** -työtilassa. Lisätietoja esiversio-ominaisuuksien ottamisesta käyttöön on kohdassa [Ominaisuuksien hallinta](hr-admin-manage-features.md).

## <a name="set-up-custom-links"></a>Mukautettujen linkkien määrittäminen

1. Valitse **Human Resourcesin parametrit** -kohdassa **Esimiehen itsepalvelu**.

2. **Esimiesten linkkien määrittämine** -kohdassa voi lisätä tai poistaa linkin sekä muokata sitä. Linkkejä voi myös ryhmittää yhteen siten, että näkyvät ryhmänä esimiehen itsepalvelussa.

   ![Mukautettujen linkkien määrittäminen esimiehen itsepalvelussa](./media/hr-employee-manager-self-service-custom-links-setup.png)

3. Voit tarkastella linkkejä siirtymällä työntekijän itsepalvelussa **Oma ryhmä** -välilehteen.

   ![Mukautettujen linkkien näyttäminen esimiehen itsepalvelussa](./media/hr-employee-manager-self-service-custom-links-view.png)
