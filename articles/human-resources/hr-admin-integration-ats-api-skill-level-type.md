# <a name="skill-level-type"></a>Osaamisaluetason tyyppi

Tässä aiheessa kuvataan Osaamisaluetason tyyppi -asetusjoukkoa Dynamics 365 Human Resourcesissa.

Fyysinen nimi: mshr_hrmskillleveltype

Tämä valintalista tarjoaa asetusjoukon arvoille, jotka luokittelevat hakijan osaamisalueille määritetyn tason.

| Arvo | Nimiö | kuvaus |
| --- | --- | --- |
| 200000000 | Todellinen | Todellinen osaamisaluetaso. |
| 200000001 | Tavoite | Osaamisalueen tason tavoitearvo. |
