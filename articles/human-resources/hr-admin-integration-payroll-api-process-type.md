# <a name="process-type"></a>Käsittelyn tyyppi

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Tässä aiheessa kuvataan Ei Kyllä -asetusjoukkoa Dynamics 365 Human Resourcesissa.

## <a name="option-set-process-type"></a>Asetusjoukko: Käsittelyn tyyppi

Fyysinen nimi: mshr_hrmCompProcessType

Tämä luettelointi tarjoaa asetusjoukon käsittelyn tyypille Human Resources -virtuaaliyksiköissä.

| Arvo | Nimiö | kuvaus |
| --- | --- | --- |
| 200000000 | None | Ei mitään. |
| 200000001 | Vuosittainen | Vuosittainen. |
| 200000002 | Vuosipäivä | Vuosipäivä. |
| 200000003 | Ei suhdanteinen | Ei suhdanteinen. |
| 200000004 | Puoli vuotta | Puolivuosittain. |
| 200000005 | Neljännesvuosittain | Neljännesvuosittain. |
| 200000006 | Kuukausittain | Kuukausittain. |
