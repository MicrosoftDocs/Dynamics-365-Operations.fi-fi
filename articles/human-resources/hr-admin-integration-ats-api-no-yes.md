# <a name="no-yes"></a>Ei Kyllä

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Tässä aiheessa kuvataan Ei Kyllä -asetusjoukkoa Dynamics 365 Human Resourcesissa.

## <a name="option-set-no-yes"></a>Asetusjoukko: Ei Kyllä

Fyysinen nimi: mshr_noyes

Tämä valintalista mahdollistaa asetusjoukon tyypillisille Boolean-ominaisuuksille Human Resourcesin virtuaaliyksiköissä.

| Arvo | Nimiö | kuvaus |
| --- | --- | --- |
| 200000000 | Nro | Nro |
| 200000001 | Kyllä | Kyllä. |
