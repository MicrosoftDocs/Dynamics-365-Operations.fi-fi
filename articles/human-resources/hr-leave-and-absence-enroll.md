# <a name="assign-workers-to-a-leave-plan"></a>Työntekijöiden liittäminen lomasuunnitelmaan

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Kun olet luonut loma- ja poissaolosuunnitelman Dynamics 365 Human Resourcesissa, määritä se työntekijöille.

## <a name="assign-a-plan-to-a-group-of-employees"></a>Suunnitelman liittäminen työntekijäryhmään

1. Valitse **Loma ja poissaolo** -sivulta suunnitelma, jonka haluat määrittää.

2. Valitse **Määritä työntekijöille**.

3. Säädä seuraavat asetukset tarpeen mukaan:

   - **Ilmoittautuminen työsuhteen alkamispäivämääränä**
   - **Rekisteröintipäivä**
   - **Päättymispäivämäärä**
   - **Tasoperuste**

4. Valitse **Työntekijöiden rekisteröinti** -kohdassa kaikki suunnitelmaan liitettävät työt.

5. Valitse **Määritä suunnitelma** sivun alareunassa.

## <a name="assign-a-plan-to-an-employee"></a>Suunnitelman liittäminen työntekijään

1. Valitse työntekijän tietueesta **Loma**.

2. Valitse **Lomat ja poissaolot**.

3. Valitse **Uusi**.

4. Kirjoita rekisteröintipäivämäärä ja valitse lomasuunnitelma.
