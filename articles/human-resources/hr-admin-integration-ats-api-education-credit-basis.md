# <a name="education-credit-basis"></a>Koulutuksen yksikköperuste

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Tässä aiheessa kuvataan Koulutuksen yksikköperuste -asetusjoukkoa Dynamics 365 Human Resourcesissa.

Fyysinen nimi: mshr_hcmeducationcreditbasis

Tämä valintalista tarjoaa asetusjoukon hakijan koulutustietueen koulutuksen yksikköperusteen arvoille.

| Arvo | Nimiö | kuvaus |
| --- | --- | --- |
| 200000000 | Tyhjä | Arvoa ei ole valittu. |
| 200000001 | Puolivuotiskausi | Lukukausi. |
| 200000002 | Vuosineljännes | Vuosineljännes. |
| 200000003 | Kolme kuukautta | Kolmannesvuosi. |
| 200000004 | Kausi | Kausi. |
| 200000005 | Muu | Muu. |
