# <a name="create-worker-benefit-plans"></a>Työntekijöiden etuussuunnitelmien luominen

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Voit luoda työntekijöiden etuussuunnitelmia Microsoft Dynamics 365 Human Resourcesissa valitaksesi etuussuunnitelmia työntekijöille ja vahvistaaksesi etuussuunnitelmavalintoja. Yleensä työntekijät valitsevat etuussuunnitelmat itse työntekijän itsepalvelun avulla, minkä jälkeen etujen järjestelmänvalvoja vahvistaa valinnat. 

1. Valitse **Etujen hallinta** -työtilassa **Suunnitelmat**-kohdasta **Työntekijän etuussuunnitelmat**.

2. Valitse **Uusi**.

3. Määritä seuraavien kenttien arvot.

   | Kenttä | Kuvaus |
   | --- | --- |
   | Kausi | Määrittää etuusjakson Suunnitelmat-pikavälilehden suunnitelmien suodattamista varten. Suodata suunnitelmia, jotta voit helpommin valita kaikkien suunnitelmatietueiden osajoukon ja siten vahvistaa osajoukon. Valitse esimerkiksi luomasi jakso nimeltään 2015 vahvistaaksesi kaikki vuoden 2015 etuusrekisteröintivalinnat. |
   | Työntekijä | Määrittää työntekijän Suunnitelmat-pikavälilehden suunnitelmien suodattamista varten. Suodata suunnitelmia, jotta voit helpommin valita kaikkien suunnitelmatietueiden osajoukon ja siten vahvistaa osajoukon. |
   | Oikeushenkilö | Määrittää yrityksen Suunnitelmat-pikavälilehden suunnitelmien suodattamista varten. Suodata suunnitelmia, jotta voit helpommin valita kaikkien suunnitelmatietueiden osajoukon ja siten vahvistaa osajoukon. |
   | Kattavuusvaihtoehto | Määrittää kattavuusasetuksen Suunnitelmat-pikavälilehden suunnitelmien suodattamista varten. Suodata suunnitelmia, jotta voit helpommin valita kaikkien suunnitelmatietueiden osajoukon ja siten vahvistaa osajoukon. |
   | Oletus | Suodattaa etuussuunnitelmat sen perusteella, ovatko ne oletusarvoisia. Suodata suunnitelmia, jotta voit helpommin valita kaikkien suunnitelmaan tietueiden osajoukon sen vahvistamista varten. |
   | Tila | Suodattaa etuussuunnitelmia niiden tilan perusteella. Suodata suunnitelmia, jotta voit helpommin valita kaikkien suunnitelmaan tietueiden osajoukon sen vahvistamista varten. |
   | Vahvistus | Määrittää vahvistustilan Suunnitelmat-pikavälilehden suunnitelmien suodattamista varten. Suodata suunnitelmia, jotta voit helpommin valita kaikkien suunnitelmatietueiden osajoukon ja siten vahvistaa osajoukon. |
   | Peruutus | Määrittää peruutustilan Suunnitelmat-pikavälilehden suunnitelmien suodattamista varten. Suodata suunnitelmia, jotta voit helpommin valita kaikkien suunnitelmatietueiden osajoukon ja siten vahvistaa osajoukon. |
   | Voimassaolopäivämäärien suodatin | Suodattaa suunnitelmat sen perusteella, ovatko ne vanhentuneita, aktiivisia vai tulevaisuudessa aktivoitavia. Valitse valintaruudut, jotka vastaavat suunnitelmia, jotka haluat nähdä Suunnitelmat-pikavälilehdessä. |
   | Suunnitelmat | Suunnitelmat-pikavälilehti sisältää ne suunnitelmat, jotka täyttävät määrittämäsi suodatusperusteet. Henkilöstöhallinnon työntekijöiden määrittämät asiaan liittyvät määritysvalinnat ja työntekijöiden valitsemat rekisteröintivaihtoehdot sisällytetään jokaiselle riville. Kelvollinen-kenttä ilmaisee, onko suunnitelman valinnan kanssa vahvistusristiriitaa. |

4. Valitse **Tallenna**.
