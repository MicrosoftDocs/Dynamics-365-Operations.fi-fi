# <a name="configure-future-life-events"></a>Tulevien elämäntapahtumien määrittäminen

[!include [banner](includes/preview-feature.md)]

Voit ajoittaa tulevia elämäntapahtumia Dynamics 365 Human Resourcesissa.

1. Valitse **Etujen hallinta** -työtilassa **Asetukset**-kohdasta **Tulevat elämäntapahtumat**.

2. Valitse **Uusi**.

3. Määritä seuraavien kenttien arvot.

   | Kenttä | Kuvaus |
   | --- | --- |
   | Elinkaaritapahtuman päivämäärä | Järjestelmä käsittelee kaikki rekisteröintijakson aikana tähän päivämäärään mennessä toteutuneet tapahtumat. |
   | Elinkaaritapahtuma on kirjattu | Elämäntapahtuman lokiin kirjaamisen päivämäärä ja aika. |
   | Lokityyppi | Näyttää, onko toiminto jokin seuraavista:</br></br>- **Päivitys** – Muutos olemassa olevaan tietueeseen, joka seuraa elämäntapahtumia</br></br>- **Lisää** – Uuden elämäntapahtumatietueen luonti |
   | Elinkaaritapahtuman tyypin tunnus | Elämäntapahtumatyypin yksilöllinen tunniste. |
   | Elinkaaritapahtuman tyyppi | Työntekijän eturekisteröinnin päivittämisen katalysaattori. Lisätietoja on osassa Elämäntapahtumien käynnistimet. |
   | Tila | Onko elämäntapahtuma käsitelty. |
   | Linja | Tulevan elämäntapahtuman rivinumero. |

4. Valitse **Tallenna**.
