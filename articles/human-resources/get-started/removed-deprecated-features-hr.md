# <a name="removed-or-deprecated-features-in-dynamics-365-human-resources"></a>Poistetut tai vanhentuneet Dynamics 365 Human Resourcesin ominaisuudet

Tässä artikkelissa käsitellään ominaisuuksia, jotka on poistettu tai joiden poistoa suunnitellaan Microsoft Dynamics 365 Human Resourcesissa.

- *Poistettu* ominaisuus ei ole enää käytettävissä tuotteessa.
- *Vanhentunutta* ominaisuutta ei enää kehitetä aktiivisesti ja se voidaan poistaa tulevassa päivityksessä.

Tämän luettelon avulla voit ottaa huomioon nämä poistuneet ja vanhentuneet ominaisuudet omassa suunnittelussasi.

## <a name="features-removed-or-deprecated-in-human-resources"></a>Poistetut tai vanhentuneet Human Resourcesin ominaisuudet 

### <a name="new-lcs-project-creation-in-human-resources-stand-alone"></a>Uuden LCS-projektin luonti Human Resourcesissa (erillinen) 

[Human Resourcesin valmistelu](/hr-admin-setup-provision)

| &nbsp; | &nbsp; |
|------------|--------------------|
| **Poiston tai vanhentumisen syy** | Korvataan projektin luonnissa talous- ja toimintosovellusinfrastruktuurissa. Lisätietoja infrastruktuurien yhdistämisestä on kohdassa [Dynamics 365 Human Resource -infrastruktuurin yhdistämistä koskevat usein kysytyt kysymykset](/hr-infrastructure-merge-faq). |
| **Onko toinen ominaisuus korvannut?**   | Kyllä |
| **Tuotealueet, joihin vaikutetaan**         | Hakemus |
| **Käytön asetukset**              | Kaikki |
| **Tila**                         | Vanhentunut: Heinäkuusta 2022 alkaen uusien projektien luontia Human Resourcesissa (erillinen) ei enää tueta. |
