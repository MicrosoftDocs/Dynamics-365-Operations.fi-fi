# <a name="gender"></a>Sukupuoli


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Tässä artikkelissa kuvataan Sukupuoli-asetusjoukkoa Dynamics 365 Human Resourcesissa.

Fyysinen nimi: mshr_hcmpersongender

Tämä valintalista määrittää sukupuolien asetusjoukon hakijalle. Tämä on käytettävissä mshr_hcmpersongender-asetusjoukossa.

| Arvo | Nimiö | kuvaus |
| --- | --- | --- |
| 200000000 | None | Sukupuolta ei ole määritetty. |
| 200000001 | Mies | Mies. |
| 200000002 | Nainen | Nainen. |
| 200000003 | Yksilöimätön | Yksilöimätön sukupuolen valinta. |
