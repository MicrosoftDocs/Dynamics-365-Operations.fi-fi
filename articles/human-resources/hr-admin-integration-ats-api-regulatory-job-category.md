# <a name="regulatory-job-category"></a>Säännösten mukainen tehtäväluokka

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Tässä aiheessa kuvataan säännösten mukaisen tehtäväluokan asetusjoukkoa Dynamics 365 Human Resourcesissa.

Fyysinen nimi: mshr_hcmregulatoryjobcategory

Tämä valintalista määrittää asetusjoukon, joka määritetään säännösten mukaisten työluokkien asetusjoukon arvoille. Tämä tieto esitetään aiemmin luodussa mshr_hcmregulatoryjobcategory-asetusjoukossa.

| Arvo | Nimiö | kuvaus |
| --- | --- | --- |
| 200000000 | None | Ei mitään. |
| 200000001 | Johto | Johtajat / ylemmän tason päälliköt. |
| 200000002 | Päällikkö | Ensimmäisen ja keskitason päälliköt. |
| 200000003 | Ammatillinen | Ammattilaiset. |
| 200000004 | Teknikko | Teknikot. |
| 200000005 | Myyjä | Myyjät. |
| 200000006 | Hallinta | Hallinnon tukityöntekijät. |
| 200000007 | Ammattityöntekijä | Ammattityöntekijät. |
| 200000008 | Toimintojen työntekijä | Toimintojen työntekijät. |
| 200000009 | Työhenkilö | Työhenkilöt/avustajat. |
| 200000010 | Huoltotyöntekijä | Huoltotyöntekijät. |
