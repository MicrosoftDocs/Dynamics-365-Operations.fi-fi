# <a name="payroll---gender-option-set"></a>Palkanlaskenta - Sukupuoli-asetusjoukko


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Tässä aiheessa kuvataan Sukupuoli-asetusjoukkoa Dynamics 365 Human Resourcesissa.

## <a name="option-set-gender-option"></a>Asetusjoukko: Sukupuoli-asetus

Tämä luettelointi tarjoaa asetusjoukon työntekijän sukupuolille. Tämä on käytettävissä mshr_hcmpersongender-asetusjoukossa.

| Arvo | Nimiö | kuvaus |
| --- | --- | --- |
| 200000000 | None | Sukupuolta ei ole määritetty. |
| 200000001 | Mies | Mies. |
| 200000002 | Nainen | Nainen. |
| 200000003 | Yksilöimätön | Yksilöimätön sukupuolen valinta. |
