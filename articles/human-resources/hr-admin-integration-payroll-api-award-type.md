# <a name="award-type"></a>Palkkion tyyppi


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Tässä artikkelissa kuvataan Palkkion tyyppi -asetusjoukko Dynamics 365 Human Resourcesissa.

## <a name="option-set-award-type"></a>Asetusjoukko: Palkkion tyyppi

Fyysinen nimi: mshr_HrmCompVarAwardEmplType

Tämä luettelointi tarjoaa asetusjoukon muuttavan kompensaation palkkiotyypille Human Resources -virtuaaliyksiköille.

| Arvo | Nimiö | kuvaus |
| --- | --- | --- |
| 200000000 | Yleinen palkkio | Yleinen palkkio. |
| 200000001 | Kiinteä kokonaissummamaksu | Kiinteä kokonaissummamaksu. |
