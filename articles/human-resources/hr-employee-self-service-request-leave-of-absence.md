# <a name="request-a-leave-of-absence"></a>Virkavapaan pyytäminen

Voit lähettää poissaoloja koskevan pyynnön ja tarkastella lomapyyntöjen tilaa kohdassa Dynamics 365 Human Resources.

## <a name="request-a-leave-of-absence"></a>Virkavapaan pyytäminen

1. Valitse **työntekijän itsepalvelu** -työtilassa**Lisää** (…) **Vapaasaldot**-ruudusta.

2. Jos haluat lähettää virkavapaapyynnön, valitse **Pyydä virkavapaata**.

3. Kirjoita **lomatyypin**, **aloituspäivämäärän** ja **päättymispäivämäärän** tiedot.

4. Jos sinun on lähetettävä tukiasiakirjoja, valitse **Lataa** kohdasta **Liitteet**.

5. Kirjoita tarvittaessa tiedot **Kommentti**-kenttään.

6. Valitse **Lähetä**, kun olet valmis lähettämään pyyntösi. Muussa tapauksessa valitse **Tallenna vedos**.


## <a name="view-leave-of-absence-request-status"></a>Tarkastele virkavapaapyynnön tilaa

1. Valitse **työntekijän itsepalvelu** -työtilassa**Lisää** (…) **Vapaasaldot**-ruudusta.

2. Jos haluat tarkastella virkavapaapyyntöjäsi, valitse **Näytä virkavapaapyyntö**.
