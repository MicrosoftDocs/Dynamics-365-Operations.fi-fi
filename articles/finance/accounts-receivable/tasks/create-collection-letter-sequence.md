9. <span data-ttu-id="946de-122">Kirjoita **Kuvaus**-kenttään arvo.</span><span class="sxs-lookup"><span data-stu-id="946de-122">In the **Description** field, type a value.</span></span>

14. <span data-ttu-id="946de-128">Syötä **Erääntynyt vähimmäissaldo** -kenttään erääntynyt vähimmäissaldo, joka tarvitaan ennen maksukehotuksen lähettämistä.</span><span class="sxs-lookup"><span data-stu-id="946de-128">In the **Minimum overdue balance** field, enter the minimum overdue balance required before a collection letter is sent.</span></span>

23. <span data-ttu-id="946de-142">Syötä **Erääntynyt vähimmäissaldo** -kenttään numero.</span><span class="sxs-lookup"><span data-stu-id="946de-142">In the **Minimum overdue balance** field, enter a number.</span></span>
