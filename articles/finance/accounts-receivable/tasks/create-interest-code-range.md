# <a name="create-an-interest-code-with-a-range"></a>Luo korkoryhmäalue

[!include [banner](../../includes/banner.md)]
Korkoryhmät voidaan määrittää laskemaan eri korkosummat arvoalueiden perusteella. Tämä menettely näyttää, miten korkoryhmä lisätään ja miten siihen lisätään alue.

1. Valitse Luotonvalvonta > Korko > Korkokoodien määrittäminen.
2. Valitse Uusi.
3. Kirjoita Korkoryhmä-kenttään korkoryhmän nimi.
4. Kirjoita Kuvaus-kenttään korkoryhmän kuvaus.
5. Valitse Kuukausi.
6. Laajenna Ansiot-osa.
7. Laajenna Ansiot valuutan mukaan -osa.
8. Määritä Kirjanpitotili-kentän arvot.
9. Valitse Korko alueen mukaan -kentässä Kuukautta.
10. ValitseLisää.
11. Kirjoita tämän valuutan ja alueen kuvaus Kuvaus-kenttään.
12. Valitse Tallenna.
13. Valitse Alueet.
14. Valitse Uusi.
15. Anna ensin Arvosta-asetukseksi 0 ja sitten kuukausikorkoprosentti, jolla korko lasketaan. Esimerkissä se on 1,5.
16. Valitse Uusi.
17. Anna seuraavaksi Arvosta-asetukseksi 4, joka on ensimmäinen kuukausi, jolla uusi korkosumma lasketaan.
18. Anna kuukausikorkoprosentti, jolla lasketaan korko kuukaudesta 4 alkaen. Tässä esimerkissä se on 2,0.
19. Valitse Uusi.
20. Anna seuraavaksi Arvosta-asetukseksi 7, joka on seuraava kuukausi, jolla uusi korkosumma lasketaan.
21. Anna kuukausikorkoprosentti, jolla lasketaan korko kuukaudesta 7 alkaen. Tässä esimerkissä se on 2,5.
22. Lopeta asetusten määritys valitsemalla Sulje.
