# <a name="establish-customer-payment-fees"></a><span data-ttu-id="24c61-103">Määritä asiakkaan maksulisät</span><span class="sxs-lookup"><span data-stu-id="24c61-103">Establish customer payment fees</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="24c61-104">Luo asiakkaan toimitusmaksut.</span><span class="sxs-lookup"><span data-stu-id="24c61-104">Create payment fees for customer payments.</span></span>

<span data-ttu-id="24c61-105">Tässä tehtävässä käytetään esittely-yritystä USMF.</span><span class="sxs-lookup"><span data-stu-id="24c61-105">This task uses the USMF demo company.</span></span>

1. <span data-ttu-id="24c61-106">Siirry kohtaan **Siirtymisruutu** > **Moduulit > Myyntireskontra >Maksujen asetukset > Toimitusmaksu**.</span><span class="sxs-lookup"><span data-stu-id="24c61-106">In the **Navigation pane**, go to **Modules > Accounts receivable > Payments setup > Payment fee**.</span></span>
2. <span data-ttu-id="24c61-107">Valitse **Uusi**.</span><span class="sxs-lookup"><span data-stu-id="24c61-107">Click **New**.</span></span>
3. <span data-ttu-id="24c61-108">Syötä **Lisämaksun tunnus** -kenttään lisämaksun tunnus.</span><span class="sxs-lookup"><span data-stu-id="24c61-108">In the **Fee ID** field, enter a Fee ID.</span></span> <span data-ttu-id="24c61-109">Lisämaksun tunnus näkyy maksukirjauskansioissa, joten määritä käsiteltävää lisämaksua kuvaava tunnus.</span><span class="sxs-lookup"><span data-stu-id="24c61-109">The Fee ID displays on payment journals, so make it descriptive to understand what fee is being assessed.</span></span>  
4. <span data-ttu-id="24c61-110">Syötä **Nimi**-kenttään lisämaksun nimi.</span><span class="sxs-lookup"><span data-stu-id="24c61-110">In the **Name** field, enter a fee name.</span></span>
5. <span data-ttu-id="24c61-111">Syötä **Lisämaksun kuvaus** -kenttään lisämaksun kuvaus.</span><span class="sxs-lookup"><span data-stu-id="24c61-111">In the **Fee description** field, enter a description for the fee.</span></span>
6. <span data-ttu-id="24c61-112">Valitse **Veloitus**-kentässä, veloitetaanko lisämaksu asiakas- vai kirjanpitotililtä.</span><span class="sxs-lookup"><span data-stu-id="24c61-112">In the **Charge** field, select whether the fee will be charged to the Customer or a Ledger account.</span></span> <span data-ttu-id="24c61-113">Jos lisämaksu koskee asiakasta, valitse Asiakas.</span><span class="sxs-lookup"><span data-stu-id="24c61-113">If the customer is assessed the fee, select Customer.</span></span> <span data-ttu-id="24c61-114">Jos organisaatio käsittelee lisämaksun kuluna, valitse Kirjanpito.</span><span class="sxs-lookup"><span data-stu-id="24c61-114">If the fee will be assess to your organization as an expense, select Ledger.</span></span> <span data-ttu-id="24c61-115">Tässä tehtävässä valitaan Asiakas.</span><span class="sxs-lookup"><span data-stu-id="24c61-115">For this task, select Customer.</span></span>  
7. <span data-ttu-id="24c61-116">Valitse **Kirjauskansion tyyppi** -kentässä kirjauskansion tyyppi, joka voi käyttää tätä toimitusmaksua.</span><span class="sxs-lookup"><span data-stu-id="24c61-116">In the **Journal type** field, select the type of journal that can use this payment fee.</span></span> <span data-ttu-id="24c61-117">Jos näitä lisämaksuja käytetään asiakkaan maksuissa, kirjauskansion tyypiksi määritetään todennäköisesti Asiakkaan maksu.</span><span class="sxs-lookup"><span data-stu-id="24c61-117">If these fees are used for customer payments, the journal type will likely be Customer payment.</span></span>  
8. <span data-ttu-id="24c61-118">Valitse **Tallenna**.</span><span class="sxs-lookup"><span data-stu-id="24c61-118">Click **Save**.</span></span>
9. <span data-ttu-id="24c61-119">Valitse **Toimitusmaksun asetukset**.</span><span class="sxs-lookup"><span data-stu-id="24c61-119">Click **Payment fee setup**.</span></span> <span data-ttu-id="24c61-120">Toimitusmaksun asetuksissa määritetään toimitusmaksun käsittelyajankohdan ehdot.</span><span class="sxs-lookup"><span data-stu-id="24c61-120">The Payment fee setup is used to define the criteria for when the payment fee will be assessed.</span></span>  <span data-ttu-id="24c61-121">Voit esimerkiksi määrittää, että lisämaksu lasketaan, jos pankkitili on USMF OPER ja maksutapa on sekki.</span><span class="sxs-lookup"><span data-stu-id="24c61-121">For example, you can define that the fee will be calculated if the bank account is USMF OPER, and the method of payment is check.</span></span>  
10. <span data-ttu-id="24c61-122">Valitse **Ryhmittelyt**-kentässä joko Taulu, Ryhmä tai Kaikki, kun määrität tätä lisämaksua käyttävät pankkitilit.</span><span class="sxs-lookup"><span data-stu-id="24c61-122">In the **Groupings** field, select either Table, Group or All to define which bank accounts will be assessed this fee.</span></span> <span data-ttu-id="24c61-123">Jos valitset Kaikki, tämä lisämaksu koskee kaikkia pankkitilejä.</span><span class="sxs-lookup"><span data-stu-id="24c61-123">If you select All, all bank accounts could be assessed this fee.</span></span>  <span data-ttu-id="24c61-124">Jos valitset Taulu, tämä lisämaksu koskee vain valitsemaasi pankkitiliä.</span><span class="sxs-lookup"><span data-stu-id="24c61-124">If you select Table, only the bank account you select could be assessed this fee.</span></span> <span data-ttu-id="24c61-125">Jos valitset Ryhmä, tämä lisämaksu koskee vain valitsemasi pankkiryhmän pankkitilejä.</span><span class="sxs-lookup"><span data-stu-id="24c61-125">If you select Group, only the bank accounts in the selected bank group could be assessed this fee.</span></span>  
11. <span data-ttu-id="24c61-126">Valitse **Maksuliikenneyhteys**-kentässä joko pankkiryhmä tai pankkitili.</span><span class="sxs-lookup"><span data-stu-id="24c61-126">In the **Bank relation** field, select either a bank group or a bank account.</span></span> <span data-ttu-id="24c61-127">Jos valitset Taulu, pankkitilit näkyvät haussa.</span><span class="sxs-lookup"><span data-stu-id="24c61-127">If you selected Table, the lookup will display bank accounts.</span></span> <span data-ttu-id="24c61-128">Jos valitset Ryhmä, pankkiryhmät näkyvät haussa.</span><span class="sxs-lookup"><span data-stu-id="24c61-128">If you selected Group, the lookup will display bank groups.</span></span>  
12. <span data-ttu-id="24c61-129">Napsauta luettelossa valitulla rivillä olevaa linkkiä.</span><span class="sxs-lookup"><span data-stu-id="24c61-129">In the list, click the link in the selected row.</span></span>
13. <span data-ttu-id="24c61-130">Valitse **Maksutapa**-kentässä maksutapa, jota varten tämä maksu arvioidaan.</span><span class="sxs-lookup"><span data-stu-id="24c61-130">In the **Method of payment** field, select the Method of payment for which this fee will be assessed.</span></span> <span data-ttu-id="24c61-131">Voit määrittää lisämaksun asiakkaille esimerkiksi silloin, kun he lähettävät maksut sekkeinä sähköisten maksujen sijaan.</span><span class="sxs-lookup"><span data-stu-id="24c61-131">For example, you may assess a fee to your customers if they send payments as a check, rather than as an electronic payment.</span></span>  
14. <span data-ttu-id="24c61-132">Etsi haluamasi tietue luettelosta ja valitse se.</span><span class="sxs-lookup"><span data-stu-id="24c61-132">In the list, find and select the desired record.</span></span>
15. <span data-ttu-id="24c61-133">Jos on tarpeen, syötä **Maksun valuutta** -kenttään maksun valuutta.</span><span class="sxs-lookup"><span data-stu-id="24c61-133">If relevant, in the **Payment currency** field, enter a payment currency.</span></span> <span data-ttu-id="24c61-134">Maksun valuuttaa käytetään lisäehtona lisämaksun määrittämisessä.</span><span class="sxs-lookup"><span data-stu-id="24c61-134">The payment currency is used as an additional criteria for whether the fee will be assessed.</span></span>  <span data-ttu-id="24c61-135">Pankki voi veloittaa lisämaksun esimerkiksi Yhdysvaltojen dollareina saapuneista maksuista, koska yleensä maksutapahtumat suoritetaan euroina.</span><span class="sxs-lookup"><span data-stu-id="24c61-135">For example, your bank may charge an extra fee for payments received in USD currency, since they normally only transact in EUR currency.</span></span>  
16. <span data-ttu-id="24c61-136">Määritä, esitetäänkö lisämaksu prosenttina, summana vai välinä.</span><span class="sxs-lookup"><span data-stu-id="24c61-136">Select whether the fee will be a percent, amount or interval.</span></span>
17. <span data-ttu-id="24c61-137">Syötä **Prosenttiosuus/summa**-kenttään maksun prosenttiosuus tai summa.</span><span class="sxs-lookup"><span data-stu-id="24c61-137">In the **Percentage/Amount field**, enter either percentage or amount of the fee.</span></span> <span data-ttu-id="24c61-138">Jos Prosentti/summa-kentän arvo on Prosentti, tähän syötetään prosenttiluku.</span><span class="sxs-lookup"><span data-stu-id="24c61-138">If the Percentage/Amount field is Percent, then the value enter here will be a percentage.</span></span> <span data-ttu-id="24c61-139">Jos Prosentti/summa-kentän arvo on Summa, tähän syötetään summa.</span><span class="sxs-lookup"><span data-stu-id="24c61-139">If the Percentage/Amount field is Amount, then the value you enter here will be an amount.</span></span> <span data-ttu-id="24c61-140">Jos Prosentti/summa-kenttä on Väli, määritä tasot Väli-välilehden avulla.</span><span class="sxs-lookup"><span data-stu-id="24c61-140">If the Percentage/Amount field is Interval, use the Interval tab to define the tiers.</span></span>  
18. <span data-ttu-id="24c61-141">Valitse **Lisämaksun valuutta** -kentässä lisämaksun valuutta.</span><span class="sxs-lookup"><span data-stu-id="24c61-141">In the **Fee currency** field, select the currency of the fee.</span></span> <span data-ttu-id="24c61-142">Tämä on valuutta, jossa lisämaksu luodaan.</span><span class="sxs-lookup"><span data-stu-id="24c61-142">This is the currency in which the fee will be created.</span></span>  
19. <span data-ttu-id="24c61-143">Valitse **Tallenna**.</span><span class="sxs-lookup"><span data-stu-id="24c61-143">Click **Save**.</span></span>
