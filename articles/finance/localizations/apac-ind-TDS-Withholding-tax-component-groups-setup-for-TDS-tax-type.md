# <a name="set-up-withholding-tax-component-groups-for-the-tds-tax-type"></a><span data-ttu-id="9678b-103">Ennakonpidätyksen osaryhmien määrittäminen TDS-verotyypille</span><span class="sxs-lookup"><span data-stu-id="9678b-103">Set up withholding tax component groups for the TDS tax type</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="9678b-104">Tässä aiheessa kuvataan, miten ennakonpidätyksen komponenttiryhmät, kuten **Vero** ja **Alihankkija**, määritetään Vero vähennettynä lähteissä (TDS) -verotyypille.</span><span class="sxs-lookup"><span data-stu-id="9678b-104">This topic explains how to set up withholding tax component groups, such as **Rent** and **Contractor**, for the Tax Deducted at Source (TDS) tax type.</span></span>

1. <span data-ttu-id="9678b-105">Siirry kohtaan **Vero \> Asetukset \> Ennakonpidätys \> Ennakonpidätyksen komponenttiryhmät**.</span><span class="sxs-lookup"><span data-stu-id="9678b-105">Go to **Tax \> Setup \> Withholding tax \> Withholding tax component groups**.</span></span>

    <span data-ttu-id="9678b-106">[![Ennakonpidätyksen komponenttiryhmät -sivu](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span><span class="sxs-lookup"><span data-stu-id="9678b-106">[![Withholding tax component groups page](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span></span>

2. <span data-ttu-id="9678b-107">Valitse **Verotyyppi**-kentässä **TDS**, haluat määrittää ennakonpidätyksen komponenttiryhmät TDS-verotyypille.</span><span class="sxs-lookup"><span data-stu-id="9678b-107">In the **Tax type** field, select **TDS** to set up withholding tax component groups for the TDS tax type.</span></span>
3. <span data-ttu-id="9678b-108">Valitse toimintoruudussa **Uusi** ja luo rivi.</span><span class="sxs-lookup"><span data-stu-id="9678b-108">On the Action Pane, select **New** to create a line.</span></span>
4. <span data-ttu-id="9678b-109">Kirjoita **Ennakonpidätyksen komponenttiryhmä** -kenttään TDS-komponenttiryhmän nimi.</span><span class="sxs-lookup"><span data-stu-id="9678b-109">In the **Withholding tax component group** field, enter a name for the TDS component group.</span></span>
5. <span data-ttu-id="9678b-110">Syötä **Kuvaus**-kenttään kuvaus.</span><span class="sxs-lookup"><span data-stu-id="9678b-110">In the **Description** field, enter a description.</span></span>
6. <span data-ttu-id="9678b-111">Valitse **Yleinen**-pikavälilehden **Tila**-kentästä TDS-komponenttiryhmän asuinaluetila.</span><span class="sxs-lookup"><span data-stu-id="9678b-111">On the **General** FastTab, in the **Status** field, select the residential status of the TDS component group.</span></span> <span data-ttu-id="9678b-112">Vaihtoehdot ovat **Asukas** ja **Ulkomailla asuva**.</span><span class="sxs-lookup"><span data-stu-id="9678b-112">The options are **Resident** and **Non-resident**.</span></span>
7. <span data-ttu-id="9678b-113">Kirjoita **Osakoodi**-kenttään osan koodi, jota käytetään TDS-komponenttiryhmässä.</span><span class="sxs-lookup"><span data-stu-id="9678b-113">In the **Section code** field, enter the section code that is applied to the TDS component group.</span></span>
8. <span data-ttu-id="9678b-114">Sulje sivu.</span><span class="sxs-lookup"><span data-stu-id="9678b-114">Close the page.</span></span>
