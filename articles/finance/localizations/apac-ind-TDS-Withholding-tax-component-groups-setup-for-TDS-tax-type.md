# <a name="set-up-withholding-tax-component-groups-for-the-tds-tax-type"></a>Ennakonpidätyksen osaryhmien määrittäminen TDS-verotyypille

[!include [banner](../includes/banner.md)]

Tässä aiheessa kuvataan, miten ennakonpidätyksen komponenttiryhmät, kuten **Vero** ja **Alihankkija**, määritetään Vero vähennettynä lähteissä (TDS) -verotyypille.

1. Siirry kohtaan **Vero \> Asetukset \> Ennakonpidätys \> Ennakonpidätyksen komponenttiryhmät**.

    [![Ennakonpidätyksen komponenttiryhmät -sivu.](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)

2. Valitse **Verotyyppi**-kentässä **TDS**, haluat määrittää ennakonpidätyksen komponenttiryhmät TDS-verotyypille.
3. Valitse toimintoruudussa **Uusi** ja luo rivi.
4. Kirjoita **Ennakonpidätyksen komponenttiryhmä** -kenttään TDS-komponenttiryhmän nimi.
5. Syötä **Kuvaus**-kenttään kuvaus.
6. Valitse **Yleinen**-pikavälilehden **Tila**-kentästä TDS-komponenttiryhmän asuinaluetila. Vaihtoehdot ovat **Asukas** ja **Ulkomailla asuva**.
7. Kirjoita **Osakoodi**-kenttään osan koodi, jota käytetään TDS-komponenttiryhmässä.
8. Sulje sivu.
