# <a name="run-the-periodic-tds-settlement-process"></a><span data-ttu-id="127fe-103">Suorita kausittainen TDS-tilitysprosessi</span><span class="sxs-lookup"><span data-stu-id="127fe-103">Run the periodic TDS settlement process</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="127fe-104">Tässä aiheessa kerrotaan, miten kausittainen Vero vähennettynä lähteissä (TDS) tilitetään.</span><span class="sxs-lookup"><span data-stu-id="127fe-104">This topic explains how to settle periodic Tax Deducted at Source (TDS).</span></span>

1. <span data-ttu-id="127fe-105">Siirry kohtaan **Vero \> Ilmoitukset \> Ennakonpidätys \> Ennakonpidätysmaksu**.</span><span class="sxs-lookup"><span data-stu-id="127fe-105">Go to **Tax \> Declarations \> Withholding tax \> Withholding tax payment**.</span></span>

    <span data-ttu-id="127fe-106">[![Ennakonpidätysmaksu-valintaikkuna](./media/apac-ind-TDS-47.png)](./media/apac-ind-TDS-47.png)</span><span class="sxs-lookup"><span data-stu-id="127fe-106">[![Withholding tax payment dialog box](./media/apac-ind-TDS-47.png)](./media/apac-ind-TDS-47.png)</span></span>

2. <span data-ttu-id="127fe-107">Valitse **Ennakonpidätysmaksu**-valintaikkunan **Verotyyppi**-kentässä **TDS**.</span><span class="sxs-lookup"><span data-stu-id="127fe-107">In the **Withholding tax payment** dialog box, in the **Tax type** field, select **TDS**.</span></span>
3. <span data-ttu-id="127fe-108">Valitse **Verotilin numero (TAN)** -kentässä Verotilin numero (TAN), jolle tilitysprosessi suoritetaan.</span><span class="sxs-lookup"><span data-stu-id="127fe-108">In the **Tax Account Number (TAN)** field, select the Tax Account Number (TAN) to run the settlement process for.</span></span>
4. <span data-ttu-id="127fe-109">Valitse **Ennakonpidätyksen komponenttiryhmä** -kentässä TDS-komponenttiryhmä, jolle tilitysprosessi suoritetaan.</span><span class="sxs-lookup"><span data-stu-id="127fe-109">In the **Withholding tax component group** field, select the TDS component group to run the settlement process for.</span></span>
5. <span data-ttu-id="127fe-110">Valitse **Tilityskausi**-kentässä TDS-tilityskausi, jolle tilitysprosessi suoritetaan.</span><span class="sxs-lookup"><span data-stu-id="127fe-110">In the **Settlement period** field, select the TDS settlement period to run the settlement process for.</span></span>

    > [!NOTE]
    > <span data-ttu-id="127fe-111">TDS-tilitysprosessi suoritetaan kaikille kausille, jotka on määritetty TDS-tilityskaudella **Kaudet**-välilehdessä **Ennakonpidätyksen tilityskaudet** -sivulla (**Vero \> Välilliset verot \> Ennakonpidätys \> Ennakonpidätyksen tilityskaudet**).</span><span class="sxs-lookup"><span data-stu-id="127fe-111">The TDS settlement process is run for all periods that are set up for the TDS settlement period on the **Periods** tab of the **Withholding tax settlement periods** page (**Tax \> Indirect taxes \> Withholding tax \> Withholding tax settlement periods**).</span></span>

6. <span data-ttu-id="127fe-112">Valitse **Päivämäärästä**-kentästä alkamispäivämäärä, josta TDS-tilitysprosessi suoritetaan.</span><span class="sxs-lookup"><span data-stu-id="127fe-112">In the **From date** field, select the start date to run the TDS settlement process from.</span></span>

    <span data-ttu-id="127fe-113">Tilityskauden tietyllä ajanjaksolla kaudelle määritettyä aloituspäivää käytetään Päivämäärästä-arvona.</span><span class="sxs-lookup"><span data-stu-id="127fe-113">For a specific period within the settlement period, the start date that is defined for the period is taken as the "from" date.</span></span> <span data-ttu-id="127fe-114">Esimerkiksi TDS-tilityskaudella on kaksi kautta: 1.4.–30.4.2009 ja 1.5.–31.5.2009.</span><span class="sxs-lookup"><span data-stu-id="127fe-114">For example, the TDS settlement period has two periods: April 1 through April 30, 2009, and May 1 through May 31, 2009.</span></span> <span data-ttu-id="127fe-115">Jos valitset alkupäivämääräksi **04/06/2009** (6. huhtikuuta 2009) **Päivämäärästä**-kenttään, tilitysprosessi suoritetaan silti 1.4.2009 alkaen.</span><span class="sxs-lookup"><span data-stu-id="127fe-115">If you select **04/06/2009** (April 6, 2009) as the start date in the **From date** field, the settlement process will still run from April 1, 2009.</span></span>

    <span data-ttu-id="127fe-116">Jos syötät myöhemmän kauden **Päivämäärästä**-kenttään mutta et tilitä tilityskauden edellistä kautta, tilitys ei tule voimaan edellisten kausien osalta.</span><span class="sxs-lookup"><span data-stu-id="127fe-116">If you enter a later period in the **From date** field, but without settling a previous period within the settlement period, the settlement won't occur for any previous periods.</span></span> <span data-ttu-id="127fe-117">Esimerkiksi TDS-tilityskaudella on kolme kautta: 1.4.–30.4.2009, 1.5.–31.5.2009 ja 1.6.–30.6.2009.</span><span class="sxs-lookup"><span data-stu-id="127fe-117">For example, the TDS settlement period has three periods: April 1 through April 30, 2009, May 1 through May 31, 2009, and June 1 through June 30, 2009.</span></span> <span data-ttu-id="127fe-118">Jos valitset **05/01/2009** (1. toukokuuta 2009) alkamispäiväksi **Päivämäärästä**-kenttään, tilitysprosessi suoritetaan vain 1.5.–31.5.2009.</span><span class="sxs-lookup"><span data-stu-id="127fe-118">If you select **05/01/2009** (May 1, 2009) as the start date in the **From date** field, the settlement process will run only from May 1 through May 31, 2009.</span></span> <span data-ttu-id="127fe-119">Tilitystä ei tehdä 1.4.–30.4.2009 väliselle ajalle.</span><span class="sxs-lookup"><span data-stu-id="127fe-119">The settlement won't occur for April 1 through April 30, 2009.</span></span>

7. <span data-ttu-id="127fe-120">Valitse **Tapahtuman päivämäärä** -kentässä päivämäärä, jolle TDS-tilitystapahtuma kirjataan.</span><span class="sxs-lookup"><span data-stu-id="127fe-120">In the **Transaction date** field, select the date to post the TDS settlement transaction.</span></span>
8. <span data-ttu-id="127fe-121">Valitse **Ennakonpidätysmaksu versio** -kentässä valitse TDS-tilityksen versio:</span><span class="sxs-lookup"><span data-stu-id="127fe-121">In the **Withholding tax payment version** field, select the TDS settlement version:</span></span>

     - <span data-ttu-id="127fe-122">**Alkuperäinen** – Tämän vaihtoehdon avulla voit suorittaa TDS-tilitysprosessin ensimmäistä kertaa.</span><span class="sxs-lookup"><span data-stu-id="127fe-122">**Original** – Use this option to run the TDS settlement process for the first time.</span></span> <span data-ttu-id="127fe-123">Alkuperäistä maksuversiota käytetään vain kerran TDS-tilitysprosessin suorittamiseen yhdistelmissä, johon sisältyy TAN, ennakonpidätyksen komponenttiryhmä ja ennakonpidätyksen tilityskausi.</span><span class="sxs-lookup"><span data-stu-id="127fe-123">The original payment version is used only one time to run the TDS settlement process for a combination of a TAN, a withholding tax component group, and a withholding tax settlement period.</span></span>
    - <span data-ttu-id="127fe-124">**Uusimmat versiot** – Käytä tätä vaihtoehtoa, jos TDS-tilitysprosessi on jo suoritettu määritetylle jaksolle.</span><span class="sxs-lookup"><span data-stu-id="127fe-124">**Latest versions** – Use this option if the TDS settlement process has already been run for the specified period.</span></span> <span data-ttu-id="127fe-125">Sisällytä takautuvat tapahtumat, jotka on kirjattu sen jälkeen, kun tilitysprosessi suoritettiin kaudelle aiemmin.</span><span class="sxs-lookup"><span data-stu-id="127fe-125">Include back-dated transactions that were posted after the settlement process was previously run for the period.</span></span> <span data-ttu-id="127fe-126">Tämän vaihtoehdon avulla voit suorittaa tilitysprosessin kuinka monta kertaa tahansa.</span><span class="sxs-lookup"><span data-stu-id="127fe-126">You can use this option to run the settlement process any number of times.</span></span>

9. <span data-ttu-id="127fe-127">Suorita TDS-selvitysprosessi ja kirjaa summat kirjanpitotileille valitsemalla **Päivitä**-valintaruutu.</span><span class="sxs-lookup"><span data-stu-id="127fe-127">Select the **Update** check box to run the TDS settlement process and post the amounts to the ledger accounts.</span></span> <span data-ttu-id="127fe-128">Jos tämän valintaruudun valinta on tyhjä, tilitysprosessia ei suoriteta eikä taloudellisia merkintöjä luoda.</span><span class="sxs-lookup"><span data-stu-id="127fe-128">If this check box is cleared, the settlement process won't be run, and the financial entries won't be generated.</span></span>
10. <span data-ttu-id="127fe-129">Suorita **OK** suorittaaksesi TDS-selvitysprosessin ja luodaksesi ennakonpidätyksen maksuraportin.</span><span class="sxs-lookup"><span data-stu-id="127fe-129">Select **OK** to run the TDS settlement process and generate the withholding tax payment report.</span></span> <span data-ttu-id="127fe-130">Tilitysprosessiin sisältyvien TDS-tapahtumien tila näkyy **Tilitetty**-muodossa **Tilitys**-sivulla (siirry kohtaan **Ostoreskontra \> Maksut \> Toimittajan maksukirjauskansio**, valitse **Rivit**, valitse **Toiminnot** ja valitse sitten **Tilitys**).</span><span class="sxs-lookup"><span data-stu-id="127fe-130">The status of TDS transactions that are included in the settlement process is shown as **Settled** on the **Settlement** page (go to **Accounts payable \> Payments \> Vendor payment journal**, select **Lines**, select **Functions**, and then select **Settlement**).</span></span>

### <a name="important-points"></a><span data-ttu-id="127fe-131">Tärkeät kohdat</span><span class="sxs-lookup"><span data-stu-id="127fe-131">Important points</span></span>

- <span data-ttu-id="127fe-132">Jos ennakonpidätyskomponenttiryhmää ei ole valittu tilitysprosessin aikana, tilitys tapahtuu kaikille valitun TAN- ja tilityskauden ennakonpidätyksen komponenttiryhmille.</span><span class="sxs-lookup"><span data-stu-id="127fe-132">If a withholding tax component group isn't selected during the settlement process, the settlement occurs for all the withholding tax component groups for the selected TAN and settlement period.</span></span> <span data-ttu-id="127fe-133">Kullekin ennakonpidätyksen komponenttiryhmälle luodaan erillinen rivi **Avoimen tapahtuman muokkaus** -sivulla.</span><span class="sxs-lookup"><span data-stu-id="127fe-133">A separate line is created for each withholding tax component group on the **Open transaction editing** page.</span></span>
- <span data-ttu-id="127fe-134">Tilitysprosessi perustuu tilityskauden arvioitavan kohdeluokan luonteeseen.</span><span class="sxs-lookup"><span data-stu-id="127fe-134">The settlement process is based on the nature of assessee category for a settlement period.</span></span> <span data-ttu-id="127fe-135">Tapahtumat, joissa arvioitavan luokan tyyppi on **Yritys**, tilitetään ja näytetään yhtenä rivinä **Avoimen tapahtuman muokkaus** -sivulla.</span><span class="sxs-lookup"><span data-stu-id="127fe-135">Transactions where the nature of assessee category is **Company** are settled and are shown as one line on the **Open transaction editing** page.</span></span> <span data-ttu-id="127fe-136">Tapahtumat, joissa arvioitavan kohteen luonne on muu kuin **Yritys**, tilitetään ja näytetään yhtenä rivinä **Avoimen tapahtuman muokkaus** -sivulla.</span><span class="sxs-lookup"><span data-stu-id="127fe-136">Transactions where the nature of assessee is something other than **Company** are settled and are shown as one line on the **Open transaction editing** page.</span></span>
- <span data-ttu-id="127fe-137">**Avoimen tapahtuman muokkaus** -sivun tilitettyjen TDS-tapahtumarivien eräpäivä perustuu **Toimittajat**-sivulla määritettyihin maksuehtoihin TDS-viranomaistoimittajalle.</span><span class="sxs-lookup"><span data-stu-id="127fe-137">The due date for the settled TDS transaction lines on the **Open transaction editing** page is based on the terms of payment that are defined for the TDS authority vendor on the **Vendors** page.</span></span> <span data-ttu-id="127fe-138">Jos TDS-viranomaistoimittajalle ei ole määritetty maksuehtoja, eräpäivänä näkyy tilityskauden viimeinen päivä.</span><span class="sxs-lookup"><span data-stu-id="127fe-138">If the terms of payments aren't defined for the TDS authority vendor, the last day of the settlement period is shown as the due date.</span></span>
