# <a name="threshold-limit-and-exception-threshold-limit"></a><span data-ttu-id="d142a-103">Raja-arvo ja poikkeusraja-arvo</span><span class="sxs-lookup"><span data-stu-id="d142a-103">Threshold limit and exception threshold limit</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="d142a-104">Tässä aiheessa kuvataan Vero vähennettynä lähteissä (TDS) raja- ja poikkeusraja-arvot.</span><span class="sxs-lookup"><span data-stu-id="d142a-104">This topic describes the threshold and exception limits for Tax Deducted at Source (TDS).</span></span> <span data-ttu-id="d142a-105">Laskujen ja maksujen TDS-tiedot lasketaan aina ottaen huomioon **Ennakonpidätyksen komponentit** -sivulla TDS-verokomponenteille määritetty raja- ja poikkeusraja.</span><span class="sxs-lookup"><span data-stu-id="d142a-105">The TDS on invoices and payments is always calculated considering the threshold limit and exception threshold limit defined for the TDS tax components on the **Withholding tax components** page.</span></span> <span data-ttu-id="d142a-106">TDS-verokomponentit liitetään TDS-verokoodeihin, jotka sisältyvät TDS-veroryhmiin.</span><span class="sxs-lookup"><span data-stu-id="d142a-106">The TDS tax components are attached to TDS tax codes, which are included in the TDS tax groups.</span></span> <span data-ttu-id="d142a-107">TDS-veroryhmät liitetään toimittajiin ja asiakkaisiin TDS:n laskemiseksi lasku- tai maksutasolla.</span><span class="sxs-lookup"><span data-stu-id="d142a-107">The TDS tax groups are attached to vendors and customers to calculate TDS at the invoice-level or payment-level.</span></span>

<span data-ttu-id="d142a-108">TDS lasketaan, jos tapahtuman summa tai tietylle toimittajan TDS-ryhmälle kirjatut kumulatiiviset tapahtumat ylittävät **Ennakonpidätyksen komponentit** -sivulla määritetyn raja-arvon.</span><span class="sxs-lookup"><span data-stu-id="d142a-108">TDS is calculated if the amount for a transaction or the cumulative transactions posted with a specific TDS group for a vendor exceeds the threshold limit specified on the **Withholding tax components** page.</span></span> <span data-ttu-id="d142a-109">TDS-tietoja ei lasketa, ennen kuin kumulatiivinen tapahtuman summa ylittää määritetyn raja-arvon.</span><span class="sxs-lookup"><span data-stu-id="d142a-109">TDS will not be calculated until the cumulative transaction amount exceeds the specified threshold limit.</span></span>

<span data-ttu-id="d142a-110">Jos poikkeusraja-arvo määritetään yhdessä TDS-komponentin raja-arvon kanssa, TDS lasketaan, kun tietty tapahtumasumma ylittää poikkeusrajan, vaikka kumulatiivinen tapahtumasumma ei olisi määritettyä raja-arvoa suurempi.</span><span class="sxs-lookup"><span data-stu-id="d142a-110">If an exception threshold limit is specified along with the threshold limit for a TDS component, TDS is calculated when a specific transaction amount exceeds the exception threshold limit even if the cumulative transaction amount does not exceed the specified threshold limit.</span></span>
