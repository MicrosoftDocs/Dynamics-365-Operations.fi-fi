# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="7568c-103">Käyttöomaisuuden poisto vapaatekstilaskua käyttäen</span><span class="sxs-lookup"><span data-stu-id="7568c-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="7568c-104">Näiden ohjeiden avulla voit poistaa käyttöomaisuuden käyttämällä vapaatekstilaskua.</span><span class="sxs-lookup"><span data-stu-id="7568c-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="7568c-105">Siirry siirtymisruudussa kohtaan **Moduulit > Myyntireskontra > Laskut > Kaikki vapaamuotoiset laskut**.</span><span class="sxs-lookup"><span data-stu-id="7568c-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="7568c-106">Valitse **Uusi**.</span><span class="sxs-lookup"><span data-stu-id="7568c-106">Select **New**.</span></span>
3. <span data-ttu-id="7568c-107">Syötä tai valitse arvo **Asiakastili**-kentässä.</span><span class="sxs-lookup"><span data-stu-id="7568c-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="7568c-108">Tarkista oletuspäivämäärä kohdaas **Lasku** ja muokkaa sitä tarvittaessa.</span><span class="sxs-lookup"><span data-stu-id="7568c-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="7568c-109">Tarkista jäljellä olevat oletusarvoiset otsikkokentät, kuten **Valuutta**, ja muokkaa sitä tarvittaessa.</span><span class="sxs-lookup"><span data-stu-id="7568c-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="7568c-110">Vahvista **Kuvaus** -ja **Päätili** -kentät **Laskurivit**-osassa.</span><span class="sxs-lookup"><span data-stu-id="7568c-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="7568c-111">Tarkista oletusarvoinen **Arvonlisäveroryhmä**- ja **Nimikkeen arvonlisäveroryhmä** -kentät.</span><span class="sxs-lookup"><span data-stu-id="7568c-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="7568c-112">Kirjoita käyttöomaisuuden **Yksikköhinta** tai **Myyntisumma**.</span><span class="sxs-lookup"><span data-stu-id="7568c-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="7568c-113">Valitse **Rivin erittely** -osa.</span><span class="sxs-lookup"><span data-stu-id="7568c-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="7568c-114">Syötä tai valitse arvo **Käyttöomaisuus**-kentässä.</span><span class="sxs-lookup"><span data-stu-id="7568c-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="7568c-115">Valitse **Kirjaa**.</span><span class="sxs-lookup"><span data-stu-id="7568c-115">Select **Post**.</span></span>
