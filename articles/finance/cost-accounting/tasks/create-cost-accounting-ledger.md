[!include [banner](../../includes/banner.md)]

<span data-ttu-id="c3202-104">Kustannuslaskennan kirjanpito edustaa koko raportointiyksikköä.</span><span class="sxs-lookup"><span data-stu-id="c3202-104">A cost accounting ledger represents the overall reporting unit.</span></span> <span data-ttu-id="c3202-105">Se määritetään kustannustason dimension, tilastodimension, kirjanpidon vuosikalenterin ja valuutan mukaan.</span><span class="sxs-lookup"><span data-stu-id="c3202-105">It is defined by a cost element dimension, statistical dimension, fiscal calendar, and currency.</span></span> <span data-ttu-id="c3202-106">Se ei riipu yrityksen konseptista.</span><span class="sxs-lookup"><span data-stu-id="c3202-106">It is agnostic to the concept of legal entities.</span></span> <span data-ttu-id="c3202-107">Yritys ja sen tiedot voidaan liittää useisiin kustannuslaskennan kirjanpitoihin.</span><span class="sxs-lookup"><span data-stu-id="c3202-107">A legal entity and its data can be associated with many cost accounting ledgers.</span></span> <span data-ttu-id="c3202-108">Tämä tallenne käyttää esittelytietojen USP2-yritystä.</span><span class="sxs-lookup"><span data-stu-id="c3202-108">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="c3202-109">Valitse Kustannuslaskenta > Kirjanpidon asetukset > Kustannuslaskennan kirjanpidot.</span><span class="sxs-lookup"><span data-stu-id="c3202-109">Go to Cost accounting > Ledger setup > Cost accounting ledgers.</span></span>
2. <span data-ttu-id="c3202-110">Valitse Uusi.</span><span class="sxs-lookup"><span data-stu-id="c3202-110">Click New.</span></span>
3. <span data-ttu-id="c3202-111">Kirjoita arvo Nimi-kenttään.</span><span class="sxs-lookup"><span data-stu-id="c3202-111">In the Name field, type a value.</span></span>

5. <span data-ttu-id="c3202-113">Syötä tai valitse arvo Kirjanpidon vuosikalenteri -kenttään.</span><span class="sxs-lookup"><span data-stu-id="c3202-113">In the Fiscal calendar field, enter or select a value.</span></span>
6. <span data-ttu-id="c3202-114">Syötä tai valitse arvo Kirjanpitovaluutta-kenttään.</span><span class="sxs-lookup"><span data-stu-id="c3202-114">In the Accounting currency field, enter or select a value.</span></span>
7. <span data-ttu-id="c3202-115">Syötä tai valitse arvo Vaihtokurssin tyyppi -kenttään.</span><span class="sxs-lookup"><span data-stu-id="c3202-115">In the Exchange rate type field, enter or select a value.</span></span>
8. <span data-ttu-id="c3202-116">Syötä tai valitse arvo Dimensio-kenttään.</span><span class="sxs-lookup"><span data-stu-id="c3202-116">In the Statistical dimension field, enter or select a value.</span></span>
9. <span data-ttu-id="c3202-117">Valitse Tallenna.</span><span class="sxs-lookup"><span data-stu-id="c3202-117">Click Save.</span></span>
