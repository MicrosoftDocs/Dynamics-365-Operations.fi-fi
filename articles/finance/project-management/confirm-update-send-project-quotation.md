[!include [banner](../includes/banner.md)]

<span data-ttu-id="d22b0-104">Kun olet luonut ja lähettänyt projektitarjouksen asiakkaalle, sinun on saatava asiakkaalta vahvistus, ennen kuin päivität tarjouksen tilaksi **Lähetetty**.</span><span class="sxs-lookup"><span data-stu-id="d22b0-104">After you create and send a project quotation to a customer, you must get confirmation from the customer before you update the quotation status to **Sent**.</span></span> <span data-ttu-id="d22b0-105">Asiakkaan pyytämät muutokset voidaan päivittää tarjoukseen.</span><span class="sxs-lookup"><span data-stu-id="d22b0-105">Any modifications requested by the customer can be updated in the quotation.</span></span> <span data-ttu-id="d22b0-106">Kun olet päivittänyt tarjouksen tilaksi **Lähetetty**, et voi muokata sitä.</span><span class="sxs-lookup"><span data-stu-id="d22b0-106">After you update the quotation status to **Sent**, you can’t modify it.</span></span> <span data-ttu-id="d22b0-107">Seuraavassa menettelyssä kuvataan vaihtoehdot, joiden mukaan tarjouksen voi lähettää vahvistusta varten, tehdä päivityksiä palautteen perusteella ja lähettää sitten tarjouksen.</span><span class="sxs-lookup"><span data-stu-id="d22b0-107">The following procedure describes the options to send a quotation for confirmation, make updates based on feedback, and then send the quotation.</span></span>

2. <span data-ttu-id="d22b0-111">Valitse **Projektitarjous**-sivulla tarjous, jonka haluat lähettää hyväksyttäväksi.</span><span class="sxs-lookup"><span data-stu-id="d22b0-111">On the **Project quotation** page, select the quotation you want to send for confirmation.</span></span>

6. <span data-ttu-id="d22b0-116">Valitse **Tulosta kohdeasetukset** -sivulla **OK**.</span><span class="sxs-lookup"><span data-stu-id="d22b0-116">On the **Print destination settings** page, select **OK**.</span></span>
