# <a name="transfer-a-quotation-to-a-project"></a>Tarjouksen siirtäminen projektiin

[!include [banner](../includes/banner.md)]

Kun projektitarjouksen tila on **vahvistettu**, voit siirtää joitakin tai kaikkia tarjousrivejä uuteen tai aiemmin luotuun projektiin käyttämällä **ohjattua projektinsiirtoa**. 

Voit käyttää ohjattua toimintoa seuraavasti.

1. Valitse **Projektinhallinta ja kirjanpito** > **Tarjoukset** > **Projektitarjoukset**.
2. Valitse siirrettävä tarjous ja valitse **Seuranta**-välilehden **Muokkaa**-ryhmästä **Siirrä projektiin**.
3. Seuraa **Siirrä tarjous projektiin**-ohjattua toimintoa yhdistääksesi ja siirtääksesi tarjouksen projektiin.
