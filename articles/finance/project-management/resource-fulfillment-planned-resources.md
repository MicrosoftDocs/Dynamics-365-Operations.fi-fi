# <a name="resource-fulfillment-for-planned-resources"></a><span data-ttu-id="d4f43-103">Resurssin toteuttaminen suunnitelluille resursseille</span><span class="sxs-lookup"><span data-stu-id="d4f43-103">Resource fulfillment for planned resources</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="d4f43-104">Projektipäällikkö voi suunnitella projektille tarvittavat resurssiroolit.</span><span class="sxs-lookup"><span data-stu-id="d4f43-104">A project manager can plan required resource roles for a project.</span></span> <span data-ttu-id="d4f43-105">Resurssipäällikkö näkee suunnitellut resurssit pyyntöinä **Resurssin toteuttaminen** -sivulla ja voi määrittää varsinaiset resurssit.</span><span class="sxs-lookup"><span data-stu-id="d4f43-105">The resource manager will see these planned resources as requests on the **Resource fulfillment** page and can assign actual resources.</span></span>

1. <span data-ttu-id="d4f43-106">Valitse **Kaikki projektit** -sivulla **XYZ-päivitysvaihe 2** -projekti.</span><span class="sxs-lookup"><span data-stu-id="d4f43-106">On the **All projects** page, select the **XYZ Upgrade Phase 2** project.</span></span>
2. <span data-ttu-id="d4f43-107">Valitse **Projekti** ja sitten **Muokkaa**.</span><span class="sxs-lookup"><span data-stu-id="d4f43-107">Select **Project**, and then select **Edit**.</span></span>
3. <span data-ttu-id="d4f43-108">Valitse **Projektiryhmä ja ajoitus** -välilehdestä **Lisää**.</span><span class="sxs-lookup"><span data-stu-id="d4f43-108">On the **Project team and scheduling** tab, select **Add**.</span></span>
4. <span data-ttu-id="d4f43-109">Valitse **Lisää roolit** -valintaikkunassa **Ohjelmistokehittäjä**-rooli.</span><span class="sxs-lookup"><span data-stu-id="d4f43-109">In the **Add roles** dialog box, select the **Software developer** role.</span></span>
5. <span data-ttu-id="d4f43-110">Valitse **Tallenna** ja sulje sitten projektisivu.</span><span class="sxs-lookup"><span data-stu-id="d4f43-110">Select **Create**, and then close the project page.</span></span>
6. <span data-ttu-id="d4f43-111">Valitse **Resurssin toteuttaminen** -sivulta **Ohjelmistokehittäjä 1** **XYZ-päivitysvaihe 2** -projektiin.</span><span class="sxs-lookup"><span data-stu-id="d4f43-111">On the **Resource fulfillment** page, select **Software developer 1** for the **XYZ Upgrade project Phase 2** project.</span></span>
7. <span data-ttu-id="d4f43-112">Valitse työntekijä ja sitten **Määritä**.</span><span class="sxs-lookup"><span data-stu-id="d4f43-112">Select a worker, and then select **Assign**.</span></span>
8. <span data-ttu-id="d4f43-113">Tarkista, että **Ohjelmistokehittäjä 1** -rivi on poistettu **XYZ-päivitysvaihe 2** -projektista.</span><span class="sxs-lookup"><span data-stu-id="d4f43-113">Verify that the line for **Software developer 1** has been removed for the **XYZ Upgrade project Phase 2** project.</span></span>
9. <span data-ttu-id="d4f43-114">Tarkista **Projektiryhmä ja ajoitus** -välilehdessä, että edellisessä vaiheessa valitsemasi työntekijä on lisätty **XYZ-päivitysvaihe 2** -projektiin **ohjelmistokehittäjäksi**.</span><span class="sxs-lookup"><span data-stu-id="d4f43-114">On the **Project team and scheduling** tab, for the **XYZ Upgrade Phase 2** project, verify that the worker that you selected in the previous step has been added as **Software developer**.</span></span>
