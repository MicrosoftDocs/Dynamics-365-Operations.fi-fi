# <a name="project-mobile-applications-overview"></a>Project-mobiilisovellusten yleiskuvaus

[!include [banner](../includes/banner.md)]

Projektin aikaan liittyviä mobiilisovelluksia on kolme, ja ne ovat Microsoft Dynamics 365 Project Timesheet, Projektin aikamerkintä ja Omat aikaraportit / Omat tarkistettavat aikaraportit (optimoitu mobiilikäyttöön).

## <a name="project-timesheet-mobile-app"></a>Project Timesheet -mobiilisovellus

Dynamics 365 Project Timesheet -mobiilisovelluksen avulla käyttäjät lähettävät ja hyväksyvät projektien aikaraportteja mobiililaitteellaan. Tämä mobiilisovellus näyttää Dynamics 365 Financein projektinhallinnan ja kirjanpidon alueen aikaraporttitoiminnot. Se auttaa parantamaan käyttäjien tuottavuutta ja tehokkuutta ja auttaa myös nopeasti projektin aikaraporttien syöttämiseen ja hyväksyntään.

## <a name="project-time-entry-workspace"></a>Projektin aikamerkintöjen mobiilityötila

Projektin aikamerkintöjen työtilan avulla käyttäjät voivat syöttää ja tallentaa projektin aikoja. Se ei kuitenkin anna käyttäjien lähettää tai hyväksyä projektin aikaraportteja. Sitä voidaan käyttää vain aikamerkintöihin. Tämä mobiilityötila on tarkoitettu käytettäväksi yhdessä Finance and Operations -mobiilisovelluksen kanssa.

## <a name="my-timesheetstimesheets-for-my-review"></a>Omat aikaraportit / Omat tarkistettavat aikaraportit

Omat aikaraportit (optimoitu mobiilikäyttöön) ja Omat tarkistettavat aikaraportit (optimoitu mobiilikäyttöön) muodostavat ratkaisun, jonka avulla työntekijät voivat luoda, lähettää ja hyväksyä projektin aikaraportteja mobiililaitteella. Työntekijät voivat käyttää Finance-toimintoja sovelluksen URL-osoitteen kautta. Tämä ratkaisu ei vaadi Finance and Operations -mobiilisovellusta.
