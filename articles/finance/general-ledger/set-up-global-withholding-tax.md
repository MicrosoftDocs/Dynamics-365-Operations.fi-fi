# <a name="set-up-global-withholding-tax"></a>Määritä yleinen ennakonpidätys

Tässä ohjeaiheessa on luettelo vaiheista myynnin ja ostojen yleisen ennakonpidätyksen määrittämiseksi. 

1. Määritä ennakonpidätysviranomaiset **Ennakonpidätysviranomaiset**-sivulla.

2. Määritä ennakonpidätyksen tilityskaudet **Ennakonpidätyksen tilityskaudet** -sivulla.

3. Määritä ennakonpidätyksen kirjanpidon kirjausryhmä **Ennakonpidätys > Kirjanpidon kirjausryhmä** -sivulla.

   > [!Note] 
   >
   > **Ennakonpidätys**-tilille määritetään päätili, jonka **Kirjaustyyppi** on Ennakonpidätys. **Ennakonpidätyksen vastatilille** määritetään myös päätili, jonka **Kirjaustyyppi** on Ennakonpidätyksen vastatili. Siirry kohtaan **Kirjanpito > Tilikartta > Tilit > Päätilit** ja määritä päätileille **Kirjaustyyppi** alilomakkeessa **Kirjauksen oikeellisuustarkistus**.

4. Määritä ennakonpidätyskoodit **Ennakonpidätyskoodit**-sivulla.

5. Määritä ennakonpidätysryhmät **Ennakonpidätysryhmät**-sivulla.

6. Määritä ennakonpidätyksen tuottotyypit **Ennakonpidätyksen** **tuottotyypit** -sivulla.

7. Määritä nimikkeen tai palvelun ennakonpidätysryhmät **Nimikkeen ennakonpidätysryhmät**-sivulla.

8. Määritä **Laskun vähimmäissumma** sivulla **Kirjanpitoparametrit > Ennakonpidätys**.
