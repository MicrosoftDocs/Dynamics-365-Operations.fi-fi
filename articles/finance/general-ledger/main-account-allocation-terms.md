[!include [banner](../includes/banner.md)]

<span data-ttu-id="c5a19-104">Tässä ohjeaiheessa on tietoja päätilin kohdistusehtojen käyttämisestä.</span><span class="sxs-lookup"><span data-stu-id="c5a19-104">This topic provides information about using allocation terms on a main account.</span></span> <span data-ttu-id="c5a19-105">Kohdistusehtoja käytetään jaettaessa summia useiden kirjanpitotiliyhdistelmien kesken.</span><span class="sxs-lookup"><span data-stu-id="c5a19-105">Allocation terms are used to distribute amounts across multiple ledger account combinations.</span></span> <span data-ttu-id="c5a19-106">Niiden avulla varmistetaan, että kulut tai tuotot veloitetaan oikealta kohteelta kirjanpidossa.</span><span class="sxs-lookup"><span data-stu-id="c5a19-106">They help ensure that expenses or revenues are charged to the correct object in accounting.</span></span>

<span data-ttu-id="c5a19-107">Kukin päätilille luomasi kohdistusehto määrittää tositteen prosenttiosuuden, joka kohdistetaan yhden lähteen päätililtä ja taloushallinnon dimensioiden yhdistelmältä.</span><span class="sxs-lookup"><span data-stu-id="c5a19-107">Each allocation term that you create on a main account defines the percentage of a voucher to be allocated from a single-source main account and a combination of financial dimensions.</span></span> <span data-ttu-id="c5a19-108">Lisäksi määritetään kohteen päätili ja taloushallinnon dimensiot, joihin summa kohdistetaan.</span><span class="sxs-lookup"><span data-stu-id="c5a19-108">In addition, you define the destination main account and financial dimensions where the amount will be allocated.</span></span>

Järjestelmä ylläpitää alkuperäistä talousdimensiota kirjaamisen yhteydessä.</span><span class="sxs-lookup"><span data-stu-id="c5a19-118">If you select yes, no additional information is specified, and the system will maintain the original financial dimensions when you post.</span></span>

<span data-ttu-id="c5a19-119">Päätiliin voi lisätä rajoittamattoman määrän kohdistusehtoja.
