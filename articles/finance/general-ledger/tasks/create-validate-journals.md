[!include [banner](../../includes/banner.md)]

<span data-ttu-id="4e741-104">Tässä tehtävässä luodaan ja vahvistetaan kirjauskansioita ja kirjauskansion rivejä.</span><span class="sxs-lookup"><span data-stu-id="4e741-104">This procedure creates and validates journals and journal lines.</span></span> <span data-ttu-id="4e741-105">Voit kokeilla tätä menettelyä USMF:n esittely-yrityksessä.</span><span class="sxs-lookup"><span data-stu-id="4e741-105">You can try this procedure using the USMF demo company.</span></span>  

1. <span data-ttu-id="4e741-106">Siirry kohtaan **Kirjanpito > Kirjauskansioviennit > Yleiset kirjauskansiot**.</span><span class="sxs-lookup"><span data-stu-id="4e741-106">Go to **General ledger > Journal entries > General journals**.</span></span>
2. <span data-ttu-id="4e741-107">Valitse **Uusi**.</span><span class="sxs-lookup"><span data-stu-id="4e741-107">Click **New**.</span></span>
3. <span data-ttu-id="4e741-108">Avaa haku valitsemalla **Nimi**-kentässä avattavan valikon painike.</span><span class="sxs-lookup"><span data-stu-id="4e741-108">In the **Name** field, click the drop-down button to open the lookup.</span></span>
4. <span data-ttu-id="4e741-109">Etsi haluamasi tietue luettelosta ja valitse se.</span><span class="sxs-lookup"><span data-stu-id="4e741-109">In the list, find and select the desired record.</span></span>
5. <span data-ttu-id="4e741-110">Valitse **Rivit**.</span><span class="sxs-lookup"><span data-stu-id="4e741-110">Click **Lines**.</span></span>
6. <span data-ttu-id="4e741-111">Anna **Tili**-kenttään soveltuva, tilityyppiin perustuva tili.</span><span class="sxs-lookup"><span data-stu-id="4e741-111">In the **Account** field enter an appropriate account based on the Account type.</span></span>
7. <span data-ttu-id="4e741-112">Kirjoita **Kuvaus**-kenttään arvo.</span><span class="sxs-lookup"><span data-stu-id="4e741-112">In the **Description** field, type a value.</span></span>
8. <span data-ttu-id="4e741-113">Anna summa tilille joko **veloituksena** tai **hyvityksenä**.</span><span class="sxs-lookup"><span data-stu-id="4e741-113">Enter an amount for the account as either a **Debit** or **Credit**.</span></span>

10. <span data-ttu-id="4e741-115">Valitse **Vahvista**.</span><span class="sxs-lookup"><span data-stu-id="4e741-115">Click **Validate**.</span></span>
11. <span data-ttu-id="4e741-116">Valitse **Vahvista**.</span><span class="sxs-lookup"><span data-stu-id="4e741-116">Click **Validate**.</span></span>
12. <span data-ttu-id="4e741-117">Valitse **Kirjaa**.</span><span class="sxs-lookup"><span data-stu-id="4e741-117">Click **Post**.</span></span>
13. <span data-ttu-id="4e741-118">Valitse **Tosite**.</span><span class="sxs-lookup"><span data-stu-id="4e741-118">Click **Voucher**.</span></span>
