# <a name="set-up-ledger-posting-groups-for-sales-tax"></a><span data-ttu-id="697f5-103">Arvonlisäveron kirjanpidon kirjausryhmien määrittäminen</span><span class="sxs-lookup"><span data-stu-id="697f5-103">Set up Ledger posting groups for sales tax</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="697f5-104">Arvonlisävero lasketaan ja kirjataan kirjanpidon kirjausryhmien päätileille.</span><span class="sxs-lookup"><span data-stu-id="697f5-104">Sales tax is calculated and posted to main accounts that are specified in Ledger posting groups.</span></span> <span data-ttu-id="697f5-105">Kirjanpidon kirjausryhmät liitetään kuhunkin arvonlisäverokoodiin.</span><span class="sxs-lookup"><span data-stu-id="697f5-105">Ledger posting groups are attached to each sales tax code.</span></span> <span data-ttu-id="697f5-106">Voit määrittää kullekin arvonlisäverokoodille kirjanpidon kirjausryhmät käyttämällä kaikissa arvonlisäverokoodeissa yhtä kirjanpidon kirjausryhmää tai liittämällä arvonlisäverokoodeille useita kirjanpidon kirjausryhmiä.</span><span class="sxs-lookup"><span data-stu-id="697f5-106">You can set up individual ledger posting groups for each sales tax code, use one ledger posting group for all sales tax codes or assign multiple ledger posting groups to the sales tax codes.</span></span> <span data-ttu-id="697f5-107">Tässä tallenteessa käytetään esittely-yritystä DEMF.</span><span class="sxs-lookup"><span data-stu-id="697f5-107">This recording uses the DEMF demo company.</span></span> 

1. <span data-ttu-id="697f5-108">Siirry kohtaan **siirtymisruutu > Moduulit > Vero > Asetukset > Arvonlisävero > Kirjanpidon kirjausryhmät**.</span><span class="sxs-lookup"><span data-stu-id="697f5-108">Go to **Navigation pane > Modules > Tax > Setup > Sales tax > Ledger posting groups**.</span></span>
2. <span data-ttu-id="697f5-109">Valitse **Uusi**.</span><span class="sxs-lookup"><span data-stu-id="697f5-109">Click **New**.</span></span>
3. <span data-ttu-id="697f5-110">Kirjoita arvo **Kirjanpidon kirjausryhmä** -kenttään.</span><span class="sxs-lookup"><span data-stu-id="697f5-110">In the **Ledger posting group** field, type a value.</span></span>
4. <span data-ttu-id="697f5-111">Kirjoita **Kuvaus**-kenttään arvo.</span><span class="sxs-lookup"><span data-stu-id="697f5-111">In the **Description** field, type a value.</span></span>
5. <span data-ttu-id="697f5-112">Valitse **Maksettava arvonlisävero** -kentässä päätili lähtevälle arvonlisäverolle, joka maksetaan veroviranomaiselle.</span><span class="sxs-lookup"><span data-stu-id="697f5-112">In the **Sales tax payable** field, select the main account for outgoing sales taxes that are payable to the tax authority.</span></span> <span data-ttu-id="697f5-113">Arvonlisäverot kerätään veroviranomaisen puolesta, kun verollisia tavaroita ja palveluja myydään.</span><span class="sxs-lookup"><span data-stu-id="697f5-113">Sales taxes are collected on behalf of the tax authority when you sell taxable goods and services.</span></span>  
6. <span data-ttu-id="697f5-114">Valitse **Saatava arvonlisävero** -kentässä päätili saapuvalle arvonlisäverolle, joka vastaanotetaan veroviranomaiselta.</span><span class="sxs-lookup"><span data-stu-id="697f5-114">In the **Sales tax receivable** field, select the main account for incoming taxes that are received from the tax authority.</span></span> <span data-ttu-id="697f5-115">Toimittajat keräävät verot veroviranomaisen puolesta, kun verollisia tavaroita ja palveluja ostetaan.</span><span class="sxs-lookup"><span data-stu-id="697f5-115">Vendors collect taxes on behalf of the tax authority when you buy taxable goods and services.</span></span> <span data-ttu-id="697f5-116">Tämä kenttä ei ole käytettävissä, jos Käytä arvonlisäverotuksen sääntöjä -vaihtoehto on valittu **Kirjanpitoparametrit**-sivulla.</span><span class="sxs-lookup"><span data-stu-id="697f5-116">This field is not available if the Apply sales tax taxation rules option is selected in the **General ledger parameters** page.</span></span> <span data-ttu-id="697f5-117">Sen sijaan toimittajille maksettavat arvonlisäverot hyvitetään samalta tililtä kuin ostot.</span><span class="sxs-lookup"><span data-stu-id="697f5-117">Instead, sales taxes that are paid to vendors are debited to the same account as the purchase.</span></span>   
7. <span data-ttu-id="697f5-118">Valitse **Käyttöverokulu**-kentässä päätili niille vähennettäville käyttöveroille, joita veroviranomainen ei peri tai joita toimittajat eivät raportoi veroviranomaiselle osana EU:n käännettyä GOT/HS-veroa.</span><span class="sxs-lookup"><span data-stu-id="697f5-118">In the **Use tax expense** field, select  the main account for posting deductible Use taxes that are not claimed or reported to the tax authority by vendors as part of EU reverse charge GST/HST.</span></span> <span data-ttu-id="697f5-119">Tapahtumassa käytettävän **arvonlisäveroryhmän** **arvonlisäverokoodin** **Käyttövero**-vaihtoehto on valittava.</span><span class="sxs-lookup"><span data-stu-id="697f5-119">The **Use tax** option must be selected for the **Sales tax code** in the **Sales tax group** that is used in the transaction.</span></span> <span data-ttu-id="697f5-120">Tämä kenttä ei ole käytettävissä, jos **Käytä arvonlisäverotuksen sääntöjä** -vaihtoehto on valittu **Kirjanpitoparametrit**-sivulla.</span><span class="sxs-lookup"><span data-stu-id="697f5-120">This field won't be available if the **Apply sales tax taxation rules** option is selected on the **General ledger parameters** page.</span></span>   
8. <span data-ttu-id="697f5-121">Valitse **Maksettava käyttövero** -kentässä päätili veroviranomaiselle maksettavien saapuvien käyttöverojen kirjaamista varten.</span><span class="sxs-lookup"><span data-stu-id="697f5-121">In the **Use tax payable** field, select the main account for posting incoming Use taxes that are payable to tax authorities.</span></span> <span data-ttu-id="697f5-122">**Arvonlisäveroryhmän** **arvonlisäverokoodin** **Käyttövero**-vaihtoehto on valittava, kun **käyttövero** kirjataan.</span><span class="sxs-lookup"><span data-stu-id="697f5-122">The **Use tax** option must be selected in the **Sales tax code** in the **Sales tax group** to post **Use tax**.</span></span> <span data-ttu-id="697f5-123">Jos **Kirjanpidon parametrit** -sivun **Käytä arvonlisäverotuksen sääntöjä** -vaihtoehto on valittu, vastakirjaus kirjataan tapahtuman kulutilille.</span><span class="sxs-lookup"><span data-stu-id="697f5-123">If **Apply sales tax taxation rules** option is selected in **General ledger parameters** page, the offset is posted to the transaction's expense account.</span></span>   
9. <span data-ttu-id="697f5-124">Valitse **Tilitystili**-kentässä päätili, jolle **Maksettava käyttövero**- ja **Saatava arvonlisävero** -kentässä määritettyjen kirjanpitotilien nettosaldo kirjataan.</span><span class="sxs-lookup"><span data-stu-id="697f5-124">In the **Settlement account** field, select the main account that the net balance of the ledger accounts specified in the **Use tax payable** and **Sales tax receivable** fields will be posted.</span></span> <span data-ttu-id="697f5-125">Saldo luodaan, kun arvonlisäveron tilitys ja työn kirjaaminen on suoritettu.</span><span class="sxs-lookup"><span data-stu-id="697f5-125">The balance will be created when the sales taxes settle and the posting job has been run.</span></span>  <span data-ttu-id="697f5-126">Jos tilityskauden veroviranomainen on liitetty toimittajan tiliin, saldo kirjataan sen sijaan toimittajan tiliin.</span><span class="sxs-lookup"><span data-stu-id="697f5-126">If the taxing authority for the settlement period is associated with a vendor account, the balance will be posted to the vendor account instead.</span></span>
10. <span data-ttu-id="697f5-127">Valitse **Toimittajan käteisalennus** -kentässä päätili, jonne tähän kirjanpidon kirjausryhmän liitetyn arvonlisäverokoodin käteisalennukset kirjataan.</span><span class="sxs-lookup"><span data-stu-id="697f5-127">In the **Vendor cash discount** field, select the main account to post cash discount for Sales tax codes associated with this Ledger posting group.</span></span> <span data-ttu-id="697f5-128">Tämä kenttä on valinnainen. Jos tiliä ei ole syötetty, käytetään **Käteisalennuksen koodit** -kohdan päätiliä.</span><span class="sxs-lookup"><span data-stu-id="697f5-128">This is optional and if no account is entered,  the main account on **Cash discount codes** will be used.</span></span> <span data-ttu-id="697f5-129">**Kirjanpidon kirjausryhmä** -kohdassa kannattaa ehkä käyttää eri tilejä, jos käytössä on arvonlisäveroryhmien Mitätöi käteisalennuksen arvonlisävero -kohta.</span><span class="sxs-lookup"><span data-stu-id="697f5-129">It can be useful to use different accounts per **Ledger posting group** if using the reverse sales tax on cash discount option on Sales tax groups.</span></span>  
11. <span data-ttu-id="697f5-130">Valitse **Asiakastapauksen alennus** -kentässä päätili, jonne tähän **kirjanpidon kirjausryhmän** liitetyn **arvonlisäverokoodin** käteisalennukset kirjataan.</span><span class="sxs-lookup"><span data-stu-id="697f5-130">In the **Customer case discount** field, select the main account to post cash discount for **Sales tax codes** associated with this **Ledger posting group**.</span></span> <span data-ttu-id="697f5-131">Tämä kenttä on valinnainen. Jos tiliä ei ole syötetty, käytetään **Käteisalennuksen koodit** -kohdan päätiliä.</span><span class="sxs-lookup"><span data-stu-id="697f5-131">This is optional and if no account is entered, the main account on the **Cash discount codes** will be used.</span></span> <span data-ttu-id="697f5-132">**Kirjanpidon kirjausryhmä** -kohdassa kannattaa ehkä käyttää eri tilejä, jos käytössä on **arvonlisäveroryhmien** Mitätöi käteisalennuksen arvonlisävero -kohta.</span><span class="sxs-lookup"><span data-stu-id="697f5-132">It can be useful to use different accounts per **Ledger posting group** if using the reverse sales tax on cash discount option on **Sales tax groups**.</span></span>  
12. <span data-ttu-id="697f5-133">Valitse **Tallenna**.</span><span class="sxs-lookup"><span data-stu-id="697f5-133">Click **Save**.</span></span>
