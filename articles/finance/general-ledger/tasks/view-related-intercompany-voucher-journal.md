# <a name="view-related-intercompany-voucher-from-journal"></a>Näytä konsernin sisäinen tosite kirjauskansiosta

[!include [banner](../../includes/banner.md)]

Liittyvän tositteen ikkunassa näkyy siirtymäyrityksen tosite,kun konsernin sisäinen tapahtuma kirjataan kirjauskansiosta.


## <a name="post-an-intercompany-journal"></a>Kirjaa konsernin sisäinen kirjauskansio
1. Valitse **Yleiset kirjauskansiot** ja **Uusi**.
2. Merkitse valittu rivi luettelossa.
3. Anna tai valitse **Nimi**-kentässä konsernin sisäisen kirjauskansio.
4. Napsauta luettelossa **Rivit**, merkitse valittu rivi.
5. Määritä **Tili**-kenttään haluamasi arvot.
6. Anna tai valitse **Kuvaus**-kentässä arvo.
7. Sulje sivu.
8. Syötä **Debet**-kenttään numero.
9. Kirjoita tai valitse **Siirtymäyritys**-kentässä siirtymäyritys.
10. Määritä **Vastatili**-kenttään haluamasi arvot.
11. Valitse **Kirjaa**.

## <a name="view-related-intercompany-voucher"></a>Näytä liittyvä konsernin sisäinen tosite
1. Valitse **Tosite**.
2. Valitse **Liittyvät tositteet** luettelossa, merkitse valittu rivi.
3. Valitse **Tosite**.
