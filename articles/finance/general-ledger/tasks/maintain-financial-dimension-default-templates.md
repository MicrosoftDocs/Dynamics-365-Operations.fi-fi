[!include [banner](../../includes/banner.md)]

1. <span data-ttu-id="3ac12-106">Valitse siirtymisruudussa **Moduulit > Kirjanpito > Tilikartta > Dimensiot > Taloushallinnon dimensioiden oletusmallit**.</span><span class="sxs-lookup"><span data-stu-id="3ac12-106">In the navigation pane, go to **Modules > General ledger > Chart of accounts > Dimensions > Financial dimension default templates**.</span></span>

3. <span data-ttu-id="3ac12-108">Kirjoita **Mallin tunnus** -kenttään mallin tunnus.</span><span class="sxs-lookup"><span data-stu-id="3ac12-108">In the **Template ID** field, enter a Template ID.</span></span>
4. <span data-ttu-id="3ac12-109">Syötä prosenttiarvo ensimmäisille dimensioarvoille.</span><span class="sxs-lookup"><span data-stu-id="3ac12-109">Enter the percentage for the first dimension values.</span></span>

6. <span data-ttu-id="3ac12-111">Valitse **Lisää**.</span><span class="sxs-lookup"><span data-stu-id="3ac12-111">Select **Add**.</span></span>
7. <span data-ttu-id="3ac12-112">Syötä toinen prosenttiarvo seuraaville dimensioarvoille.</span><span class="sxs-lookup"><span data-stu-id="3ac12-112">Enter another percent for the next dimension values.</span></span> <span data-ttu-id="3ac12-113">Tässä esimerkissä voit kirjoittaa prosenttiosuuden, jotta vähemmän kuin 100 % on varattuna.</span><span class="sxs-lookup"><span data-stu-id="3ac12-113">For this example, enter a percentage so you still have less than 100% allocated.</span></span> 
8. <span data-ttu-id="3ac12-114">Anna taloushallinnon dimensioarvot, jotka vastaavat annettua prosenttiarvoa.</span><span class="sxs-lookup"><span data-stu-id="3ac12-114">Enter the financial dimension values that correspond to the percent entered.</span></span>
9. <span data-ttu-id="3ac12-115">Valitse **Lisää**.</span><span class="sxs-lookup"><span data-stu-id="3ac12-115">Select **Add**.</span></span>
10. <span data-ttu-id="3ac12-116">Anna toinen prosenttiarvo, jonka myötä kokonaisuuden prosenttiarvo on 100 %.</span><span class="sxs-lookup"><span data-stu-id="3ac12-116">Enter another percent, which will make the total percent equal to 100%.</span></span>
11. <span data-ttu-id="3ac12-117">Anna taloushallinnon dimensioarvot, jotka vastaavat annettua prosenttiarvoa.</span><span class="sxs-lookup"><span data-stu-id="3ac12-117">Enter the financial dimension values that correspond to the percent entered.</span></span>
12. <span data-ttu-id="3ac12-118">Valitse **Tallenna**.</span><span class="sxs-lookup"><span data-stu-id="3ac12-118">Select **Save**.</span></span>
