# <a name="invoice-capture-received-files"></a>Invoice Capture -ratkaisun vastaanotetut tiedostot

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

Invoice Capture -ratkaisussa **Vastaanotetut tiedostot** -sivu on keskeinen paikka, jossa laskutiedostot vastaanotetaan eri lähteistä.

## <a name="upload-invoice-files"></a>Laskutiedostojen lataaminen

Ostoreskontran käsittelijät voivat ladata laskukuvat valitsemalla **Lataa tiedosto** ja avaamalla **Lataa**-valintaikkuna. Kun ostoreskontran käsittelijä valitsee tiedoston, **Lataa**-painike on käytettävissä.

## <a name="include-or-obsolete-invoice-files"></a>Laskutiedostojen sisällyttäminen tai merkitseminen vanhentuneiksi

Käyttäjät voivat valita laskuja, joissa on virheitä tai varoituksia, ja sisällyttää laskut tai merkitä ne vanhentuneiksi valitsemalla toimintoruudussa vastaavan painikkeen. Vanhentuneeksi merkitty lasku ei näy **Vastaanotetut tiedostot (vanhentuneet)** -näkymässä.

## <a name="view-captured-invoices"></a>Näytä siepatut laskut

Kun vastaanotettu tiedosto on tunnistettu onnistuneesti, tekoälymalli palauttaa siepatun laskun tiedot. Tiedot syötetään Microsoft Dataverseen. Tämän jälkeen ostoreskontran käsittelijä voi vahvistaa laskun tiedot valitsemalla **Näytä siepattu lasku** -kohdan. Käyttäjät voivat ladata alkuperäisen laskutiedoston tarvittaessa.
