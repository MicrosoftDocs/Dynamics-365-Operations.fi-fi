# <a name="localization-and-regulatory-features"></a>Lokalisointi ja sääntelyn ominaisuudet

Microsoft Dynamics-356, työvaiheen sisältää joitakin maa-/ aluekohtaiset asetukset ja prosesseja. Tämä asetus on käytössä, active oikeushenkilön ensisijaisen osoitteen perusteella. Tässä aiheessa on luettelo resursseista, jotka auttavat sinua Lisätietoja maa-/ aluekohtaisia toimintoja. 

## <a name="developer-and-isv-information"></a>ISV-ja kehittäjä
Seuraavissa resursseissa on tietoja, jotka auttavat sovelluskehittäjille ja valmistajille, jotka luodaan maa-/ aluekohtaisia mukautuksia.
-   [Erillinen lokalisoinnin mallit](separate-localization-models.md)
-   [Maan/alueen yhteydessä sovelletaan](apply-country-context.md)
-   [Sääntelyn tutkinnot](regulatory-certifications.md)
-   [Luokitella lokalisoinnin ominaisuuksia](classify-localization-features.md)
-   [Maakoodit - ISO 3166](https://www.iso.org/iso-3166-country-codes.html)

## <a name="localization-updates"></a>Lokalisoinnin päivitykset
Seuraavissa resursseissa on tietoja lokalisoinnin suunnitellut ja uusia ominaisuuksia. 
-   [Lokalisoinnin portal](https://mbs.microsoft.com/customersource/northamerica/ax/support/support-news/GFMLocalizationPortalMC)

-   [Microsoft ilmoittaa tietoja ilmoituksista maa-asetuksen ja seurata tilaa sääntelyn ominaisuudet](submit-localization-alerts.md)
