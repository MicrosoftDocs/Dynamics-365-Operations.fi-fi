# <a name="set-up-fixed-asset-groups"></a>Määritä käyttöomaisuusryhmät

[!include[task guide banner](../../includes/task-guide-banner.md)]

Tässä menettelyssä näytetään, miten uusi käyttöomaisuusryhmä luodaan. Siinä käytetään kirjanpitäjän roolia ja USMF-yrityksen esittelytietoja.

1. Siirry kohtaan Käyttöomaisuus > Asetukset > Käyttöomaisuusryhmät.
2. Valitse Uusi.
3. Kirjoita arvo Käyttöomaisuusryhmä-kenttään.
4. Kirjoita arvo Nimi-kenttään.
    * Käyttöomaisuuserien automaattinen numerointi ja käyttöomaisuuserien automaattinen numerointi korvaavat käyttöomaisuusparametrien asetukset. Voit muuttaa asetusta tässä, jos tämän käyttöomaisuusryhmän käyttöomaisuuserillä on eri numerointi kuin muilla ryhmillä.  
5. Valitse Kirjat.
6. Syötä tai valitse arvo kentässä Kirja.
    * Jos Laske poisto -kentän arvoksi on määritetty Kyllä, käyttöomaisuuskirja sisällytetään poistoehdotuksiin. Jos Laske poisto -kentän arvoksi on määritetty Ei, käyttöomaisuudelle ei tehdä automaattisesti poistoa.  
7. Määritä käyttöomaisuuserän käyttöikä vuosina.
    * Ota huomioon, että Poistokaudet-kentän arvo lasketaan käyttöiän määrittämisen jälkeen.  
8. Valitse vaihtoehto Poistomenetelmä-kentässä.
9. Sulje sivu.
